# Life Design Project

このプロジェクトは、個人の人生後半をより良く設計するための再利用可能な人生設計フレームワークです。

## 目的

- 人生の理想状態を言語化する
- 不安・課題・リスクを整理する
- 仕事・健康・家族・お金・時間のバランスを可視化する
- 最終的に「いい人生だった」と思える状態を設計する

## 基本フロー

1. 対象者に質問する
2. 本音・価値観・現状を記録する
3. 情報を構造化する
4. 理想と現実のギャップを整理する
5. 行動可能な人生設計に落とす

## 役割

- 平山：インタビュアー、本音を引き出す対話者
  - 対話で得た情報をAIを活用して整理・構造化する
  - 思考の可視化を行い、人生設計として形にする
- ChatGPT：設計者、質問設計、分析、構造化
- Claude Code：実作業、ファイル作成、Markdown整理
- 対象者：自分の人生を言語化する人

## AI活用の方針

このプロジェクトではAIを活用して思考整理・可視化を行うが、
主役はあくまで人間同士の対話である。

AIは以下の役割として使用する：

- 情報の構造化
- 思考の整理
- 可視化の補助
- 行動計画への変換

最終的な意思決定および価値判断は対象者本人に委ねる。
