# File Update Rules

## 基本ルール

- 事実と推測を分ける
- 本人発言はなるべく原文に近く残す
- 分析は分析フォルダーに書く
- 行動計画は life_design フォルダーに書く
- 可視化用の要約は visualization フォルダーに書く

## 更新時の注意

- 既存情報を削除しない
- 新しい情報は日付付きで追記する
- 不明点は「未確認事項」に入れる
- ChatGPTから渡された構造を優先する

## 可視化データルール

- スコアや数値はCSVに必ず記録する
- Markdownは説明・文脈に使う
- JSONはWeb表示用の統合データとして扱う
- 同じ情報を重複管理しない（役割を分ける）

## AI活用ルール

- AIは思考整理と可視化の補助として使用する
- AIが結論を決めることはしない
- 本人の発言・価値観を最優先する
- 推測と事実を必ず分ける
- 人間の対話を起点とする
