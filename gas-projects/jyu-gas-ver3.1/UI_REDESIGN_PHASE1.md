## 推奨案（案A）の具体的セル配置

### レイアウト設計（行7〜8）— 最終確定 2026-03-22

> ⚠️ **実シート確認済み: Row 9 は Case1 開始のため使用不可。行7〜8 のみ使用。**

```
Row 7  : A7:「会計区分」   B7:[dropdown] 保険のみ/保険+自費/自費のみ
         C7:「自費メニュー」D7:[dropdown] 手技50分/運動療法/セルフケア/ジム体験/その他
         E7:「自費金額（円）」F7:[数値入力]
         G7:「会計合計」   H7:[数式 =IF(F7="",E3,E3+F7)]  ← 薄黄背景（表示専用）

Row 8  : A8:「慢性候補」  B8:[チェックボックス]
         C8:「次回予約」  D8:[チェックボックス]
         E8:「新規区分」  F8:[dropdown] 保険新規/自費直新規/再来
         G8:「メニューコード」H8:[テキスト入力]  ※将来: menu_id
```

> **H7 の会計合計** は GAS の `setupSelfPayValidation_V3_` が自動設置（手動設置不要）:
> `=IF(F7="",E3,E3+F7)` — E3=窓口負担額（E2=来院合計なので注意）。自費なし時は E3 のみ表示、自費あり時は合計表示

### E列（表示エリア）の扱い

| セル | 現状 | 変更後 |
|---|---|---|
| E2 | 来院合計（保険） | **変更なし** — 保険算定の来院合計を表示し続ける |
| E3 | 窓口負担額 | **変更なし** |
| E4 | 保険請求額 | **変更なし** |
| E5 | 要確認フラグ | **変更なし** + 条件付き書式追加（TRUE時: 背景橙） |
| E6 | 要確認理由 | **変更なし** + 条件付き書式追加（内容あり時: 橙） |

**要確認/要確認理由の扱い決定:**
→ **E5:E6 に保持する。削除・移動しない。**
理由: 保険算定コンプライアンスの最重要表示。文字を小さく/グレーにするのではなく、
`要確認=TRUE` のとき条件付き書式で**橙背景** にすることで「ふだんは目立たず、問題あり時に強調」を実現する。
GASコード変更不要（Sheets 条件付き書式のみ）。

## 設置手順（GAS 1回実行で完了）

> **手動でラベル・チェックボックス・数式を設置する必要はない。**
> GASメニューを1回実行すればすべて自動生成される。

| 手順 | 操作 |
|---|---|
| 1 | 患者画面を開く |
| 2 | メニュー「柔整システム」→「**会計ブロック自動生成（患者画面 行7〜8）**」を実行 |
| 3 | ダイアログ「完了しました」を確認 |
| 4 | 行7〜8 にラベル・プルダウン・チェックボックス・会計合計（H7）が生成されていることを確認 |
| 5 | E5 に条件付き書式設定: `=E5=TRUE` → 背景橙、文字黒 ← **これだけ手動** |

> GAS が自動生成する内容:
> - A7〜H7 / A8〜H8 のラベル文字・背景色・太字
> - B7 / D7 の プルダウン検証
> - B8 / D8 のチェックボックス
> - F8 の新規区分プルダウン
> - H7 の会計合計数式 `=IF(F7="",E3,E3+F7)`（E3=窓口負担額）
> - A7:H8 の外枠ボーダー
> - 旧残骸クリア: B9 / D9 / F9 の FALSE 残留（旧チェックボックス）を自動削除
> - 旧残骸クリア: A55:H62（Phase0初回設置時の旧会計ブロック）を自動削除
