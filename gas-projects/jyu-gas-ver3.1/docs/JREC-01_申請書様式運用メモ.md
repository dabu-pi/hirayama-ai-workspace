## 2. 継続月数・頻回の制度定義（D2保留理由）

### 2-1. 継続月数の制度定義

「継続月数」は **単純な受傷日からの経過月数ではない**。

| 項目 | 内容 |
|---|---|
| 正しい定義 | 「初検日を含む月（初検日が16日以降の場合は翌月）以降、**1月に10回以上の施術が連続した月数**」 |
| 誤った解釈 | 受傷日または初検日から当月末日までの単純経過月数（`calcMonthsElapsed_V3_` の戻り値をそのまま使うだけ） |
| 起算ルール | 初検日が1〜15日 → その月から起算 / 初検日が16〜31日 → 翌月から起算 |
| 連続要件 | 月10回未満の月が1ヶ月でも入れば連続が途切れる |

### 2-2. 頻回の制度定義

「頻回」は **単なる「月○回」の表示ではない**。

| 項目 | 内容 |
|---|---|
| 制度上の意味 | 長期かつ頻回施術（月10回以上×5ヶ月連続）に対する逓減率50%適用の判定基準の一部 |
| §12との関係 | `calcLongTermCoef_V3_` の長期頻回判定と整合が必要 |
| 申請書上の記載 | M31「経過」欄に記載するかどうか含め、公式記載要領の再確認が必要 |

### 2-3. M31「経過」欄への書込案の保留理由

| 問題点 | 内容 |
|---|---|
| 制度根拠が弱い | `○ヶ月 月○回` を経過欄に書く公式記載要領上の根拠が未確認 |
| 継続月数の計算が未整合 | `calcMonthsElapsed_V3_` ベース案は「連続月数」を正確に計算しない |
| 書込内容が未確定 | 何を・どのように記載すべきかが整理できていない |

> **結論: D2（継続月数・頻回）は制度定義の再確認が完了するまで実装しない。**
> INJURY_ROWS への `contMonths`/`freqDays` 追加は廃止確定。M31 書込方式は維持予定だが書込内容は保留。

## 4. 申請書欄 公式定義整理（2026-03-20）

> 調査方針: 厚生労働省通知・国保連記載要領を一次情報として優先。

> 根拠の強さを各欄に明記する。

### U1 — 都道府県番号

| 項目 | 内容 |
|---|---|
| 欄の意味 | 施術所（施術機関）が所在する都道府県を示す番号 |
| 桁数 | 2桁 |
| 記載内容 | 全国共通の都道府県コード（01=北海道、13=東京、14=神奈川… 47=沖縄） |
| 注意 | **保険者の都道府県ではなく、施術機関の所在都道府県を記載する** |

### U4 — 単併区分

| 項目 | 内容 |
|---|---|
| 欄の意味 | 申請書1枚に対して何件の給付が重なるかを示す区分 |
| 取りうる値 | **単独** / **2件** / **3件** |
| 定義 | 「単独」= 健保（または国保）単独での請求。「2件」= 健保＋地方単独公費1件。「3件」= 健保＋地方単独公費2件 |
| 地方単独事業の例 | 重度障害者医療費助成・ひとり親家庭等医療費助成・小児医療費助成 等 |
| 判定の考え方 | 同一申請書上で都道府県・市町村の地方単独事業との公費併用件数を数える。公費なし=単独、公費1件=2件、公費2件=3件 |
| JREC-01での正本候補 | **当面は固定「単独」で対応**（公費併用患者がいない場合）。将来の公費対応時は患者マスタへの「公費区分」フィールド追加が必要 |
| 実装方針 | 設定シートまたは患者マスタから取得。現時点では固定値「単独」を記載 |
| 将来設計 | 公費併用を吸収するため、単独/2件/3件を独立フィールドとして保持する方針を維持する |
| 根拠資料 | 神奈川県国保連記載要領（令和3年12月）スニペット確認 / 一次PDFは未取得 |
| 根拠の強さ | ★★☆ 二次情報（国保連記載要領参照。厚労省通知原文の直接確認は未達成） |

### U6 — 給付割合

| 項目 | 内容 |
|---|---|
| 欄の意味 | 保険者が負担する割合（療養費のうち何割を保険者が支払うか） |
| 取りうる値 | **7** / **8** / **9**（単位: 割） |
| 一部負担金割合との関係 | 給付割合＋一部負担金割合＝10割。例: 給付7割=一部負担3割 |
| 申請書への記載 | **申請書には一部負担金割合ではなく、給付割合（7/8/9）を記載する** |
| 根拠の強さ | ★★☆ 二次情報（国保連記載要領参照） |

**給付割合 対応表:**

| 給付割合 | 一部負担金割合 | 対象 |
|---|---|---|
| 9割 | 1割 | 後期高齢者（一般・低所得）/ 乳幼児・就学前（地域によっては8〜9割の公費追加あり） |
| 8割 | 2割 | 前期高齢者（70〜74歳）一般・低所得「高一」/ 後期高齢者（令和4年10月〜 2割負担化対象者） |
| 7割 | 3割 | 70歳未満（本人・家族）/ 現役並所得高齢者（高七・後期現役並） |

**JREC-01での正本候補:**

| 取得元 | 内容 |
|---|---|
| `V3TR_pickBurdenDigit_` の戻り値 | 患者マスタの `一部負担金割合` (1/2/3) から逆算で 9/8/7 を導出 |
| 計算式 | `給付割合 = 10 − 一部負担金割合`（1割負担→9割給付、2割→8割、3割→7割） |
| py 側の現状 | `CELL_MAP["給付割合"]` がない。セル番地未確認。書込処理なし |

### U7 — 請求区分（新規・継続）

| 項目 | 内容 |
|---|---|
| 欄の意味 | 当該申請書が初回（新規）か継続月か、または同一月内の治癒再発かを示す |
| 取りうる値 | **新規** / **継続** / **新規＋継続の両方** |
| セル番地（確認済） | DH31（マージセル DH31:DV31） |
| 根拠の強さ | ★★★ 複数の国保連記載要領スニペットで内容一致 |

**判定基準（公式記載要領ベース）:**

| 状況 | 記載 |
|---|---|
| 当該月が初検月（施術開始月）| 「新規」に○ |
| 当該月が2ヶ月目以降（継続月）| 「継続」に○ |
| 同月内に治癒後、新たな負傷が発生した場合 | 「新規」と「継続」**両方に○** |

**JREC-01での判定ロジック（案）:**

```
初検年月日の年月 == 対象月 → 新規
初検年月日の年月 < 対象月  → 継続
同月内に複数case（case2がある場合）→ 両方○を検討
```

> **重要**: 「新規＋継続の両方」の判定が必要なケースがある（同月内治癒再発）。
> case2 が存在し、かつ case2 の初検月 = 対象月かつ case1 が継続の場合 → 両方○。

## 5. 設計方針まとめ（将来の公費対応を見据えて）

| 方針 | 内容 |
|---|---|
| 独立フィールド保持 | U4（単併区分）・U5（本家区分）・U6（給付割合）は、それぞれ独立したフィールドとして transferCols に持つ |
| 現在の実装 | U4=固定「単独」 / U5=続柄+保険者番号+年齢から判定（複雑） / U6=一部負担金割合から逆算 |
| 公費対応時 | 患者マスタに「公費区分」を追加し、U4の件数を変動できる設計を維持する |
| コード値管理 | U5のコード体系（2本人/6家族/8高一/0高七/6後期）は設定シートで管理しない。制度仕様のため固定定義とする |
