# JREC-01 施術者向け手順書

作成日: 2026-03-30
対象: 施術担当者（患者画面の入力・保存・施術記録が主業務）
前提: システムの初期設定は院長が完了済み

---

## ▼ この手順書でわかること

- 患者画面（スプレッドシート）の入力方法
- 新患・再来の違い
- 保険/自費の判断と入力
- 来院記録の保存方法
- 要確認フラグへの対応

---

## 1. 毎日の開始前確認

```
[ ] スプレッドシート「来店管理施術録ver3.1」を開く
[ ] 「患者画面」シートを確認する
[ ] 今日の来院予定患者を確認する
[ ] 前日以前の「要確認フラグ」（E5）が残っていないか確認する
    → 残っていれば院長に報告してから作業開始
```

## 2. 患者画面の入力手順（標準フロー）

### STEP 1: 患者を選択する

1. **B2（患者検索プルダウン）** から患者名を選ぶ
2. **C2（患者ID）** に患者IDが自動表示されることを確認する
3. 新患の場合: 受付から問診票・保険証コピーを受け取り、マスタ登録を先に行う（→ §3 を参照）

### STEP 2: 来院日を入力する

- **B4（来院日）** に今日の日付を入力する
- 形式: `2026-03-30`（YYYY-MM-DD）

### STEP 3: 会計区分を入力する

- **B7（会計区分）** のプルダウンから選択する

| 選択肢 | 使うケース |
|---|---|
| 保険のみ | 外傷性の負傷（捻挫・打撲・骨折・脱臼）のみ |
| 保険+自費 | 外傷施術 + 自費メニュー（慢性ケア等）を同日に行う |
| 自費のみ | 慢性症状・肩こり・ジム・パーソナルトレーニング等 |

> **迷ったら院長に確認する。** 保険が使えない症状を保険請求するのは不正請求になる。

### STEP 4: 保険の施術内容を入力する（保険あり来院の場合）

#### ケース1（C10 付近）に入力する

| 入力項目 | 場所 | 内容 |
|---|---|---|
| 区分 | C10 | 初検 / 後療（プルダウン）|
| 部位1 | A12 | 左肩 / 右膝 等 |
| 傷病名1 | B12 | 捻挫 / 打撲 等 |
| 受傷日 | C12 | YYYY-MM-DD |
| 治療法 | E12 | チェックボックス（冷・温・電） |
| 近接部位 | F12 | チェック（近接部位がある場合）|
| 骨折チェック | 判定欄 | 骨折の場合はチェック |

> **初検か後療か:**
> - 初検 = 今回この負傷で初めて来院した日（またはエピソードが 31 日以上空いた場合）
> - 後療 = 同じ負傷の継続来院

> **電気治療（電療）のチェックは、プレオープン中は入れない。** 機器が未設置のため。

### STEP 5: 自費明細を入力する（自費ありの場合）

1. 患者画面の **Drawing ボタン（自費明細入力）** を押す
2. ダイアログが開いたら自費メニューを選ぶ
3. 内容を確認して「保存して閉じる」を押す
4. **D7（自費メニュー）** と **F7（自費金額）** が自動更新されることを確認する

> ダイアログが開かない場合: ページを更新してから再度ボタンを押す

### STEP 6: 追加情報を入力する

| 入力項目 | 場所 | 内容 |
|---|---|---|
| 慢性候補フラグ | B8 | チェックボックス（慢性疼痛患者予備軍の場合）|
| 次回予約あり | D8 | チェックボックス（次回予約を取った場合）|
| 新規区分 | F8 | プルダウン（任意）|

### STEP 7: 保存する

1. GASメニュー「**柔整ツール**」>「**登録・計算・出力**」をクリックする
   （または画面上の保存ボタンが設置されている場合はそちら）
2. 保存完了のアラートが出ることを確認する
3. **E2（来院合計）・E3（窓口負担額）** が自動計算されることを確認する
4. **E5（要確認フラグ）** が **橙色背景** になっていないか確認する

### STEP 8: 保存後の確認

```
[ ] E2（来院合計）が 0 でない（保険ありの場合）
[ ] E3（窓口負担額）が表示されている
[ ] E5（要確認フラグ）が赤/橙になっていない
    → なっている場合 → E6（要確認理由）を読んで院長に報告
[ ] 自費ありなら H7（会計合計）が正しい
[ ] 保存後に患者画面がクリアされている（B7〜H8 の入力欄が空になる）
```

## 3. 新患マスタ登録手順

> **実施タイミング:** 受付から問診票・保険証コピーを受け取った後、初期評価前に実施する。
> 院長と連携して登録内容を確認すること。

新患の場合は来院記録の前に必ず患者マスタを登録する。

1. スプレッドシートの「**患者マスタ**」シートを開く
2. 最終行の次の行に以下を入力する

| 列 | 入力内容 | 注意点 |
|---|---|---|
| 患者ID | 英数字（例: `tanaka_hanako`）| スペースなし。ひらがな不可 |
| 氏名（漢字）| 田中 花子 | 保険証の表記に合わせる |
| 氏名（カナ）| タナカ ハナコ | 保険証の表記に合わせる |
| 生年月日 | YYYY-MM-DD | 例: 1980-04-15 |
| 性別 | 男 or 女 | |
| 保険種別 | 協会けんぽ / 国民健康保険 / 後期高齢者 等 | 保険証から転記 |
| 保険者番号 | 8桁 | 保険証から転記 |
| 記号 | 保険証の記号 | 誤記に注意 |
| 番号 | 保険証の番号 | 誤記に注意 |
| 一部負担割合 | 0.1 / 0.2 / 0.3 | 後期=0.1 / 一般=0.3 等 |
| 続柄 | 本人 / 家族 | |

3. 入力後、院長に確認を依頼する

> **保険者番号・記号・番号の転記ミスが最大のリスク。** 必ず保険証の現物と照合する。

## 4. 区分（初検/後療）の判断基準

| 状況 | 区分 | 備考 |
|---|---|---|
| 同じ負傷で初めて来院する | **初検** | |
| 同じ負傷の 2 回目以降 | **後療** | 前回から 30 日以内の継続 |
| 同じ負傷で前回から 31 日以上空いた | **初検**（再初検） | 30 日ルール → 自動判定されるが区分入力は「初検」を選ぶ |
| 別の部位・別の負傷 | **初検**（新規ケース） | |
| 治癒後に別の負傷で来院 | **初検**（新規ケース） | |

> **30 日ルール:** システムが自動で判定する。迷ったら院長に確認。

## 5. 保険/自費の判断基準

| 症状・状況 | 判定 | 根拠 |
|---|---|---|
| 捻挫・打撲（いつ・どこで・どうして が明確）| 保険 OK | 外傷性あり |
| 骨折・脱臼（医師同意あり）| 保険 OK | 外傷性あり + 同意あり |
| 骨折・脱臼（医師同意なし）| **自費のみ** | 応急手当以外は同意が必要 |
| 肩こり・腰痛（慢性・原因不明）| **自費のみ** | 療養費支給対象外 |
| 筋肉疲労・単なる疲れ | **自費のみ** | 療養費支給対象外 |
| 保険医療機関で同一負傷を治療中 | **自費のみ** | 重複算定不可 |
| 外傷 + 慢性ケアを同日施術 | 保険+自費 | 外傷分は保険、慢性分は自費 |

> **不明な場合はその日のうちに院長に確認する。** 判断を翌日に持ち越さない。

## 6. 初検時相談支援料の記録

初検日に以下の3点を患者に説明した場合、相談支援料（100円）が算定される。

説明した内容を施術録（患者画面の経過・所見欄）に簡潔に記録すること。

| 説明項目 | 内容 |
|---|---|
| ① 日常生活上の励行・禁止事項 | 安静・入浴・運動の注意点 |
| ② 施術方法の説明 | 今回の治療内容・施術手順の説明 |
| ③ 受領委任の説明 | 保険の仕組み・委任状について |

> **記録がなければ算定できない。** 説明した内容を患者画面の所見・経過欄に入力すること。

## 7. 要確認フラグへの対応

E5（要確認）は **保存後に** GASが自動セットするフラグ。保存後に橙色背景になっていたら E6（要確認理由）を読む。

| よくある要確認理由 | 意味 | 対応 |
|---|---|---|
| 当月初検料算定済みのため抑制 | 同月に別の部位で初検をすでに算定している | 院長に確認。2部位目は同月初検料が出ない場合あり |
| 同一caseKeyで再検料算定済み | 再検料はエピソード内1回のみ | 自動処理。確認のみ |
| 受傷日経過日数による算定不可 | 電療・温罨法の算定ルールに抵触 | 院長に確認 |
| 長期継続理由書が必要 | 3か月超で継続書が必要な可能性 | 院長に相談 |
| 近接部位のため算定制限 | 近接した部位は算定制限あり | 院長に確認 |

**原則: 要確認フラグが立ったら院長に報告してから会計に進む。**

## 8. 施術録への記録義務（法的要件）

制度上、施術録には以下を必ず記録すること（5年間保存義務）。

```
必須記録事項:
[ ] 受給資格情報（保険種別・記号番号・保険者番号）
[ ] 氏名・性別・生年月日・続柄
[ ] 一部負担割合
[ ] 負傷年月日・時間・原因（いつ・どこで・どうして）
[ ] 負傷の状況・程度・症状
[ ] 負傷名（捻挫・打撲 等）
[ ] 初検年月日・施術終了年月日
[ ] 転帰（治癒・中止・転医）
[ ] 施術回数
[ ] 骨折・脱臼の場合は同意医師氏名・同意年月日
[ ] 施術内容・経過
[ ] 初検時相談支援の内容（初検日のみ）
[ ] 施術明細（初検料・後療料・罨法料・電療料 等）
```

> **現在の JREC-01 システムは、上記の多くを来院ヘッダ・施術明細・来院ケースに自動記録する。**
> ただし「負傷の原因・経過」の自由記述部分は患者画面の所見・経過欄に手入力が必要。

---

## 9. プレオープン中の特別ルール

1. **電気治療（電療）は施術・算定しない。** 機器未設置のため。
2. **院長が確認するまで申請書は生成しない。**
3. **新患はマスタ登録を院長が確認してから来院記録を保存する。**
4. **判断に迷う症例はその場で院長に確認する。翌日に持ち越さない。**
5. **保存後にエラーが出た場合、院長に即報告する。**

---

## 10. よくある操作ミスと対処

| ミス | 対処 |
|---|---|
| 保存後にデータが消えた | 「来院ヘッダ」シートを確認する。保存完了なら来院ヘッダに行が追加されている |
| 同日二重登録エラーが出た | その患者はすでに当日登録済み。「来院ヘッダ」で確認し、修正が必要なら院長に相談 |
| 患者名が B2 のプルダウンに出ない | 患者マスタにまだ登録されていない。マスタ登録してから再試行 |
| E3（窓口負担額）が 0 になる | 保険ありの来院で 0 は異常。要確認フラグも確認し、院長に報告 |
| 自費ダイアログが「読み込み中」のまま止まる | ページを更新して再度開く。改善しなければ院長に報告 |
| 電療チェックを入れてしまった | 保存前なら外す。保存してしまった場合は院長に報告し、修正してもらう |
