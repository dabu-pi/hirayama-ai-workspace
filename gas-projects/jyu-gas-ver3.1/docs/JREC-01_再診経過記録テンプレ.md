# 再診時経過記録テンプレ（JREC-01）

作成日: 2026-03-30
用途: 再診（2回目以降の来院）時に施術者が毎回記録する。
記入後: 患者画面（JREC-01）に入力する際の参照元として使う。
保管: 施術録として5年間保管義務あり。

## 6. 患者画面（JREC-01）入力チェックリスト

```
[ ] 来院日（B4）を入力した
[ ] 区分（C10）を選んだ（後療）
[ ] 部位・傷病名を確認した（変更なければ前回を維持）
[ ] 治療法チェックを入力した（電療は除く）
[ ] 会計区分（B7）を選んだ
[ ] 自費ありなら自費明細ダイアログで入力した
[ ] 保存（登録・計算・出力）を押した
[ ] E5（要確認フラグ）が橙色でないことを確認した
[ ] E3（窓口負担額）が表示されていることを確認した
```

## 転帰（施術終了）の記録

施術を終了する場合（治癒・中止・転医）は以下も記録する。

```
終了日: ___年___月___日
転帰: □ 治癒　□ 中止（理由: ___________________）　□ 転医（紹介先: ___________________）

総施術回数: _____ 回　　総施術期間: ___年___月___日 〜 ___年___月___日

JREC-01 患者画面での操作:
→ 来院ケースの「終了日」セルに終了日を入力する（これでシステム上もエピソード終了）
```
