### プレオープン中の制限事項

| 制限 | 内容 |
|---|---|
| 電気治療 | 機器未選定のため施術不可。メニュー `INS_OPTION_ELECTRO15` は選ばない |
| 請求前確認 | プレオープン中はすべての来院記録を院長が目視確認してから申請書生成 |
| 骨折・脱臼 | 医師同意が必要。同意なしの場合は保険算定不可（自費のみ）|

## 1. 全体フロー（1来院の標準フロー）

> **2026-03-30 担当役割正本:**
> 受付 = マスタ登録 / UI入力（B2/B4）/ 会計 　　院長 = 施術内容 / 保存 / E5確認

```
【来院】
  ↓
①受付（受付担当）
  ├─ [新患]
  │   問診票を渡す → 記入を待つ → 問診票回収・記入漏れ確認（6項目）
  │   → 保険証の確認・コピー取得（有効期限確認）
  │   → 患者マスタ登録（保険証コピーから転記 — 記号番号照合必須）
  │   → UI画面：患者ID（B2）と来院日（B4）を入力
  │   → 問診票＋保険証コピーを院長へ引き継ぐ
  └─ [再来]
      患者氏名確認 → 保険証変更がないか確認（月初は必ず）
      → UI画面：患者ID（B2）と来院日（B4）を入力
      → 院長へ引き継ぐ（今日の訴えを伝える）
  ↓
②問診・初期評価（院長）
  ├─ [新患] 問診票確認 → 初期評価テンプレ記入（赤旗チェック含む）
  │          → 保険/自費の一次判定
  └─ [再来] 前回記録確認 → 経過確認
  ↓
③保険/自費 判定（院長）
  └─ 保険: 外傷性あり（捻挫・打撲・骨折・脱臼）
  └─ 自費: 慢性症状・予防・ジム
  └─ 混在: 外傷+慢性ケアを同日施術
  ↓
④患者画面入力（院長）
  └─ B2/B4 は受付が入力済みであることを確認
  └─ 区分選択（初検/後療）→ 部位・傷病名・治療法 入力
  └─ 会計区分 入力（B7）
  └─ 自費メニューあり → 自費明細ダイアログで入力
  ↓
⑤保存（院長 → GASが自動算定）
  └─ 「登録・計算・出力」ボタンを押す
  └─ 来院合計・窓口負担額が自動表示される
  ↓
⑥E5確認（院長）
  └─ 保存後に E5（要確認）が橙背景になっていないか確認
  └─ 橙背景の場合は E6（要確認理由）を読み、自分で対処してから会計へ
  ↓
⑦会計・窓口徴収（受付）
  └─ 窓口負担額（E3）を患者から徴収
  └─ 自費ありなら H7（会計合計）を徴収
  └─ D8「次回予約」チェックボックスを確認・入力
  ↓
⑧次回案内（受付）
  └─ 次回予約の有無を確認・記録
```

## 3. 保険/自費 判定フロー

```
患者の訴え
  ↓
【外傷性か？】
  YES → 骨折・脱臼・打撲・捻挫 が明確か？
        YES → 保険算定対象
               → 骨折・脱臼は医師同意が必要
        NO  → 自費のみ
  NO  → 慢性症状・肩こり・筋疲労など
        → 自費のみ

【保険+自費の同日施術の場合】
  → 会計区分「保険+自費」を選択
  → 保険部分は通常通り算定
  → 自費部分は自費明細ダイアログで入力

【判定に迷う場合】→ 院長に確認。推測で保険請求しない
```

## 4. 新患マスタ登録手順（施術者・院長が実施）

> **実施タイミング:** 問診票回収・保険証コピー取得後、初期評価前に院長が登録する。
> 受付者はマスタ登録を行わない。

1. スプレッドシートを開く
2. 「患者マスタ」シートを開く
3. 以下の項目を入力する（問診票・保険証コピーから転記）

| 列 | 入力内容 | 必須 |
|---|---|---|
| 患者ID | 英数字 / 例: `tanaka_hanako` | ✅ |
| 氏名（漢字）| 田中 花子 | ✅ |
| 氏名（カナ）| タナカ ハナコ | ✅ |
| 生年月日 | YYYY-MM-DD | ✅ |
| 性別 | 男 / 女 | ✅ |
| 保険種別 | 協会けんぽ / 国民健康保険 / 後期高齢者 等 | ✅ |
| 保険者番号 | 8桁 | ✅ |
| 記号番号 | 保険証の記号・番号 | ✅ |
| 一部負担割合 | 0.1 / 0.2 / 0.3 のいずれか | ✅ |
| 続柄 | 本人 / 家族 | ✅ |

> **保険証は必ずコピーを取る。** 記号・番号の転記ミスが返戻の最大原因。

## 5. 月末・請求前確認（院長必須）

詳細フローは `docs/JREC-01_月次運用フロー.md` を参照。

プレオープン中に追加で必要な確認:

| 確認項目 | 内容 |
|---|---|
| 要確認フラグ | 来院ヘッダで E5（要確認）= TRUE の行が残っていないか |
| 新患の同意 | 初検時相談支援料（100円）の算定漏れがないか |
| 保険/自費の分類 | 自費のみ来院に保険算定が混入していないか |
| 骨折・脱臼の同意 | 医師同意書が取得されているか |
| 電気治療 | プレオープン中は電療算定がゼロであることを確認 |

## 6. 事故ポイント・危険箇所

| リスク | 具体的な事故 | 対策 |
|---|---|---|
| **保険請求対象外の施術を保険請求** | 肩こり・慢性症状を外傷扱いで請求 → 不正請求 | 新患ごとに負傷原因を確認。「いつ・どこで・どうして」を必ず記録 |
| **保険者情報の入力ミス** | 記号番号ミス → 全件返戻 | 保険証コピーと照合。登録後に院長が目視確認 |
| **骨折・脱臼の医師同意なし請求** | 同意なしで請求 → 返戻・不正請求 | 同意書受領前は「自費のみ」で処理。院長に確認してから変更 |
| **要確認フラグ放置** | 算定ミスを見逃す | 保存後に E5 が橙背景になっていたら、会計・申請書生成に進まず院長確認 |
| **電療メニューを誤選択** | 電療機器なしで算定 | プレオープン中 `INS_OPTION_ELECTRO15` は選ばない（メニューに出るが使用禁止）|
| **旧カルテに入力** | 旧「整骨院 電子カルテ」に入力してしまう | 毎回「来店管理施術録ver3.1」を使用する。URL・ブックマークで区別 |
| **同日二重登録** | 同じ患者を同じ日に2回保存 | GASが自動検出してエラー停止するが、エラーメッセージを読んで対処 |

---

## 7. プレオープン中の必須ルール

1. **院長確認なしに申請書を生成しない**
2. **電気治療メニューは選ばない**
3. **保険請求に迷ったら自費として処理し、後から院長確認**
4. **新患は必ず問診票を記入してからマスタ登録**
5. **保存後に E5（要確認）が橙背景になっていたら即座に院長へ報告し、会計・申請書に進まない**
