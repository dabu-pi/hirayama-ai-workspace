# JREC-01 受付者向け手順書

作成日: 2026-03-30
対象: 受付担当者（接骨院での受付・会計業務）
前提: システムの設定は院長が完了済み。受付者はスプレッドシートの読み取りと会計徴収が主業務。

---

## ▼ この手順書でわかること

- 来院時に何を確認するか
- 新患と再来の違い
- 会計金額の見方と徴収
- 次回案内の仕方
- 困ったときの対応

---

## 1. 来院時の確認（全患者共通）

### チェックリスト

```
[ ] 患者の氏名を確認する
[ ] 新患か再来かを確認する
[ ] 保険証を確認する（月初・保険証変更時は必ずコピーを取る）
[ ] 今日の負傷・訴えを簡単にメモする（施術者に引き継ぐ）
[ ] 問診票（新患の場合）を渡す
```

---

## 2. 新患の受付手順

### STEP 1: 問診票を渡す

1. `docs/JREC-01_新患問診票テンプレ.md` の問診票を印刷して渡す
2. 「受付カウンターで記入をお願いします」と伝える
3. 記入済みの問診票を受け取り、記入漏れがないか確認する

```
確認項目:
[ ] 氏名（漢字・カナ）
[ ] 生年月日
[ ] 連絡先（電話番号）
[ ] 保険証情報（種類・記号・番号）
[ ] 負傷した日時・原因
[ ] 現在の症状・部位
```

### STEP 2: 保険証の確認・コピー

1. 保険証を受け取る
2. 有効期限を確認する（期限切れなら確認が必要）
3. 保険証のコピーを取る
4. 院長または施術者に問診票・保険証コピーを渡す

### STEP 3: 患者マスタ登録（院長・施術者が実施）

> 受付者はマスタ登録を行わない。問診票・保険証コピーを渡して施術者に依頼する。

---

## 3. 再来患者の受付手順

### STEP 1: 本日の状況確認

1. 「本日はどちらの部位ですか？」と確認する
2. 保険証が変わっていないか確認する
   - 月初は毎月確認する
   - 「保険証はお変わりありませんか？」と聞く
3. 変更があれば院長に報告して、マスタ更新してもらう

### STEP 2: 施術者への引き継ぎ

1. 患者の氏名と来院目的を施術者に伝える
2. 保険証の変更があれば伝える
3. 以降は施術者が対応する

---

## 4. 会計の手順

### 4-1. 金額の確認方法

施術者が来院記録を保存すると、患者画面に自動で金額が表示される。

| 見る場所 | 内容 |
|---|---|
| スプレッドシートの「患者画面」| 施術者が入力・保存したシート |
| **E3（窓口負担額）** | **患者から徴収する保険分の金額** |
| **H7（会計合計）** | **保険+自費合計（自費ありの場合はこちら）** |
| F7（自費金額） | 自費のみの金額 |
| E2（来院合計） | 保険算定合計（参考）|

> **会計で徴収する金額:**
> - 保険のみ → **E3（窓口負担額）**
> - 保険+自費 → **H7（会計合計）**
> - 自費のみ → **F7（自費金額）**

### 4-2. 患者画面の開き方

1. 来院管理施術録ver3.1 を開く
2. 「患者画面」シートをクリックする
3. B2 のプルダウンで患者名を選ぶ（施術者がすでに選んでいれば確認のみ）
4. E3（または H7）の金額を確認する

> **施術者が保存した後に確認する。** 保存前は金額が0になる場合があります。

### 4-3. お会計の伝え方

```
保険のみの場合:
「本日の窓口負担は ○○ 円です。」

保険+自費の場合:
「本日は保険分 ○○ 円と自費分 ○○ 円で合計 ○○ 円です。」

自費のみの場合:
「本日の施術料は ○○ 円です。」
```

### 4-4. 領収書について

> ⚠️ 現時点では領収書の自動発行機能はシステムにありません（仮説）。
> 院長から指示があった場合は手書き領収書を発行してください。

## 5. 次回予約・案内

### 次回予約の確認

1. 施術後に「次回のご予約はどうしますか？」と確認する
2. 予約する場合は予約帳（またはカレンダー）に記録する
3. 「次回予約あり」の場合は施術者に伝える（患者画面 D8 にチェックを入れてもらう）

### 次回来院の目安を伝える

```
施術者から指示があれば：
「次回は○日後（○月○日頃）にいらしてください。」

指示がない場合は施術者に確認してから伝える。
```

---

## 6. 困ったときの対応

| 状況 | 対応 |
|---|---|
| 保険証を忘れた | 当日は確認できないことを伝え、次回持参をお願いする。保険情報が不明のまま保険請求はできないため院長に相談 |
| 金額に疑問がある | 院長または施術者に確認する。患者に「確認してから連絡します」と伝える |
| E3 や H7 が 0 になっている | 施術者がまだ保存していない可能性がある。施術者に確認する |
| 要確認フラグ（E5）が橙背景で光っている | 院長に報告する（会計処理は院長確認後）|
| 新患で問診票記入漏れがある | 施術者に確認し、記入を依頼する。特に保険証情報・負傷原因は必須 |
| 患者から「電気治療はできますか？」と聞かれた | 「現在機器の準備中のため、もう少々お待ちください」と伝える |

---

## 7. プレオープン中の特別ルール

1. **電気治療はできない**（機器未設置）。「現在準備中です」と案内する
2. **お会計の金額は院長に確認を取ってから徴収する**（プレオープン中のみ）
3. 不明な点はすべて院長に確認する。推測で対応しない
