# JREC-01 受付者向け手順書

作成日: 2026-03-30
対象: 受付担当者（接骨院での受付・会計業務）
前提: システムの設定は院長が完了済み。受付者はスプレッドシートの読み取りと会計徴収が主業務。

---

## ▼ この手順書でわかること

- 来院時に何を確認するか
- 新患と再来の違い
- 会計金額の見方と徴収
- 次回案内の仕方
- 困ったときの対応

---

## 1. 来院時の確認（全患者共通）

### チェックリスト

```
[ ] 患者の氏名を確認する
[ ] 新患か再来かを確認する
[ ] 保険証を確認する（月初・保険証変更時は必ずコピーを取る）
[ ] 今日の負傷・訴えを簡単にメモする（施術者に引き継ぐ）
[ ] 問診票（新患の場合）を渡す
```

---

## 2. 新患の受付手順

> **受付者の役割（2026-03-30 正本）:**
> 問診票配布・回収・確認 → 保険証コピー取得 → **患者マスタ登録** → **UI画面（B2/B4）入力** → 院長へ引き継ぎ

### STEP 1: 問診票を渡す

1. 「`JREC-01_新患問診票`（印刷済み）」を渡す
2. 「受付カウンターで記入をお願いします」と伝える
3. 記入済みの問診票を受け取り、記入漏れがないか確認する

```
確認項目:
[ ] 氏名（漢字・カナ）
[ ] 生年月日
[ ] 連絡先（電話番号）
[ ] 保険証情報（種類・記号・番号）
[ ] 負傷した日時・原因（いつ・どこで・どうして）
[ ] 現在の症状・部位
```

### STEP 2: 保険証の確認・コピー

1. 保険証を受け取る
2. 有効期限を確認する（期限切れなら院長に相談）
3. 保険証のコピーを取る

### STEP 3: 患者マスタ登録（受付が実施）

> **受付者が患者マスタに登録する。** 保険証コピーと問診票を見ながら以下を入力する。

1. スプレッドシートの「**患者マスタ**」シートを開く
2. 最終行の次の行に以下を入力する

| 列 | 入力内容 | 必須 |
|---|---|---|
| 患者ID | 英数字（例: `tanaka_hanako`）スペース・ひらがな不可 | ✅ |
| 氏名（漢字）| 保険証の表記に合わせる | ✅ |
| 氏名（カナ）| 保険証の表記に合わせる | ✅ |
| 生年月日 | YYYY-MM-DD 形式 | ✅ |
| 性別 | 男 / 女 | ✅ |
| 保険種別 | 協会けんぽ / 国民健康保険 / 後期高齢者 等 | ✅ |
| 保険者番号 | 8桁（保険証から転記）| ✅ |
| 記号 | 保険証の記号（誤記注意）| ✅ |
| 番号 | 保険証の番号（誤記注意）| ✅ |
| 一部負担割合 | 0.1 / 0.2 / 0.3 | ✅ |
| 続柄 | 本人 / 家族 | ✅ |

> ⚠️ **保険者番号・記号・番号の転記ミスが返戻の最大原因。** 必ず保険証の現物と照合すること。
> 入力後に院長へ確認を依頼する。

### STEP 4: UI画面で患者IDと来院日を入力（受付が実施）

1. 「来店管理施術録ver3.1」を開く
2. 「患者画面」シートを選ぶ
3. **B2（患者検索プルダウン）** から今日の患者名を選ぶ
4. **B4（来院日）** に今日の日付を入力する（例: `2026-03-30`）
5. **それ以外の項目（会計区分・施術内容等）は入力しない** — 院長が担当

> B2・B4 の入力だけ行い、「登録・計算・出力」ボタンは**押さない**。保存は院長が実施する。

### STEP 5: 問診票＋保険証コピーを院長へ引き継ぐ

1. 問診票と保険証コピーをセットにして院長へ渡す
2. 「患者ID: ○○○ をB2/B4に入力しました」と一言伝える
3. 以降の施術内容入力・保存は院長が担当する

## 3. 再来患者の受付手順

### STEP 1: 本日の状況確認

1. 「本日はどちらの部位ですか？」と確認する
2. 保険証が変わっていないか確認する
   - 月初は毎月確認する
   - 「保険証はお変わりありませんか？」と聞く
3. 変更があれば院長に報告して、マスタ更新してもらう

### STEP 2: UI画面で患者IDと来院日を入力（受付が実施）

1. 「来店管理施術録ver3.1」を開く
2. 「患者画面」シートを選ぶ
3. **B2（患者検索プルダウン）** から患者名を選ぶ
4. **B4（来院日）** に今日の日付を入力する（例: `2026-03-30`）
5. **それ以外は入力しない** — 施術内容・保存は院長が担当

### STEP 3: 院長への引き継ぎ

1. 患者の氏名・今日の訴えを院長に伝える
2. 保険証の変更があれば伝える
3. 「患者ID: ○○○ をB2/B4に入力しました」と一言添える
4. 以降の施術内容確認・修正・保存は院長が担当する

## 4. 会計の手順

### 4-1. 金額の確認方法

施術者が来院記録を保存すると、患者画面に自動で金額が表示される。

| 見る場所 | 内容 |
|---|---|
| スプレッドシートの「患者画面」| 施術者が入力・保存したシート |
| **E3（窓口負担額）** | **患者から徴収する保険分の金額** |
| **H7（会計合計）** | **保険+自費合計（自費ありの場合はこちら）** |
| F7（自費金額） | 自費のみの金額 |
| E2（来院合計） | 保険算定合計（参考）|

> **会計で徴収する金額:**
> - 保険のみ → **E3（窓口負担額）**
> - 保険+自費 → **H7（会計合計）**
> - 自費のみ → **F7（自費金額）**

### 4-2. 患者画面の開き方

1. 来院管理施術録ver3.1 を開く
2. 「患者画面」シートをクリックする
3. B2 のプルダウンで患者名を選ぶ（施術者がすでに選んでいれば確認のみ）
4. E3（または H7）の金額を確認する

> **施術者が保存した後に確認する。** 保存前は金額が0になる場合があります。

### 4-3. お会計の伝え方

```
保険のみの場合:
「本日の窓口負担は ○○ 円です。」

保険+自費の場合:
「本日は保険分 ○○ 円と自費分 ○○ 円で合計 ○○ 円です。」

自費のみの場合:
「本日の施術料は ○○ 円です。」
```

### 4-4. 領収書について

> **領収書の自動発行機能はシステムに未実装（確認済み）。**
> プレオープン中は手書き領収書で対応する。領収書の書式・用紙は院長が用意する。
> 「領収書をください」と言われたら院長に確認してから発行すること。

## 5. 次回予約・案内

### 次回予約の確認

1. 施術後に「次回のご予約はどうしますか？」と確認する
2. 予約する場合は予約帳（またはカレンダー）に記録する
3. 「次回予約あり」の場合は施術者に伝える（患者画面 D8 にチェックを入れてもらう）

### 次回来院の目安を伝える

```
施術者から指示があれば：
「次回は○日後（○月○日頃）にいらしてください。」

指示がない場合は施術者に確認してから伝える。
```

## 6. 困ったときの対応

| 状況 | 対応 |
|---|---|
| 保険証を忘れた | 当日は確認できないことを伝え、次回持参をお願いする。保険情報が不明のまま保険請求はできないため院長に相談 |
| 金額に疑問がある | 院長または施術者に確認する。患者に「確認してから連絡します」と伝える |
| E3 や H7 が 0 になっている | 施術者がまだ保存していない可能性がある。施術者に確認する |
| 要確認フラグ（E5）が橙背景で光っている | 施術者が保存した後に出る警告。院長に報告し、院長確認が終わるまで会計に進まない |
| 新患で問診票記入漏れがある | 施術者に確認し、記入を依頼する。特に保険証情報・負傷原因は必須 |
| 患者から「電気治療はできますか？」と聞かれた | 「現在機器の準備中のため、もう少々お待ちください」と伝える |

---

## 7. プレオープン中の特別ルール

1. **電気治療はできない**（機器未設置）。「現在準備中です」と案内する
2. **お会計の金額は院長に確認を取ってから徴収する**（プレオープン中のみ）
3. 不明な点はすべて院長に確認する。推測で対応しない
