# 廃棄物日報システム SPEC（HAIKI-05）

最終更新: 2026-05-04  
仕様バージョン: v2026-02-19h（確定版・統合版） ← シート内「仕様書」シートと一致

---

## システム目的

収集運搬一覧（入力UI）→ 日報（帳票）→ 月報（提出用）→ 年度積算（内部管理）→ PDF出力 → 提出管理

**最優先方針:** 提出事故ゼロ（曖昧・不整合は必ず停止）

---

## シート構成（確定・稼働中）

| シート名 | 役割 |
|---|---|
| 収集運搬一覧 | 入力UI。日付・排出者・住所・車種・車番・燃やすごみ・不燃ごみ・資源ごみ |
| 日報 | 帳票テンプレ（原本）。作業用は 日報_YYYY-MM |
| 月報 | 提出用テンプレ（原本）。作業用は 月報_YYYY-MM |
| 仕様書 | このファイルの正本（シート内に埋め込み済み） |
| 提出済み確認 | 月別提出ログ（ステータス / PDFリンク / 生成日時） |
| 設定 | 年度選択（現在: 2026） |
| 年度積算_YYYY-MM月分 | 年度積算帳票（月単位で自動生成）。現在 02・03・04月分が存在 |

---

## 用語定義

| 用語 | 定義 |
|---|---|
| UIレコード | 収集運搬一覧の1行 |
| 案件（Case） | 対象月内の「排出者氏名 + 排出先住所」で定義。日付は含まない |

## 日報仕様

### テンプレ固定
- 原本は直接編集しない
- 作業用シート名: `日報_YYYY-MM`

### ヘッダー
| セル | 内容 |
|---|---|
| K4:AO4 | YYYY年 M月 |
| K5:AO5 | 1〜31 |
| K6:AO6 | 曜日 |

### 案件ブロック
1案件 = 4行

| 行 | 内容 |
|---|---|
| 1 | 燃やすごみ |
| 2 | 不燃ごみ |
| 3 | 資源ごみ |
| 4 | 予備 |

### 左側情報
| 列 | 内容 |
|---|---|
| E | 排出者氏名（4行結合） |
| F | 排出先住所（4行結合） |
| I | 車種 / 車番 |

### AP列（重要）
AP列 = SUM(K〜AO)  
AP(燃やす) / AP(不燃) / AP(資源) → 月報の転記元

## 単月月報仕様（提出用）

### テンプレ
- 作業用シート名: `月報_YYYY-MM`
- 転記元: `日報_YYYY-MM`

### 月報レイアウト（月列）
| 列 | 月 |
|---|---|
| B | 4月 |
| C | 5月 |
| D | 6月 |
| E | 7月 |
| F | 8月 |
| G | 9月 |
| H | 10月 |
| I | 11月 |
| J | 12月 |
| K | 1月 |
| L | 2月 |
| M | 3月 |
| N | 年間合計 |

### 事業所表示
- A30:A47 / 1社2行
  - 上段: 資源ごみ以外（燃やす＋不燃）
  - 下段: 資源ごみ
- J4: 例 `6月分`
- A48:A49（結合）: 上段=資源ごみ以外合計 / 下段=資源ごみ合計
- PDF: 1ファイル / 保存先=設定シートのPDFフォルダ

## 年度積算仕様

| 項目 | 内容 |
|---|---|
| テンプレ | 月報原本 |
| 転記元 | 収集運搬一覧 |
| 起算月 | 4月始まり |
| 年度定義 | `year = month >= 4 ? year : year - 1` |
| 集計範囲 | YYYY-04-01 〜 (YYYY+1)-03-31 |
| J4表示例 | `2026年度（4月〜11月分）` ← 最後にデータが存在する月まで表示 |
| 事業所 | 年度内に出現した事業所のみ / 出現順 / 1事業所2行 |
| 9社超え | ページ分割（9社/ページ）。月報テンプレを複数コピー |
| ページ番号 | J3セル（例: `1/2`, `2/2`） |
| 合計 | 各ページ A48:A49 = そのページ分合計 / 最終ページ = 年度全体総合計 |
| PDF | ページごとに別PDF / 例: `年度積算_2026年度_1-2.pdf` |

---

## 停止条件

- 日報不存在（単月月報生成時）
- 数量非数値
- 日付不正
- 住所空欄
- 同一案件キーで住所複数
- 案件ブロック不足
- 排出者数 > 処理可能上限
- ステータス=SENT で再生成しようとした
- レイアウト破壊検出

---

## 処理フロー

```
【単月月報】
収集運搬一覧 → 日報生成 → 単月月報生成 → PDF出力 → 提出済み確認ログ

【年度積算】
収集運搬一覧 → 年度抽出 → 月別集計 → 9社単位ページ分割 → 年度積算帳票生成 → PDF出力（複数）
```

---

## 設計思想

- 月報は「日報確定データ」を参照
- 年度積算は「元データ直接参照」
- レイアウトは絶対に崩さない
- 提出事故ゼロ
