## 補足

- `参照しているGAS` / `参照されているGAS` は、現時点でローカルコード、ユーザー提供GASテキスト、取得できた数式から確認できたものだけ記載し、未取得分は `未取得` とした。
- 商品一覧/見積/競合GASのコード内容はテキストから確認できたが、Apps Script の `scriptId` と実トリガーはまだ未取得のため追加調査が必要。
- 調査フェーズ3では、Apps Script API が現行 `.clasprc.json` スコープ不足で `403 ACCESS_TOKEN_SCOPE_INSUFFICIENT` となり、CLIからの `scriptId` 自動回収はできないことを確認した。
- 機械処理しやすい一覧として `docs/tab-classification.csv` も併せて作成する。
