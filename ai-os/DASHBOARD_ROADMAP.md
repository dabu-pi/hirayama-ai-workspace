## STOP 条件

以下の状態になった場合は **即座に停止** し、理由を Run_Log またはローカル文書に残す。

| 状況 | アクション |
|---|---|
| 自動処理が `Run_Log` 以外のシートに書き込みそうになった | 即停止 + 書き込み先制限を見直す |
| 意図しない既存行の上書きが発生した | 即停止 + 追記専用設計へ戻す |
| 管理コストが本業プロジェクトより大きくなった | 自動化範囲を縮小する |
| スプレッドシートIDや認証情報がコミットされそうになった | 即停止 + `.gitignore` 確認 |
| 実装が複雑化し「最小自動化」を外れた | 要件を削って Phase 2 をやり直す |
