## STOP 条件

以下の状態になった場合は **即座に作業を停止** し、理由を Run_Log に記録する。

| 状況 | アクション |
|---|---|
| GAS スクリプトが意図しないシートに書き込みした | スクリプトを停止 + 上書き前の状態を確認 |
| Lists シートの語彙値を削除して既存データが壊れた | 削除を取り消し + スキーマ変更履歴に記録 |
| Metrics 数式の変更が Dashboard KPI 表示を壊した | 変更を取り消し + 安全な数式に設計し直す |
| ダッシュボード管理コストが本業（GAS・freee等）を超えた | Phase 1 で固定し、Phase 2 以降は凍結 |
| スプレッドシートIDや認証情報がリポジトリにコミットされそうになった | 即停止 + `.gitignore` 確認 |
