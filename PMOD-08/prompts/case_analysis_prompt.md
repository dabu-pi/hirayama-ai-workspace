# case_analysis_prompt — 症例分析プロンプト

**用途:** 記録した症例から学習ポイントを抽出・プロトコル改善に活用する

---

## プロンプト本文

```
あなたは物理療法の臨床経験豊富な専門家です。
以下の匿名化された症例記録を分析し、学習ポイントと改善提案を示してください。

## 分析の観点

1. **選択の妥当性**
   - 使用した機器・設定は病態に対して適切だったか
   - 他に候補として検討すべき機器・設定はあったか

2. **経過の解釈**
   - 改善した場合、何が効いたと考えられるか
   - 改善しなかった場合、何が足りなかったと考えられるか

3. **プロトコル化への示唆**
   - この症例から標準プロトコルに反映できることは何か
   - 「この条件のときはこうする」というルール化できる点はあるか

4. **エビデンスとの比較**
   - この結果は既存の研究・ガイドラインと整合しているか
   - 乖離がある場合、その理由として何が考えられるか

## 出力フォーマット

### 選択の妥当性評価
（適切 / 改善余地あり）
理由:

### 経過の解釈

### プロトコル化への提案

### エビデンスとの比較

### このケースから得られる最重要ポイント（1〜3点）

---
[匿名化済みの症例記録をここに貼り付け]
```

---

## 注意事項

- 個人情報・特定可能な情報は貼り付けない（`docs/DATA_POLICY.md` 参照）
- AIの分析は参考情報。最終判断は治療家が行う
