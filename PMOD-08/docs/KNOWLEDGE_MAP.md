# KNOWLEDGE_MAP — 物療学習の全体地図

**プロジェクトID:** PMOD-08  
**最終更新:** 2026-04-10  
**位置づけ:** 「何をどの順番でどう理解するか」の全体地図。機器・人体・病態・症状・臨床判断を一本につなぐ

---

## このドキュメントの目的

物療機器を「名前で覚える」のではなく、
**「体内で何を起こすか」で理解する**ための学習地図を提供する。

機器の名称・ブランドは変わるが、生体物理の原理は変わらない。
原理から理解すれば、未知の機器に出会ったときも本質的な判断ができる。

## 1. 物療の大分類

物療機器は「何をエネルギー源にするか」で大別できる。

| 大分類 | エネルギー | 代表的機器 |
|---|---|---|
| 電気療法 | 電気（電流・電圧・電場） | 低周波・中周波・高周波・干渉波・ハイボルテージ・EMS・TENS |
| 超音波療法 | 機械的振動（音波） | 超音波治療器（連続・パルス） |
| ショックウェーブ | 高圧衝撃波 | 体外衝撃波（ESWT） |
| 熱療法 | 熱エネルギー | ホットパック・ラジオ波（RF）・遠赤外線 |
| 光線療法 | 光（可視・赤外・レーザー） | 低出力レーザー（LLLT）・LED |
| 磁気療法 | 磁場 | 磁気治療器 |
| 機械的療法 | 力学的刺激 | 牽引・振動・圧迫 |

## 2. 電気療法の周波数別整理

電気療法は周波数によって「体内での振る舞い」が変わる。
ただし、**周波数だけで効果は決まらない**（後述）。

| 分類 | 周波数帯 | 体内での主な振る舞い |
|---|---|---|
| 直流（DC） | 0 Hz | イオントフォレシス・創傷治癒 |
| 低周波（LF） | 1〜1,000 Hz | 神経・筋の直接刺激 → 筋収縮・疼痛調整（ゲートコントロール） |
| 中周波（MF） | 1,000〜100,000 Hz | 皮膚抵抗が下がり、深部への電流透過が向上 |
| 高周波（RF） | 100,000 Hz〜 | 熱産生（ジュール熱）→ 深部温熱 |
| 干渉波（IFC） | 中周波2波の干渉により低周波帯の変調波を深部で生成 | 深部での低周波刺激効果 |
| ハイボルテージ（HV） | 低周波帯・高電圧・短パルス | 皮膚抵抗を突破して深部刺激、浮腫吸収・疼痛調整 |
| EMS | 低〜中周波 | 筋収縮誘発（運動代替・筋力増強・廃用予防） |

---

## 3. 周波数だけで決まらない理由

**周波数は「体内で何を起こすか」の一因子に過ぎない。**
実際の反応は以下の因子が複合的に作用する。

### 3-1. 波形（Waveform）

| 波形 | 特徴 | 用途 |
|---|---|---|
| 正弦波（Sine wave） | 滑らかな交流 | 干渉波・超音波 |
| 方形波（Square wave） | 急峻な立ち上がり → 神経刺激に有利 | 低周波・EMS |
| 三角波（Triangle wave） | ゆっくりした立ち上がり → 適応しやすい | 廃用筋への慣らし |
| スパイク波 | 非常に短い高電圧パルス | ハイボルテージ |

### 3-2. パルス幅（Pulse width / Phase duration）

- **パルス幅が広い** → 神経・筋への充分な電荷で収縮しやすい
- **パルス幅が狭い** → 刺激は弱く、感覚レベルの刺激（TENS的）に近づく
- 目安：感覚刺激 < 100μs、筋収縮誘発 > 200μs

### 3-3. 電流強度（Amplitude / Intensity）

- 神経・筋の閾値を超えると収縮・興奮が起きる
- 閾値以下 → 感覚刺激（疼痛調整・末梢循環）
- 閾値以上 → 筋収縮誘発
- 「強ければ良い」ではない。患者の組織状態・感受性・目的に合わせる

### 3-4. Duty Cycle（通電率）

- Duty Cycle = ON時間 / (ON時間 + OFF時間)
- 筋収縮誘発では ON:OFF = 1:3〜1:5 程度（筋の回復時間を確保）
- 連続通電は疼痛調整や循環改善に適す
- 高い Duty Cycle での筋収縮誘発は筋疲労・過負荷のリスクあり

### 3-5. 通電時間・治療時間

- 短すぎると効果不十分、長すぎると皮膚刺激・過剰反応のリスク
- 一般的目安：局所 10〜20分 / 広域 15〜30分
- 急性炎症期は短め（5〜10分）から様子見

### 3-6. 電極配置

- **電流は電極間を最短距離で流れようとする**（ただし組織の電気抵抗で迂回する）
- 対象組織を電流経路に含めるよう電極を配置する
- 単極法（active/dispersive）vs 双極法（2電極）
- 関節をまたぐ場合は、電流が関節軟骨を通るルートを考慮する

### 3-7. 対象組織の特性

| 組織 | 電気抵抗 | 熱伝導性 | 超音波吸収 |
|---|---|---|---|
| 皮膚（角質層） | 非常に高い | 低 | 低 |
| 脂肪 | 高い | 低 | 低 |
| 筋肉 | 中程度 | 中 | 中 |
| 骨 | 非常に高い | 中 | 高（反射） |
| 腱・靭帯 | 高い | 低 | 高い吸収 |
| 神経 | 低い | — | — |

### 3-8. 患者側の因子

- **皮下脂肪の厚さ** → 深部への電流・熱の到達を妨げる
- **筋量** → EMS効果に影響。廃用筋は閾値が高くなっている
- **皮膚状態** → 乾燥・傷・かさぶたは抵抗・リスクを高める
- **感受性の個人差** → 高齢者・神経障害患者・小児は感覚が鈍化している場合あり
- **疾患・服薬** → ペースメーカー・妊娠・金属インプラント・抗凝固剤 等

## 4. 目的別整理（何を変えたいか）

### 4-1. 疼痛調整

| メカニズム | 機器・方法 | 説明 |
|---|---|---|
| ゲートコントロール（Gate theory） | TENS・低周波（高頻度） | Aβ線維の活性化で痛覚シグナルを抑制 |
| 内因性オピオイド放出 | 低周波（低頻度）・干渉波 | エンドルフィン・エンケファリン産生 |
| 中枢性感作の修飾 | — | 慢性疼痛への関与。機器で直接は難しい |
| 炎症抑制・浮腫軽減 | ハイボルテージ・超音波（パルス） | 局所炎症の軽減 → 二次的疼痛軽減 |

**設定の指針:**
- 即時的な疼痛抑制（ゲートコントロール）：高頻度（80〜150 Hz）・弱い強度
- 遅延性の疼痛抑制（オピオイド）：低頻度（1〜10 Hz）・筋収縮閾値付近

---

### 4-2. 筋収縮誘発

| 目的 | 機器 | 設定方針 |
|---|---|---|
| 廃用筋の賦活 | EMS・低周波（中頻度） | 低強度から開始、収縮を確認しながら増加 |
| 筋力増強 | EMS（高強度） | 自力収縮との同調が望ましい |
| 痙縮筋の拮抗筋刺激 | FES的使用 | 専門知識が必要 |
| 排泄・腹部筋刺激 | EMS | 専門的使用 |

**注意:** 急性炎症・腫脹・関節不安定性のある部位への強い筋収縮誘発は禁忌に近い場合あり

---

### 4-3. 温熱効果

| 分類 | 機器 | 特徴 |
|---|---|---|
| 表層温熱 | ホットパック・赤外線 | 皮膚・皮下組織（〜1cm） |
| 深部温熱 | ラジオ波（RF）・超音波（連続） | 筋・関節・腱（2〜5cm以上） |
| 選択的温熱 | 超音波（周波数による） | 1MHz：深部、3MHz：表層 |

**温熱の生体効果:**
- 局所血流増加
- 組織柔軟性向上（コラーゲン組織の粘弾性変化）
- 代謝亢進
- 神経伝導速度の変化（疼痛閾値の上昇）
- 筋痙縮の緩和

**禁忌:** 急性炎症・出血リスク・感覚障害・循環障害・悪性腫瘍部位

---

### 4-4. 循環改善

| 機器 | 主な効果 |
|---|---|
| 低周波（高頻度連続） | 筋ポンプ効果・浮腫軽減 |
| ハイボルテージ | 浮腫吸収（マイナス極）・循環促進 |
| 超音波（パルス） | 微細マッサージ効果・リンパ流促進 |
| 干渉波 | 深部循環・筋弛緩を通じた間接的促進 |

---

### 4-5. 創傷修復・組織修復

| 機器 | 効果 | 証拠レベル |
|---|---|---|
| 超音波（パルス低強度：LIPUS） | 骨折治癒促進・腱・軟部組織修復 | 中〜高（骨折で特に） |
| ショックウェーブ（ESWT） | 腱付着部・石灰化腱炎・筋膜の修復促進 | 中程度（腱障害で研究多） |
| 低出力レーザー（LLLT） | 細胞代謝活性化・創傷治癒 | 研究あり、標準化が課題 |
| 直流（DC）・高電圧直流 | 創傷治癒への影響 | 研究あり |

## 5. 「機器名ベース」から「生体反応ベース」への転換

### 機器名ベースの思考（避けるべき）

> 「肩こりには干渉波」「腰痛には低周波」
> → 機器を「固定の処方箋」として使う。なぜそれを使うか説明できない。

### 生体反応ベースの思考（目指すもの）

> 「この患者は深部の筋緊張が主訴。表層より深い組織への熱・機械的刺激が必要。
> 中周波干渉波（深部透過）またはラジオ波（深部温熱）が候補。
> 炎症の有無・皮下脂肪量・患者の感受性で最終選択する。」

### 転換のための問いかけ

1. **この患者で今、体内の何が問題になっているか？**
   → 炎症？ 筋緊張？ 神経感作？ 循環障害？ 浮腫？

2. **物療で変えられる反応は何か？**
   → 疼痛閾値を上げる？ 血流を増やす？ 筋を収縮させる？ 組織を温める？

3. **その反応を起こすには、どんなエネルギー・どんな設定が必要か？**
   → 深さ、強度、周波数、パルス幅、時間

4. **この患者の組織特性・禁忌に照らして選択肢を絞る**
   → 皮下脂肪量、感覚状態、金属、ペースメーカー 等

5. **反応を確認して次回に活かす**
   → 反応良好 → プロトコル化。反応不十分 → 条件を変える。
