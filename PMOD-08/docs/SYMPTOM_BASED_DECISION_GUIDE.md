# SYMPTOM_BASED_DECISION_GUIDE — 症状から逆算する物療選択ガイド

**プロジェクトID:** PMOD-08  
**最終更新:** 2026-04-10  
**位置づけ:** 「患者が今どんな状態か」から出発して物療を選ぶ実践ガイド

---

## 基本思想

疾患名から機器を選ぶのではなく、  
**「今この患者に何が起きているか（症状の種類と状態）」から逆算して機器と設定を選ぶ。**

同じ「腰痛」でも、急性炎症と慢性拘縮では選ぶ機器は変わる。
「症状の種類」→「狙う反応」→「候補機器」→「設定の方針」の順で考える。

## 症状タイプ別ガイド

---

## Type 1: 痛み優位（Pain-Dominant）

### 評価すること

| 評価項目 | 確認のポイント |
|---|---|
| 痛みの性質 | 鋭い・鈍い・灼熱感・ズキズキ・持続性・間欠性 |
| 発症時期 | 急性（72時間以内）/ 亜急性 / 慢性（3ヶ月以上）|
| 炎症の有無 | 発赤・熱感・腫脹・安静時痛 |
| 部位と広がり | 限局 vs 広範（広範は中枢性感作を考慮）|
| 増悪・軽快因子 | 動作 / 安静 / 時間帯 |
| VAS/NRS スコア | 0〜10 で定量化 |

### 狙う反応

| 状況 | 狙う反応 |
|---|---|
| 急性炎症期 | 炎症の管理・発痛物質の除去 |
| 亜急性〜慢性 | ゲートコントロール（即時）+ 内因性オピオイド（遅延）|
| 慢性・中枢性感作 | 疼痛教育との統合。物療は補助的な活動参加支援 |

### 候補機器と設定

| 状況 | 機器 | 周波数 | 強度 | 時間 |
|---|---|---|---|---|
| 急性炎症の疼痛 | TENS・HV | 80〜150 Hz | 感覚閾値程度 | 15〜20分 |
| 慢性疼痛（即時鎮痛）| TENS・干渉波 | 80〜150 Hz | じんわり感じる程度 | 20〜30分 |
| 慢性疼痛（持続鎮痛）| 低周波（低頻度）| 2〜4 Hz | 収縮を感じる程度 | 20〜30分 |
| 深部の筋・関節痛 | 干渉波・深部温熱 | 80〜100 Hz変調 | 中程度 | 20分 |

### 再評価ポイント

- 治療直後の VAS/NRS の変化を確認
- 翌日の痛みレベルを確認（良好 / 変化なし / 悪化）
- 悪化の場合 → 強度・時間・機器の見直し、レッドフラッグの再確認

## Type 2: 筋出力低下優位（Muscle Weakness / Disuse Dominant）

### 評価すること

| 評価項目 | 確認のポイント |
|---|---|
| 廃用の程度 | 安静・不動の期間・筋萎縮の有無 |
| 痛みの影響 | 痛みで力が入らない vs 真の筋力低下 |
| 神経障害の有無 | 神経根・末梢神経の関与 |
| 対象筋の収縮確認 | 自力での収縮が出来るか |

### 狙う反応

- 廃用筋への電気的収縮誘発 → 筋萎縮の進行抑制・収縮パターンの再学習
- 痛みを下げて自力収縮を引き出す → 運動療法へのブリッジ

### 候補機器と設定

| 目的 | 機器 | 設定方針 |
|---|---|---|
| 廃用予防・初期筋賦活 | EMS・低周波（筋収縮モード）| 20〜50 Hz・ON:OFF=1:3〜1:5・収縮が出る強度から |
| 高齢者の廃用筋 | EMS（低強度から） | 低強度で収縮確認してから漸増 |
| 痛みで力が入らない | TENS先行 → EMS追加 | 痛みを下げてから収縮誘発 |
| 術後廃用 | EMS（早期から） | 主治医の許可範囲内で |

### 注意点

- 急性炎症部位の直上への収縮誘発は禁忌
- 関節不安定性のある部位は固定した上で行う
- 廃用が強い場合は必ず自力収縮を補助する位置づけ。EMS だけに頼らない

### 再評価ポイント

- 収縮の質と量（収縮できているか・どの程度か）
- 自力収縮の変化（補助なしで動けるようになってきたか）
- 筋周囲径の変化

## Type 3: 拘縮優位（Stiffness / Contracture Dominant）

### 評価すること

| 評価項目 | 確認のポイント |
|---|---|
| 拘縮の原因 | 廃用性 / 瘢痕性 / 炎症性 |
| 可動域制限の部位 | 関節包 / 筋・筋膜 / 腱 |
| 硬さの性質 | 動き始めが一番硬い vs 動くほど硬くなる |
| 疼痛との関係 | 制限範囲内で痛みがあるか・エンドレンジで痛いか |
| 炎症の有無 | 安静時痛・熱感・腫脹があれば炎症性 |

### 狙う反応

- コラーゲン組織の粘弾性変化（43〜45°Cで最大）→ 深部温熱
- 温熱後に徒手・牽引・ストレッチを組み合わせる
- 慢性炎症性拘縮は炎症管理が先決

### 候補機器と設定

| 部位・状況 | 機器 | 設定方針 |
|---|---|---|
| 深部関節包（肩・股・膝）| ラジオ波・超音波 1MHz 連続 | 温感が続く出力・15〜20分 |
| 表層筋・筋膜 | ホットパック + ストレッチ | 表層温熱 + 徒手 |
| 頚部〜腰部の筋緊張 | 干渉波・ラジオ波 | 温感程度・中等度出力 |
| 瘢痕拘縮 | 超音波（連続・中等度）+ ストレッチ | 温めながら動かす |

### 注意点

- 炎症性拘縮（関節内腫脹・熱感あり）への温熱は禁忌
- 急性期に無理に可動域を出さない（炎症悪化・再損傷）
- 温めた後にすぐ動かすこと（冷えると元に戻る）

### 再評価ポイント

- 治療前後の可動域の変化（角度で記録）
- 患者の「動きやすさ」の主観的変化
- 温熱中・後の痛みの変化

## Type 4: 浮腫優位（Edema Dominant）

### 評価すること

| 評価項目 | 確認のポイント |
|---|---|
| 浮腫の種類 | 急性炎症性 / 廃用性 / リンパ性 / 静脈性 |
| 発症時期 | 受傷直後 / 慢性 |
| DVT リスク | 長期臥床・手術後・深部の硬い腫脹（レッドフラッグ！）|
| 左右差 | 片側のみの急性浮腫は DVT 除外が必要 |

### 候補機器と設定

| 状況 | 機器 | 設定方針 |
|---|---|---|
| 急性炎症性浮腫 | HV（マイナス極）| 低強度・20分。RICE と組み合わせ |
| 廃用性浮腫 | EMS（筋ポンプ）| 収縮を引き出す程度 |
| 亜急性〜慢性の浮腫 | 干渉波・低周波 | 循環改善モード |

### 絶対禁忌

- DVT 疑いの下肢に電気・温熱・マッサージを行わない（血栓遊離リスク）
- リンパ浮腫は専門家の徒手リンパドレナージが主体。物療は補助的

### 再評価ポイント

- 周径測定（cm）での浮腫の変化
- 皮膚の張りと熱感の変化

## Type 5: 創傷・組織修復優位（Wound / Repair Dominant）

### 評価すること

| 評価項目 | 確認のポイント |
|---|---|
| 修復フェーズ | 炎症期 / 増殖期 / リモデリング期（`docs/PATHOLOGY_AND_HEALING.md`）|
| 組織の種類 | 筋 / 腱 / 靭帯 / 皮膚 / 骨 |
| 修復の遅れ | 通常の期間を超えている理由（糖尿病・循環障害・栄養不足等）|

### 候補機器と設定

| フェーズ・組織 | 機器 | 設定 |
|---|---|---|
| 増殖期以降の腱・靭帯 | 超音波パルス（LIPUS）| 0.5〜1.0 W/cm²・20〜50%パルス・10分 |
| 慢性腱症 | ESWT | 放射状 or 焦点式・3〜5回コース |
| 骨折修復促進 | LIPUS | 0.03 W/cm²・20分/日 |
| 創傷 | HV（マイナス極）| 低強度・清潔な電極使用 |

### 注意点

- 急性炎症期（72時間以内）の創傷に超音波連続波は禁忌
- 機器・プローブの清潔管理（創傷部位では特に）
- 「修復促進=早く治る」を過剰に患者に約束しない

### 再評価ポイント

- 組織修復の進行（圧痛の減少・可動域の改善・腫脹の軽減）
- 機能的なアウトカム（痛みなしで荷重できるか等）

## Type 6: 神経症状優位（Neuropathic / Neurological Dominant）

### 評価すること

| 評価項目 | 確認のポイント |
|---|---|
| 症状の分布 | デルマトーム / 末梢神経支配域 / 広汎（中枢性）|
| 性質 | しびれ / 灼熱感 / 電気が走る / 感覚鈍麻 |
| 筋力への影響 | 筋力低下・反射消失の有無 |
| 進行性か | 悪化傾向 → レッドフラッグ評価（`docs/RED_FLAGS_AND_REFERRAL.md`）|
| 経過 | 受傷・手術後 / 慢性化 / 原因不明 |

### 候補機器と設定

| 目的 | 機器 | 設定方針 |
|---|---|---|
| 神経症状の疼痛調整 | TENS | 症状領域に電極を配置・高頻度 80〜100Hz |
| 神経根近傍の浮腫軽減 | HV・干渉波 | 低〜中強度 |
| 神経の循環改善 | 温熱（軽度）| 感覚障害がなければ温熱補助 |

### 注意点

- **感覚障害のある部位への電気・温熱は熱傷・電気熱傷リスク。使うなら低強度で厳密に確認**
- 進行性・急速に悪化する神経症状は物療より先に医師評価
- 電気刺激が症状を一時的に増悪させる場合は中止・設定変更

### 再評価ポイント

- しびれの分布・強度の変化（VAS or NRS）
- 感覚検査（軽触覚・温冷覚）の変化
- 筋力の変化

## 複合症状のとき（複数の Type が重なる場合）

多くの患者は複数の症状タイプを持つ（例：痛み + 拘縮 + 廃用）。

### 優先順位の原則

1. **レッドフラッグを除外する**（常に最優先）
2. **炎症の有無を確認する**（急性炎症があれば温熱を除外）
3. **最も患者が困っている主訴から狙う**
4. **フェーズを確認して機器を絞る**（`docs/PATHOLOGY_AND_HEALING.md`）
5. **組み合わせは2〜3機器まで**（情報が多すぎると反応が見えにくくなる）
