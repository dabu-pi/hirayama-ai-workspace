# PATHOLOGY_AND_HEALING — 病理と組織治癒

**プロジェクトID:** PMOD-08  
**最終更新:** 2026-04-10  
**位置づけ:** 「今この患者はどのフェーズか」を判断し、物療選択に反映するための病理・治癒ガイド

---

## 基本思想

物療の選択ミスの多くは、**「今、患者の組織がどの状態か」を無視すること**から起きる。

- 炎症が活発な時期に温熱を当てる
- まだ炎症期なのに修復促進の超音波を使う
- 完全に修復したのに「念のため」同じプロトコルを続ける

病態のフェーズを正しく理解することが、適切な物療選択の前提。

---

## 1. 組織修復の3フェーズ

### Phase 1: 炎症期（Inflammatory Phase）

**期間:** 急性損傷後 0〜72時間（〜5日）

| 項目 | 内容 |
|---|---|
| トリガー | 組織損傷・血管破綻・細菌感染 |
| 主な変化 | 血管拡張・透過性増大・好中球・マクロファージが集結・壊死組織処理 |
| 発痛物質 | ブラジキニン・プロスタグランジンE2・ヒスタミン・サブスタンスP |
| 臨床所見 | 発赤（Rubor）・熱感（Calor）・腫脹（Tumor）・疼痛（Dolor）・機能障害（Functio laesa） |
| 役割 | 修復のための「準備と清掃」。この反応は必要な過程 |

**物療方針:**

| 許可 | 禁忌 |
|---|---|
| TENS（疼痛調整）| 温熱全般（炎症悪化）|
| ハイボルテージ（浮腫吸収・疼痛）| 強い電気刺激（炎症部位刺激）|
| 冷却（アイシング）| 超音波連続波 |
| 固定・保護 | EMS（損傷部位の筋収縮）|

> **「炎症を無闇に消そうとしない」** 炎症は修復の始まり。過剰な鎮炎は修復を遅らせる可能性がある。

---

### Phase 2: 増殖期（Proliferative Phase）

**期間:** 3〜5日〜3週間（損傷の程度と組織種類で大きく異なる）

| 項目 | 内容 |
|---|---|
| 主な変化 | 肉芽組織形成・コラーゲン合成（Type III）・血管新生・上皮化 |
| 役割 | 「仮の橋渡し」として組織を埋める |
| 臨床所見 | 急性炎症症状が和らぐ。腫脹が徐々に軽減。圧痛は残る |
| リスク | 過負荷 → 修復組織の再損傷。過度な安静 → 廃用 |

**物療方針:**

| 目的 | 機器 | 設定 |
|---|---|---|
| 組織修復促進 | 超音波（パルス波・低強度）| 0.5〜1.0 W/cm²・20〜50%パルス |
| 疼痛調整 | TENS・低周波 | 高頻度 or 低頻度 |
| 循環促進（浮腫残存） | ハイボルテージ・干渉波 | 軽〜中程度 |
| 筋機能維持（廃用予防） | EMS（弱め） | 損傷部位外から |

---

### Phase 3: リモデリング期（Remodeling Phase）

**期間:** 3週〜1〜2年（特に腱・靭帯・骨は長い）

| 項目 | 内容 |
|---|---|
| 主な変化 | Type IIIコラーゲン→Type Iコラーゲンへの転換・線維の再配列・瘢痕成熟 |
| 役割 | 機能的な強度と柔軟性の回復 |
| 臨床所見 | 痛みは軽減。可動域制限・筋力低下・拘縮が残ることある |
| リスク | 不動による過剰な瘢痕・拘縮。過負荷による再損傷 |

**物療方針:**

| 目的 | 機器 | 設定 |
|---|---|---|
| 柔軟性回復（拘縮） | 深部温熱（ラジオ波・超音波1MHz連続）| 温感が続く強度 |
| 筋機能回復 | EMS・低周波（筋収縮モード） | 徐々に強度・負荷を上げる |
| 瘢痕・腱のリモデリング | ショックウェーブ（慢性腱症）| 3〜5回コース |

## 2. 急性損傷と慢性化の違い

### 急性損傷（Acute Injury）

- 明確な受傷機転がある
- 炎症の5徴が明確（発赤・熱感・腫脹・疼痛・機能障害）
- 組織修復の3フェーズを比較的順に経過する
- 適切な管理で回復が期待できる

### 慢性化（Chronification）

慢性化は「急性損傷が長引いた」だけではない。以下のパターンがある。

| パターン | 説明 | 物療での考慮 |
|---|---|---|
| 修復の停滞 | 炎症→増殖のサイクルが完結しない | 修復促進（超音波・HV）+ 負荷管理 |
| 過負荷の繰り返し | 修復中に再損傷を繰り返す | 負荷の制限 + 組織修復サポート |
| 廃用 | 不動・避けすぎで組織が弱くなる | EMS + 段階的負荷 |
| 中枢性感作 | 脳・脊髄レベルで痛みが増幅・固着 | 物療の限界を理解。認知行動的アプローチと統合 |
| 腱症（Tendinopathy） | 腱組織の変性（修復≠再生）| ショックウェーブ・偏心性負荷 |

## 3. 主要病態別の理解と物療方針

### 腱障害（Tendinopathy / Tendinitis）

| 病態 | 急性腱炎 | 慢性腱症 |
|---|---|---|
| 組織変化 | 炎症細胞の浸潤・浮腫 | コラーゲン構造の乱れ・血管新生 |
| 発症 | 突然・外傷・過負荷 | 繰り返しの微小損傷 |
| 物療 | TENS・アイシング・RICE | ショックウェーブ・超音波パルス・偏心性運動 |
| 注意 | 急性期に温熱・強い超音波禁忌 | 慢性腱症は長期管理。即効は期待しない |

### 筋損傷（Muscle Strain）

| Grade | 程度 | 物療方針 |
|---|---|---|
| Grade I（軽度）| 筋線維の微細損傷。腫脹なし | 軽い温熱・TENS・早期運動 |
| Grade II（中等度）| 部分断裂。腫脹・内出血あり | 急性期：RICE・TENS。3日後〜超音波パルス |
| Grade III（重度）| 完全断裂 | 外科的管理の可能性。物療は補助的 |

### 靭帯損傷（Ligament Sprain）

| Grade | 程度 | 物療方針 |
|---|---|---|
| Grade I | 伸張のみ・不安定性なし | RICE → 早期運動 |
| Grade II | 部分断裂・軽度不安定性 | RICE → 超音波パルス・低強度電気 |
| Grade III | 完全断裂 | 整形外科受診。手術or長期保存療法の判断 |

### 関節拘縮（Joint Contracture）

| 原因 | 物療方針 |
|---|---|
| 不動（廃用性拘縮） | 深部温熱（温めてから動かす）+ EMS + 牽引 |
| 瘢痕化（外傷後） | 深部温熱 + 徒手的牽引・モビリゼーション |
| 炎症性拘縮 | 炎症制御が先。温熱は慎重に |

### 浮腫（Edema）

| 種類 | 主な原因 | 物療方針 |
|---|---|---|
| 急性炎症性浮腫 | 血管透過性増大 | ハイボルテージ（マイナス極）・RICE |
| 廃用性浮腫 | 循環低下・重力 | EMS（筋ポンプ）・挙上 |
| リンパ浮腫 | リンパ管障害 | 専門家の徒手リンパドレナージが主体 |

### 疼痛過敏・慢性疼痛

| タイプ | 説明 | 物療方針 |
|---|---|---|
| 末梢性感作 | 炎症物質による侵害受容器の閾値低下 | 炎症制御が先決。TENS補助 |
| 中枢性感作 | 脊髄・脳での過剰増幅 | 物療単独では限界。認知・行動的介入との統合が必要 |
| 痛みの破局化 | 痛みへの過剰な恐怖・回避 | 物療 + 教育・エクスポージャー |

## 4. 病態把握のための臨床評価のポイント

### 急性 vs 慢性を見分ける

| 項目 | 急性 | 慢性 |
|---|---|---|
| 受傷機転 | 明確 | 不明確か繰り返し |
| 炎症の5徴 | あり | 少ない・不明確 |
| 痛みの性質 | 鋭い・局所的 | 鈍い・広範・移動性 |
| 動作との関係 | 特定動作で明確な増悪 | 多様・一定しない |
| 夜間痛・安静時痛 | 炎症期は強い | 慢性でも存在（中枢性感作）|

### フェーズ判断のチェックリスト

- [ ] 発症から何日か（急性期目安: 72時間以内）
- [ ] 炎症の5徴はあるか
- [ ] 受傷機転はあるか
- [ ] 以前に同部位の損傷歴があるか
- [ ] 安静にすると楽になるか（炎症性）vs 動き始めが一番痛い（拘縮・廃用）
- [ ] レッドフラッグはないか（`docs/RED_FLAGS_AND_REFERRAL.md` 参照）

## 5. 「物療で治るもの」と「物療では治せないもの」

### 物療で直接的に狙えること

- 疼痛の一時的な軽減（中枢・末梢の両方）
- 局所の血流改善
- 筋収縮の誘発（廃用予防・筋力補助）
- 組織修復の促進（超音波・HV・LIPUS）
- 温熱による柔軟性改善
- 浮腫の軽減

### 物療では直接的に変えられないこと

- 軟骨の再生
- 靭帯完全断裂の修復（程度次第で手術適応）
- 神経軸索の再生（時間が必要）
- 中枢性感作（認知・行動的アプローチが必要）
- 生活習慣・姿勢・動作パターンの改善（運動療法・指導が必要）
- 患者の心理社会的因子（恐怖・不安・社会的孤立等）
