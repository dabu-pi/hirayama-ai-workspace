# HUMAN_BIOLOGY_MAP — 物療を学ぶための人体知識地図

**プロジェクトID:** PMOD-08  
**最終更新:** 2026-04-10  
**位置づけ:** 物療機器を適切に選ぶために「人体側で理解すべきこと」の全体地図

---

## 基本思想

> **「機器を知る前に、人体を知る」**

物療機器の選択で失敗する最大の理由は、「機器の機能」を知っていても
「患者の体内で何が起きているか」を知らないこと。

この地図は、物療選択に直結する人体知識を体系化したもの。
各知識は独立して存在するのではなく、**「どの物療の選択に関わるか」という軸で整理**する。

---

## 人体知識の全体構造

```
[解剖学]          何が、どこに、どんな構造で存在するか
      ↓
[生理学]          それぞれが普通はどう機能しているか
      ↓
[病理学]          何が壊れたり・狂ったりしているか
      ↓
[組織治癒]        壊れたものはどう修復されるか
      ↓
[疼痛科学]        「痛み」という現象はどう起きているか
      ↓
[症状・疾患理解]  患者に何が起きているか
      ↓
[臨床判断]        物療で何を、どう変えるか
```

## 3. 病理学 / 組織治癒（Pathology & Healing）

### 組織修復の3フェーズ

| フェーズ | 期間目安 | 主な出来事 | 物療の方針 |
|---|---|---|---|
| 炎症期 | 0〜72時間（〜1週） | 血管反応・発痛物質・好中球 | 温熱禁忌。RICE。TENS/HVは可 |
| 増殖期 | 3日〜3週 | 肉芽組織・コラーゲン合成 | 超音波パルス・低強度電気で修復促進 |
| リモデリング期 | 3週〜1〜2年 | コラーゲン再構成・瘢痕成熟 | 温熱・牽引・運動で柔軟性回復 |

### 慢性化のパターン

- **廃用:** 不動による筋萎縮・関節拘縮・骨密度低下
- **過負荷:** 腱障害（腱症）・疲労骨折・滑液包炎
- **慢性炎症:** 線維化・拘縮・滑膜の肥厚

**詳細:** `docs/PATHOLOGY_AND_HEALING.md`

## 4. 疼痛科学（Pain Science）

### 痛みの4つの理解軸

| 軸 | 内容 | 物療との関係 |
|---|---|---|
| 侵害受容 | 侵害刺激→侵害受容器→脊髄→脳 | TENSのゲートコントロールの対象 |
| 末梢性感作 | 発痛物質による侵害受容器の閾値低下 | 急性炎症期の過敏性の原因 |
| 中枢性感作 | 脊髄・脳での過剰な痛み増幅 | 慢性疼痛の核心。物療だけでは不十分 |
| 痛みの認知・情動 | 恐怖・回避・catastrophizing | 物療効果の個人差に大きく影響 |

### 重要な臨床的含意

- **痛み ≠ 組織損傷の程度**（特に慢性疼痛で乖離が大きい）
- 「組織が治っているのに痛い」→ 中枢性感作を疑う
- 物療で「体を治す」だけでは慢性疼痛に限界がある

**詳細:** `docs/PAIN_SCIENCE_BASICS.md`

## 5. 神経生理（Neurophysiology）

### 治療家が知るべき神経の基礎

| 項目 | 内容 | 物療との関係 |
|---|---|---|
| 神経線維の分類 | Aα（運動）/ Aβ（触覚）/ Aδ・C（痛覚） | どの線維を刺激するかで効果が変わる |
| 順行性・逆行性伝導 | 刺激は両方向に伝わる | TENS効果の解釈に関係 |
| ゲートコントロール理論 | Aβ活性化→痛み信号抑制 | 高頻度TENSの根拠 |
| 内因性オピオイド系 | 低頻度刺激→エンケファリン・エンドルフィン | 低頻度TENSの根拠 |
| 神経可塑性 | 繰り返し刺激による神経回路変化 | 慢性疼痛への長期的な影響 |
| 交感神経と痛み | 交感神経緊張→慢性疼痛の維持 | 温熱・リラクゼーション効果の意義 |

## 6. 筋機能・運動学（Muscle Function & Kinesiology）

### 物療と筋の関係

| 項目 | 内容 | 物療との関係 |
|---|---|---|
| 筋収縮の種類 | 求心性・遠心性・等尺性 | EMSの設定・負荷選択 |
| 筋の電気特性 | 筋線維の収縮閾値・持久性 | EMSの周波数・強度選択 |
| 廃用性萎縮 | 不動2週で顕著な萎縮開始 | EMSの早期介入根拠 |
| 筋痙縮 | 過度な収縮→虚血→痛み→痙縮の悪循環 | 温熱・電気による筋弛緩の根拠 |
| 筋と関節の関係 | 筋機能低下→関節ストレス増大 | 物療 + 運動療法統合の重要性 |

## 7. 循環・浮腫（Circulation & Edema）

### 物療が影響する循環因子

| 因子 | 物療との関係 |
|---|---|
| 毛細血管血流 | 温熱・電気で局所血流増加 |
| 静脈還流 | 筋収縮（EMS）でポンプ作用 |
| リンパ管 | ハイボルテージ・超音波でリンパ流促進 |
| スターリングの法則 | 毛細血管圧・膠質浸透圧のバランスが浮腫を決定 |
| 炎症性浮腫 | 血管透過性↑→滲出。急性期は温熱で悪化 |

## 8. 筋・腱・靭帯・関節包・軟骨・骨の違い

物療は組織の種類によって「狙えること」が違う。

| 組織 | 血流 | 修復力 | 物療の主なターゲット |
|---|---|---|---|
| 筋 | 豊富 | 良好 | 収縮誘発・弛緩・温熱 |
| 腱 | 乏しい | 遅い | 超音波LIPUS・ショックウェーブ |
| 靭帯 | 乏しい | 遅い | 超音波・低強度電気（修復促進） |
| 関節包 | 中程度 | 中程度 | 温熱（柔軟性）・牽引 |
| 軟骨 | ほぼなし | 非常に遅い | 温熱（間接）・牽引（関節内圧軽減） |
| 骨 | 骨幹部から | 良好 | LIPUS（骨折修復）|
| 神経 | 中程度 | 非常に遅い | TENS（機能補助）・温熱（伝導改善）|

## 9. 炎症の理解

| 炎症の段階 | 主な特徴 | 物療の方針 |
|---|---|---|
| 急性炎症（血管性） | 発赤・熱感・腫脹・疼痛・機能障害の5徴 | 温熱禁忌・冷却・TENS・HVは検討可 |
| 亜急性 | 症状軽減・修復開始 | 超音波パルス・低強度電気 |
| 慢性炎症 | 組織変性・線維化・持続する低グレード炎症 | 温熱・ショックウェーブ・運動療法 |

**「炎症を全部抑えようとしない」が原則。**  
炎症は修復の第一ステップ。急性期の過剰な鎮静よりも、適切な管理が重要。

## 知識の統合マトリクス

「何を理解すれば、何が選べるようになるか」

| 理解する知識 | できるようになること |
|---|---|
| 解剖（組織の深さ・構造） | 電極・プローブの配置決定、深度選択 |
| 生理（膜電位・感覚神経） | TENSの周波数選択の根拠理解 |
| 生理（炎症・修復） | 急性期禁忌と亜急性期以降の許可判断 |
| 組織治癒の3フェーズ | 「今どのフェーズか」で使う機器を変える |
| 疼痛科学（末梢・中枢感作） | 慢性疼痛患者への過剰な期待を持たせない説明 |
| 神経生理（Aβ/C線維） | TENSの周波数・強度・パルス幅選択の根拠 |
| 筋機能 | EMSの目的と設定の論理的決定 |
| 循環・浮腫の生理 | ハイボルテージ・温熱の効果の説明 |
