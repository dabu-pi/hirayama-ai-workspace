# LEARNING_GOALS — 学習目標の定義

**プロジェクトID:** PMOD-08  
**最終更新:** 2026-04-10

---

## 全体目標

「物療機器を体系的に理解し、患者ごとに根拠を持った選択ができる」状態になること。

---

## Phase別 学習目標

### Phase 1 目標: 体系的な理解

- 主要物療機器の種類と特性を説明できる
- 「周波数」「パルス幅」「duty cycle」など設定パラメータの意味がわかる
- 「この機器は体内で何を起こすか」を説明できる
- 目的別（疼痛・筋収縮・温熱・循環・修復）で機器を分類できる

### Phase 2 目標: 臨床的判断

- 患者の状態（急性・慢性・組織・深さ）から機器の候補を挙げられる
- 禁忌を即座に確認できる
- 「なぜその機器を選んだか」を患者に説明できる
- 設定（周波数・強度・時間）を目的に応じて決められる

### Phase 3 目標: 症状別実践

- 主要7症状で「何をどう使うか」を即座に答えられる
- 設定の具体的な数値を示せる
- 治療後の反応を評価し、次回設定を調整できる

### Phase 4〜 目標: 知識の体系化と共有

- 症例データから再現性のあるプロトコルを作れる
- 知識をnote記事として他者に伝えられる品質で書ける
- AIを使って知識の誤りや陳腐化を検証できる
