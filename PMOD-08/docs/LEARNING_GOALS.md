# LEARNING_GOALS — 学習目標の定義

**プロジェクトID:** PMOD-08  
**最終更新:** 2026-04-10

---

## 全体目標

「物療機器 × 人体 × 病態 × 症状・疾患」を統合して理解し、  
患者ごとに根拠を持って機器・設定を選択し、患者に説明できる状態になること。

---

## Phase別 学習目標

### Phase 1 目標: 物療の全体像と人体の基礎理解

**人体側の理解（必須基盤）**

- [ ] 物療ターゲット組織（筋・腱・靭帯・神経・骨等）の特性の違いを説明できる
- [ ] 組織修復の3フェーズ（炎症期・増殖期・リモデリング期）で物療方針が変わることを説明できる
- [ ] ゲートコントロール理論と内因性オピオイド放出の違いを説明できる
- [ ] 末梢性感作と中枢性感作の違いを患者に説明できる言葉で言える
- [ ] 急性炎症期に温熱が禁忌な理由を生理学で説明できる

**機器の理解**

- [ ] 主要物療機器の種類と体内での主な反応を説明できる
- [ ] 「周波数」「パルス幅」「duty cycle」の意味と効果の関係がわかる
- [ ] 目的別（疼痛・筋収縮・温熱・循環・修復）で機器を分類できる
- [ ] 自院保有機器の全リストと主な用途を把握している

**安全管理**

- [ ] レッドフラッグを10項目以上挙げられる（`docs/RED_FLAGS_AND_REFERRAL.md`）
- [ ] DVT疑いに物療が禁忌な理由を説明できる
- [ ] ペースメーカー患者への対応を説明できる

---

### Phase 2 目標: 評価と選択の原則

- [ ] 患者の状態（急性・慢性・組織・深さ・フェーズ）から機器の候補を挙げられる
- [ ] 症状の6タイプ（痛み優位・筋出力低下・拘縮・浮腫・創傷・神経症状）で判断ガイドを使いこなせる
- [ ] 「なぜその機器を選んだか」を患者に説明できる
- [ ] 設定（周波数・強度・時間・Duty Cycle）を目的に応じて決められる
- [ ] 治療後の反応を評価し、次回設定を調整できる
- [ ] 炎症期 vs 慢性期で使う機器が変わることを即座に判断できる

---

### Phase 3 目標: 症状別・疾患別の実践

- [ ] 整骨院でよく見る主要7疾患・症状で「何をどう使うか」を答えられる
  - 慢性腰痛 / 頚部痛・肩こり / 五十肩 / 腱障害 / 足関節捻挫 / 膝痛 / 神経症状
- [ ] 各疾患で「物療の限界」（治せないこと）を患者に説明できる
- [ ] 運動療法・生活指導との組み合わせを患者に提示できる
- [ ] 「物療だけで治る」という過剰な期待を作らない説明ができる

---

### Phase 4〜 目標: 知識の体系化と共有

- [ ] 症例データから再現性のあるプロトコルを作れる
- [ ] 知識をnote記事として他者に伝えられる品質で書ける
- [ ] AIを使って知識の誤りや陳腐化を検証できる
- [ ] エビデンスの強さと臨床的実用性を分けて説明できる

---

## 自己評価チェックリスト（Phase 1 完了の目安）

### 人体理解

- [ ] 腱と筋の修復速度の違いと理由を説明できる
- [ ] 急性炎症の5徴（発赤・熱感・腫脹・疼痛・機能障害）を挙げられる
- [ ] ゲートコントロール理論に基づくTENSの設定根拠を説明できる
- [ ] 中枢性感作とは何かを1〜2文で患者に説明できる

### 機器理解

- [ ] 低周波・干渉波・ハイボルテージの使い分けを説明できる
- [ ] ショックウェーブと超音波の適応の違いを説明できる
- [ ] 1MHz と 3MHz の違いを深度で説明できる
- [ ] 自院保有機器の全リストを作成した

### 安全管理

- [ ] 受診前にレッドフラッグチェックを系統的に行える
- [ ] 「医師に紹介すべきケース」を3つ以上即座に答えられる
