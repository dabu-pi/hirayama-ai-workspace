# DISEASE_AND_MODALITY_MATRIX — 疾患・症状別 物療候補一覧

**プロジェクトID:** PMOD-08  
**最終更新:** 2026-04-10  
**位置づけ:** 疾患・症状ごとの候補機器と狙う反応を一覧化する速引きリファレンス

---

## 使い方

- 「研究上の評価」と「臨床上の実用性」は分けて記載している
- エビデンスレベルは `docs/EVIDENCE_POLICY.md` の L1〜L6 を使用
- 「候補機器」はあくまで候補。患者の状態・禁忌・設備で最終選択する
- フェーズ（急性/亜急性/慢性）が大きく変われば選択も変わる

## マトリクス

### 頚部痛・肩こり（慢性）

| 項目 | 内容 |
|---|---|
| 主病態 | 筋緊張・筋虚血・筋膜疼痛 |
| 主対象組織 | 僧帽筋・肩甲挙筋・頚部深層筋・筋膜 |
| 物療の目的 | 筋弛緩・血流改善・疼痛調整 |
| 候補機器 | 干渉波 / ラジオ波（深部温熱）/ 低周波（高頻度）|
| 狙う生体反応 | 深部血流増加・筋スパズム解除・ゲートコントロール |
| 使う理由 | 深部筋は表層温熱では届かない。干渉波で深部刺激+循環改善 |
| 注意点 | 頚動脈洞周囲への通電禁忌。神経根症状の除外 |
| 研究上の評価 | TENS・干渉波の肩こりへの短期鎮痛効果あり（L2〜L3）。長期効果は限定的 |
| 臨床上の実用性 | 干渉波・ラジオ波は患者満足度高い。即時効果が感じられる |
| 患者説明要点 | 「深部の筋を緩め、血流を改善します。継続と運動が大切です」|

### 慢性腰痛（非特異的）

| 項目 | 内容 |
|---|---|
| 主病態 | 筋緊張・椎間関節・椎間板・中枢性感作 |
| 主対象組織 | 脊柱起立筋・多裂筋・腰椎椎間関節 |
| 物療の目的 | 疼痛調整・深部温熱・廃用筋賦活 |
| 候補機器 | TENS / 干渉波 / ラジオ波 / EMS（体幹筋）|
| 狙う生体反応 | ゲートコントロール・深部温熱・多裂筋収縮誘発 |
| 使う理由 | 腸腰筋・多裂筋は深部にあり、深部温熱が有効 |
| 注意点 | 急性炎症期は温熱禁忌。慢性期は中枢性感作への配慮 |
| 研究上の評価 | TENS の慢性腰痛への効果は研究で結果が分かれる（L2〜L3）。ラジオ波の長期エビデンスは少ない |
| 臨床上の実用性 | 干渉波・ラジオ波との組み合わせ + 運動療法が実務で多用 |
| 患者説明要点 | 「痛みを一時的に和らげながら、動かしやすい状態を作ります。運動との組み合わせが重要です」|

---

### 坐骨神経痛（椎間板ヘルニア・神経根症状）

| 項目 | 内容 |
|---|---|
| 主病態 | 神経根への圧迫・炎症・末梢性感作 |
| 主対象組織 | 神経根・椎間板・椎間孔周囲 |
| 物療の目的 | 疼痛調整・神経周囲浮腫軽減・筋痙縮緩和 |
| 候補機器 | TENS / 低周波（高頻度）/ 腰椎牽引（慎重）|
| 狙う生体反応 | 神経症状域のゲートコントロール・椎間孔の拡大（牽引）|
| 使う理由 | 神経根症状は末梢の組織だけでなく神経経路全体に痛みが出る |
| 注意点 | 進行性筋力低下・排尿障害は即医師へ。牽引は急性期慎重・症状悪化時は中止 |
| 研究上の評価 | 牽引の腰椎ヘルニアへのエビデンスは限定的（L3）。TENSは鎮痛補助として使用 |
| 臨床上の実用性 | 急性期はTENSで疼痛管理が主体。牽引は症状の軽快期に段階的に |
| 患者説明要点 | 「神経の興奮を和らげます。症状が広がったり強くなる場合は教えてください」|

### 変形性膝関節症

| 項目 | 内容 |
|---|---|
| 主病態 | 関節軟骨の変性・骨棘・滑膜炎・大腿四頭筋廃用 |
| 主対象組織 | 関節軟骨（間接的）・滑膜・大腿四頭筋 |
| 物療の目的 | 疼痛調整・大腿四頭筋賦活・温熱による可動域改善 |
| 候補機器 | TENS / 低周波 / EMS（大腿四頭筋）/ 超音波（1MHz）|
| 狙う生体反応 | ゲートコントロール・筋収縮誘発・関節周囲組織の温熱 |
| 使う理由 | 大腿四頭筋の廃用が痛みと機能低下の悪循環に関係する |
| 注意点 | 急性滑膜炎（腫脹・熱感）への温熱は禁忌 |
| 研究上の評価 | EMS・TENSの変形性膝関節症への疼痛改善効果はSRあり（L1〜L2）。軟骨修復効果は期待できない |
| 臨床上の実用性 | EMSと運動療法の組み合わせが長期的機能維持に有効 |
| 患者説明要点 | 「軟骨が再生するわけではありませんが、痛みを管理しながら膝周囲の筋を強くして、日常生活を楽にします」|

### 足関節捻挫（急性）

| 項目 | 内容 |
|---|---|
| 主病態 | 外側靭帯の損傷・急性炎症・浮腫 |
| 主対象組織 | 前距腓靭帯・踵腓靭帯 |
| 物療の目的 | 浮腫軽減・疼痛管理（急性期）/ 修復促進（亜急性期）|
| 候補機器 | HV（急性期）/ 超音波パルス（亜急性以降）/ TENS |
| 狙う生体反応 | 浮腫吸収・ゲートコントロール・組織修復促進 |
| 使う理由 | 急性期は温熱禁忌。HVのマイナス極が浮腫吸収に有効とされる |
| 注意点 | 骨折除外が最優先（Ottawa Ankle Rules）。急性期の温熱は禁忌 |
| 研究上の評価 | 超音波の捻挫への修復促進効果あり（L2〜L3）。HVの証拠は限定的 |
| 臨床上の実用性 | 急性期はRICE + HV/TENS。亜急性期以降に超音波パルスを追加 |
| 患者説明要点 | 「最初は腫れを引かせることが目標です。靭帯の修復を助ける治療を段階的に進めます」|

### 腱障害（アキレス腱・膝蓋腱・足底腱膜炎 等の慢性）

| 項目 | 内容 |
|---|---|
| 主病態 | 腱組織の変性（腱症）・コラーゲン構造の乱れ |
| 主対象組織 | 腱・腱付着部 |
| 物療の目的 | 組織修復促進・疼痛調整 |
| 候補機器 | ESWT（ショックウェーブ）/ 超音波パルス（LIPUS）|
| 狙う生体反応 | 腱の血管新生促進・石灰化破壊（石灰化腱炎）・コラーゲン再構成 |
| 使う理由 | 腱は血流が乏しく修復が遅い。ESWTが血管新生・修復を促進するとされる |
| 注意点 | 抗凝固剤使用中はESWT禁忌。治療中の痛みが強い場合は強度調整 |
| 研究上の評価 | アキレス腱・足底腱膜炎へのESWTはRCT・メタ解析あり（L1〜L2）。ただしプロトコルにばらつき |
| 臨床上の実用性 | 保存療法に反応しない慢性腱障害の第2選択として有用。偏心性収縮訓練との組み合わせを推奨 |
| 患者説明要点 | 「腱の修復を促す衝撃波治療です。3〜5回のコースで効果が出ることが多く、治療中の痛みはある程度伴います」|

### 肩関節拘縮（凍結肩・五十肩）

| 項目 | 内容 |
|---|---|
| フェーズ | 凍結期（急性炎症）→ 拘縮期 → 解凍期 |
| 主対象組織 | 関節包・肩峰下滑液包・腱板 |
| 物療の目的 | 急性期：疼痛管理。拘縮期：深部温熱で関節包軟化 + 可動域訓練 |
| 候補機器 | 超音波 1MHz 連続 / ラジオ波（深部温熱）/ TENS（疼痛）|
| 狙う生体反応 | 関節包の温熱による粘弾性改善 → 徒手・運動療法の効果最大化 |
| 注意点 | 急性炎症期（夜間痛が強い時期）の温熱は禁忌。拘縮期に移行したことを確認 |
| 研究上の評価 | 超音波・温熱の五十肩への効果は限定的なエビデンス（L3）。運動療法との組み合わせが重要 |
| 臨床上の実用性 | 深部温熱後すぐに可動域訓練を行うことで可動域改善が得やすい |
| 患者説明要点 | 「関節包を内側から温めて柔らかくした上で動かします。温めた後すぐ動かすことが大切です」|

---

### EMS / 廃用性筋萎縮（高齢者・術後）

| 項目 | 内容 |
|---|---|
| 主病態 | 不動・廃用による筋萎縮・筋力低下 |
| 主対象組織 | 大腿四頭筋・腸腰筋・体幹筋 等 |
| 物療の目的 | 筋収縮誘発・廃用進行の抑制・筋力回復の補助 |
| 候補機器 | EMS（Neuromuscular Electrical Stimulation）|
| 狙う生体反応 | 筋線維の動員・タンパク合成促進・筋ポンプによる循環改善 |
| 注意点 | 感覚障害のある高齢者は刺激の過剰に注意。関節不安定性がある場合は固定を優先 |
| 研究上の評価 | ICU患者・術後廃用への早期EMS介入のエビデンスあり（L1〜L2）|
| 臨床上の実用性 | 運動療法が困難な時期の橋渡しとして有効。運動療法への移行が最終目標 |
| 患者説明要点 | 「電気で筋肉を動かすリハビリです。自分で動かす練習の補助として使います」|

## 「研究上の評価」と「臨床上の実用性」の分離について

この表は以下を分けて記載している。

| 項目 | 意味 |
|---|---|
| 研究上の評価 | RCT・SR等の研究で示された効果の証拠の強さ |
| 臨床上の実用性 | 実際の整骨院臨床での使いやすさ・患者受け入れ・現実的な期待値 |

研究が少ない = 効果がないではない。研究の限界（対象・プロトコルのばらつき等）を理解して使う。
(`docs/EVIDENCE_POLICY.md` 参照)
