# COMMON_CONDITIONS_MAP — 整骨院でよくみる症状・疾患マップ

**プロジェクトID:** PMOD-08  
**最終更新:** 2026-04-10  
**位置づけ:** 整骨院・接骨院でよく遭遇する症状・疾患を病態・物療視点で体系化する

---

## 基本思想

疾患名を覚えることが目的ではない。  
**「この患者に今何が起きているか」を病態レベルで理解し、物療の狙いを決める**ことが目的。

---

## 1. 頚部痛・肩こり

| 項目 | 内容 |
|---|---|
| 主病態 | 筋緊張・筋虚血・筋膜疼痛・頚椎椎間板・神経根刺激 |
| 主な対象組織 | 僧帽筋・肩甲挙筋・菱形筋・頚部深層筋・筋膜・頚椎椎間板・神経根 |
| 物療の主目的 | 筋弛緩・血流改善・疼痛調整 |
| よく使う評価 | 頚椎可動域・圧痛点・神経症状の有無・姿勢 |
| 注意点 | 神経根症状（しびれ・放散痛）があれば除外診断を先行。頚動脈への電極配置禁忌 |
| 運動療法との接続 | 姿勢指導・頚部安定化・スマートフォン使用習慣の改善 |
| レッドフラッグ | 進行性の上肢筋力低下・排尿障害・転倒後の頚部痛 |

---

## 2. 肩関節痛（五十肩・腱板損傷含む）

| 項目 | 内容 |
|---|---|
| 主病態 | 関節包の拘縮（凍結肩）・腱板の変性・滑液包炎・石灰化腱炎 |
| 主な対象組織 | 棘上筋腱・関節包・肩峰下滑液包・烏口肩峰靭帯 |
| 物療の主目的 | 温熱で関節包を柔軟化・疼痛調整・炎症管理 |
| よく使う評価 | 可動域（外転・外旋・内旋）・夜間痛の有無・インピンジメントテスト |
| 注意点 | 拘縮期は温熱+ストレッチ。急性炎症期は温熱禁忌。石灰化はESWT適応 |
| 運動療法との接続 | ペンジュラム運動・壁つたい・腱板強化 |
| レッドフラッグ | 急速な筋萎縮・強い夜間痛（腫瘍・感染の除外） |

---

## 3. 背部痛

| 項目 | 内容 |
|---|---|
| 主病態 | 脊柱起立筋の緊張・胸椎椎間関節・筋膜疼痛・椎骨圧迫骨折（高齢者） |
| 主な対象組織 | 脊柱起立筋・菱形筋・広背筋・胸椎椎間関節包 |
| 物療の主目的 | 筋弛緩・血流改善・疼痛調整 |
| よく使う評価 | 圧痛点・胸椎可動域・神経症状の有無 |
| 注意点 | 高齢者の背部痛は圧迫骨折を先に除外。肺・腎等の内臓由来も念頭に |
| 運動療法との接続 | 胸椎モビリティ・体幹安定性 |
| レッドフラッグ | 高齢者・骨粗鬆症の軽微な外傷後痛（圧迫骨折）・夜間に増悪する背部痛 |

---

## 4. 腰痛（非特異的腰痛・椎間板ヘルニア・脊柱管狭窄症等）

| 項目 | 内容 |
|---|---|
| 主病態 | 筋緊張・椎間関節炎・椎間板変性・椎間板ヘルニアによる神経根圧迫・中枢性感作 |
| 主な対象組織 | 腸腰筋・脊柱起立筋・多裂筋・腰椎椎間板・椎間関節包 |
| 物療の主目的 | 疼痛調整・深部温熱・筋弛緩・廃用筋の賦活 |
| よく使う評価 | VAS/NRS・前屈後屈・SLRテスト・可動域・日常生活制限 |
| 注意点 | 急性期は温熱禁忌。神経根症状は牽引のリスク・ベネフィットを評価。慢性腰痛は中枢性感作に注意 |
| 運動療法との接続 | 体幹インナーマッスルトレーニング・ヒップヒンジ・姿勢指導 |
| レッドフラッグ | 排尿・排便障害（馬尾）・両下肢筋力低下・夜間痛・発熱 |

---

## 5. 坐骨神経痛様症状

| 項目 | 内容 |
|---|---|
| 主病態 | 椎間板ヘルニアによる神経根圧迫・梨状筋症候群・脊柱管狭窄症 |
| 主な対象組織 | 腰椎椎間板・神経根・梨状筋・坐骨神経 |
| 物療の主目的 | 疼痛調整（神経症状の緩和）・筋弛緩（梨状筋）|
| よく使う評価 | SLRテスト・感覚検査・筋力検査・デルマトーム・歩行評価 |
| 注意点 | 神経症状が進行・拡大している場合は物療より先に医師評価。強い牽引は慎重に |
| 運動療法との接続 | 腰椎安定化・梨状筋ストレッチ・神経モビライゼーション |
| レッドフラッグ | 進行性筋力低下・排尿排便障害 |

---

## 6. 膝痛（半月板・靭帯・変形性膝関節症等）

| 項目 | 内容 |
|---|---|
| 主病態 | 変形性膝関節症・半月板変性・靭帯損傷・滑液包炎・膝蓋腱炎 |
| 主な対象組織 | 大腿四頭筋・膝関節軟骨・半月板・膝蓋靭帯・滑液包 |
| 物療の主目的 | 疼痛調整・廃用筋（大腿四頭筋）の賦活・深部温熱 |
| よく使う評価 | 関節可動域・腫脹・大腿四頭筋筋力・歩行評価 |
| 注意点 | 急性滑膜炎・大きな関節液貯留への温熱は禁忌。変形性膝関節症は「治す」より「機能管理」 |
| 運動療法との接続 | 大腿四頭筋強化・ハムストリングス柔軟性・体重管理 |
| レッドフラッグ | 著明な腫脹（骨折・感染）・急激な可動域消失 |

---

## 7. 股関節周囲痛

| 項目 | 内容 |
|---|---|
| 主病態 | 変形性股関節症・大転子滑液包炎・腸腰筋腱炎・梨状筋症候群 |
| 主な対象組織 | 股関節軟骨・腸腰筋・外旋筋群・腸脛靭帯 |
| 物療の主目的 | 深部温熱・筋弛緩・疼痛調整 |
| よく使う評価 | Thomas test・Patrick test・可動域・歩行評価 |
| 注意点 | 深部にあるため表層温熱では届かない。ラジオ波・超音波1MHzを考慮 |
| 運動療法との接続 | 外転筋強化・体重管理・歩行パターン改善 |
| レッドフラッグ | 鼠径部の強い痛み＋発熱（化膿性関節炎）・股関節の急な動けなさ（脱臼） |

---

## 8. 足関節捻挫

| 項目 | 内容 |
|---|---|
| 主病態 | 外側靭帯（前距腓・踵腓）の部分〜完全断裂・腓骨筋腱損傷 |
| 主な対象組織 | 前距腓靭帯・踵腓靭帯・腓骨筋腱 |
| 物療の主目的 | 急性期：浮腫軽減・疼痛調整。亜急性期以降：組織修復 |
| よく使う評価 | 圧痛部位・不安定性テスト（前方引き出し・距骨傾斜）・腫脹の程度 |
| 注意点 | 腓骨・第5中足骨骨折の除外（Ottawa Ankle Rules）。温熱は急性期禁忌 |
| 運動療法との接続 | バランストレーニング・足関節安定化・テーピング |
| レッドフラッグ | 骨突出部の強い圧痛（骨折の可能性）・荷重不能 |

---

## 9. 肉離れ（筋肉損傷）

| 項目 | 内容 |
|---|---|
| 主病態 | 筋線維の部分〜完全断裂 |
| 主な対象組織 | ハムストリングス・大腿四頭筋・腓腹筋・内転筋群 |
| 物療の主目的 | 急性期：浮腫・疼痛管理。亜急性期：修復促進。慢性期：瘢痕リモデリング |
| よく使う評価 | 損傷グレード評価・触診・可動域・筋力テスト |
| 注意点 | 重度（Grade III）は外科的管理の検討。完全断裂部への直接刺激禁忌 |
| 運動療法との接続 | 段階的ストレッチ・遠心性収縮訓練・競技復帰プログラム |

---

## 10. 打撲（挫傷）

| 項目 | 内容 |
|---|---|
| 主病態 | 軟部組織への直達外力→出血・炎症 |
| 主な対象組織 | 筋・皮下組織・骨膜 |
| 物療の主目的 | 急性期：浮腫・疼痛管理。亜急性期：組織修復補助 |
| 注意点 | 骨折の除外（圧痛・変形確認）。急性期は温熱禁忌 |

---

## 11. 腱障害（腱炎・腱症）

| 項目 | 内容 |
|---|---|
| 代表例 | アキレス腱・膝蓋腱・足底腱膜・棘上筋腱・共通伸筋腱（テニス肘）|
| 主病態 | 急性腱炎（炎症）または慢性腱症（コラーゲン変性）|
| 物療の主目的 | 急性：疼痛調整。慢性：ESWT・超音波パルスで修復促進 |
| 注意点 | 急性腱炎への強い超音波連続波は禁忌。慢性腱症は長期管理の意識が必要 |
| 運動療法との接続 | 偏心性収縮訓練（エクセントリック）が慢性腱症の最重要介入 |

---

## 12. しびれを伴う症状

| 原因分類 | 例 | 物療の方針 |
|---|---|---|
| 神経根圧迫（頚椎・腰椎）| ヘルニア・狭窄症 | 牽引（慎重）・TENS・温熱 |
| 末梢神経絞扼 | 手根管症候群・肘部管症候群 | TENS・温熱・超音波（腱鞘周囲）|
| 循環障害 | レイノー現象・血行不良 | 温熱・電気 |
| 中枢性 | 脳・脊髄疾患 | 物療は補助的。医師の評価が先 |

**注意:** 進行性・急速に拡大するしびれはレッドフラッグ

---

## 13. 急性外傷（一般）

| フェーズ | 対応方針 |
|---|---|
| 直後〜48時間 | PRICE（保護・安静・冷却・圧迫・挙上）。骨折除外が優先 |
| 48時間〜1週 | 冷却から温熱へ段階的移行（炎症症状の程度を見て）|
| 1週以降 | 組織修復促進・運動療法開始 |

---

## 14. 慢性疼痛

| 項目 | 内容 |
|---|---|
| 定義 | 3ヶ月以上続く、または再発を繰り返す疼痛 |
| 特徴 | 中枢性感作・破局化・恐怖回避・社会的影響が大きい |
| 物療の役割 | 「治す」より「活動参加を支える・痛みをコントロールする」 |
| 重要な視点 | 疼痛教育（Pain Neuroscience Education）との統合が不可欠 |
| 注意 | 物療への過剰な依存（passive coping）を作らない |

---

## 15. 高齢者の機能低下関連症状

| 項目 | 内容 |
|---|---|
| 主病態 | サルコペニア・廃用性筋萎縮・変形性関節症・骨粗鬆症 |
| 物療の主目的 | 廃用予防（EMS）・疼痛管理・転倒予防に向けた機能維持 |
| 注意点 | 感覚鈍麻・皮膚脆弱性・複数の禁忌事項（ペースメーカー等）を確認 |
| 運動療法との接続 | レジスタンストレーニング・バランス訓練・ADL指導 |

---

## 16. スポーツ障害

| 分類 | 代表例 | 物療の特徴 |
|---|---|---|
| 急性（外傷）| 捻挫・肉離れ・打撲 | 急性管理→早期復帰プログラム |
| 慢性（障害）| 腱障害・疲労骨折・オーバーユース | 修復促進＋負荷管理 |
| 成長期特有 | オスグッド・踵骨骨端症 | 骨端線注意。過剰刺激禁忌 |
