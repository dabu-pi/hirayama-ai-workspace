## 基本思想

「周波数80Hzだとゲートコントロールになる」を知っているだけでは不十分。
**「なぜそうなるか」を生理学で説明できて初めて、応用できる。**

このドキュメントは専門書ではなく、
**臨床判断に使える整理**を目指す。

## 1. 膜電位と活動電位の超基礎

### 安静時膜電位

- 細胞外はNa⁺が多い、細胞内はK⁺が多い → 電位差が生じる
- 安静時は細胞内が **−70mV**（神経）

### 脱分極と活動電位

- 十分な刺激が加わると → Na⁺チャネルが開く → 細胞内に流入 → 電位が上昇（脱分極）
- 閾値（約 −55mV）を超えると → **活動電位が発生**（全か無かの法則）
- 活動電位は軸索を伝わって → 他の神経・筋へ信号を伝える

### 物療との関係

| 物療の種類 | 生理的な出来事 |
|---|---|
| TENS・低周波 | 外部電気刺激 → 皮膚下の神経の膜電位を変化させる → 活動電位を誘発 |
| EMS | 運動神経または筋線維に閾値以上の刺激 → 活動電位 → 収縮 |
| 超音波 | 機械的刺激 → 細胞膜への物理的作用（脱分極を起こすわけではない） |

## 2. 感覚神経の種類と刺激効果

### 神経線維の分類

| 線維 | 直径 | ミエリン | 速度 | 主な機能 |
|---|---|---|---|---|
| Aα | 大 | あり | 速い | 固有感覚（筋紡錘・腱器官） |
| Aβ | 中 | あり | 速い | 触覚・圧覚・振動覚 |
| Aδ | 小 | 薄くあり | 中 | 鋭い痛み・冷覚 |
| C線維 | 極小 | なし | 遅い | 鈍い痛み・灼熱感・温覚 |

### 刺激強度と反応する線維

```
刺激が弱い ─→ Aβ線維が反応（触覚・圧覚）
           ─→ ゲートコントロール（鎮痛）
刺激が強い ─→ Aδ・C線維も反応
           ─→ 痛みを起こす可能性も
```

**臨床的含意:**
- TENS（ゲートコントロール目的）：Aβが反応する「じんわりビリビリ」程度
- 筋収縮誘発（EMS）：運動神経（Aα）が反応する強さ
- 不快感を伴う強さはC線維を刺激している可能性 → 設定を見直す

## 3. ゲートコントロール理論（疼痛調整）

### 原理

1. 侵害刺激（組織損傷等）がAδ・C線維を興奮させる
2. 侵害刺激が脊髄後角に伝わり、**脳で「痛み」として認知**される
3. 一方、Aβ線維（触覚）への刺激は脊髄後角の**抑制性介在ニューロンを活性化**
4. 抑制性介在ニューロンが痛み伝達を「閉じる」（ゲートを閉める）

### 物療での応用

| 方法 | 設定 | 根拠 |
|---|---|---|
| 高頻度TENS | 80〜150 Hz・感覚刺激レベル | Aβ継続活性化でゲートを閉める |
| 干渉波（高変調周波数） | 80〜100 Hz変調 | 深部でのAβ刺激 |
| マッサージ・温熱 | — | 皮膚・筋のAβ線維刺激 |

**限界:** 即時的・局所的な鎮痛効果。電気を止めると効果は消える傾向。

## 4. 内因性オピオイド系（遅延性鎮痛）

### 原理

1. 低頻度電気刺激（1〜10 Hz）や強刺激が筋収縮を伴う場合
2. 脊髄・脳から **エンケファリン・エンドルフィン・ダイノルフィン** が放出される
3. オピオイド受容体（μ・δ・κ）に結合 → **鎮痛効果**
4. 効果の発現は遅い（20〜30分後）が持続性がある

### 物療での応用

| 方法 | 設定 | 備考 |
|---|---|---|
| 低頻度TENS | 1〜10 Hz・筋収縮を感じる程度の強さ | ナロキソン（オピオイド拮抗薬）で効果が消えることを確認済み |
| 鍼灸 | — | 同様の機序と考えられる |
| 運動 | — | ランニング後の爽快感（エンドルフィン仮説）|

## 5. 運動神経刺激と筋収縮

### 経路

```
電気刺激 → 運動神経の活動電位 → 神経筋接合部 → アセチルコリン放出
→ 筋線維の筋小胞体からCa²⁺放出 → アクチン・ミオシン滑走 → 筋収縮
```

### EMS の設定と筋収縮の関係

| 設定 | 効果 |
|---|---|
| 低頻度（<10 Hz） | 単発収縮（Twitch）。強い収縮は難しい |
| 中頻度（20〜50 Hz） | 強縮（Tetanus）。最大収縮に近い |
| Duty Cycle 1:3〜1:5 | ON:OFF比で筋に回復時間を与える |
| 高強度 | 大きな運動単位（速筋）まで動員 |

**廃用萎縮筋は閾値が高い:**
長期不動の筋は収縮を誘発するのに通常より強い刺激が必要。
強度を徐々に上げ、筋収縮の有無を確認しながら進める。

## 6. 温熱反応の生理

### 組織温度と生体反応

| 温度 | 主な生体反応 |
|---|---|
| 37〜40°C | 通常体温。変化なし |
| 40〜44°C | 血流増加・疼痛閾値上昇・コラーゲン伸張性向上 |
| 44〜47°C | 炎症反応の誘発リスク（長時間NG） |
| 47°C以上 | 組織損傷・熱傷 |

### 温熱の治療効果

| 効果 | 機序 |
|---|---|
| 血流増加 | 血管拡張（一酸化窒素・ヒスタミン産生）|
| 筋弛緩 | 筋紡錘感度低下・抑制系の活性化 |
| 疼痛閾値上昇 | C線維の不活性化・神経伝導速度の変化 |
| コラーゲン伸張性向上 | 43〜45°Cで粘弾性が変化。ストレッチとの組み合わせ効果大 |
| 代謝亢進 | 温度10°C上昇で代謝が2〜3倍（組織修復の加速）|

### 表層温熱 vs 深部温熱

| 種別 | 到達深度 | 機器 | 向いている用途 |
|---|---|---|---|
| 表層温熱 | 皮膚〜1〜2cm | ホットパック・赤外線 | リラクゼーション・前処置 |
| 深部温熱（中周波） | 2〜3cm | ラジオ波 | 深部筋・筋膜・関節周囲 |
| 深部温熱（超音波1MHz） | 3〜5cm | 超音波治療器 | 深部腱・関節包 |

## 7. 循環改善の機序

### 局所血流を増やすメカニズム

| 経路 | 物療 |
|---|---|
| 熱産生 → 血管拡張 | 温熱全般 |
| 筋ポンプ作用 → 静脈還流促進 | EMS・低周波筋収縮モード |
| 軸索反射（皮膚刺激 → 血管拡張）| TENS・低周波 |
| 直接的な血管拡張（生化学的）| 超音波（一部の研究）|

### 浮腫軽減

| 種類 | 機序 | 物療 |
|---|---|---|
| 急性炎症性浮腫 | 血管透過性↑による滲出 | RICE + ハイボルテージ（マイナス極）|
| リンパ浮腫 | リンパ管障害 | 徒手リンパドレナージ主体。超音波補助的 |
| 廃用性浮腫 | 循環低下・重力 | EMS（筋ポンプ）・弾性包帯 |

## 8. 炎症期・修復期・成熟期での生理変化と物療方針

### 炎症期（Inflammatory Phase）

**何が起きているか:**
- 損傷 → 血管反応（充血・透過性増大）→ 発痛物質（ブラジキニン・プロスタグランジン）が放出
- 好中球・マクロファージが集結し壊死組織を処理
- 局所が発赤・熱感・腫脹・疼痛

**物療の方針:**
- 温熱は禁忌（血流をさらに増やし炎症悪化）
- 冷却は炎症初期（48時間以内）の浮腫・疼痛管理に有効
- TENS・ハイボルテージ（マイナス極）は使用可能（疼痛調整・浮腫軽減）

### 増殖期（Proliferative Phase）

**何が起きているか:**
- 肉芽組織形成・コラーゲン合成（Type III→ Type I へ）
- 血管新生
- 組織の「仮の修復」が進む

**物療の方針:**
- 超音波パルス・LIPUS が修復促進の根拠あり
- 低強度電気・MENS（微小電流）が細胞遊走を助けるとされる
- 過度な温熱は修復過程を乱す可能性あり

### 成熟期 / リモデリング期（Remodeling Phase）

**何が起きているか:**
- コラーゲン線維の再配列（機能的方向へ）
- 瘢痕組織の強化・成熟
- 可動域・柔軟性の回復が必要

**物療の方針:**
- 深部温熱 + 牽引・ストレッチ（温めながら伸ばす）
- EMS で筋機能回復
- ショックウェーブ（慢性腱症への）が有用とされる

## 9. 高周波・低周波・中周波・干渉波を生理反応で比較

| 種別 | 皮膚透過 | 深度 | 主な神経反応 | 温熱効果 | 主な用途 |
|---|---|---|---|---|---|
| 低周波（1〜1000Hz） | 皮膚抵抗あり | 表〜中程度 | Aβ・Aδ・C・運動神経 | 少ない | 疼痛調整・筋収縮 |
| 中周波（1〜100kHz） | 抵抗↓・通過しやすい | 深い | 閾値↑・神経刺激しにくい | 少ない | 皮膚透過を利用した深部への媒体 |
| 高周波（>100kHz） | 通過が速い | 深い | 神経への直接刺激は起きにくい | 大きい（ジュール熱）| 深部温熱 |
| 干渉波 | 中周波で透過 | 深い | 交差点で低周波成分 → 深部刺激 | 少ない | 深部の疼痛・循環 |
| ハイボルテージ | 高電圧で皮膚抵抗突破 | 深い | 広範な神経刺激 | 少ない | 浮腫・疼痛・創傷 |
