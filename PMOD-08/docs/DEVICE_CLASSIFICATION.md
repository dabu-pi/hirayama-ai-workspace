## 電気療法系

### 低周波治療器（Low Frequency）
- **周波数:** 1〜1,000 Hz
- **主な効果:** 疼痛調整（ゲートコントロール）、筋収縮誘発、循環改善
- **設定の目安:** 疼痛→高頻度(80〜150Hz)、筋収縮→中周波(20〜50Hz)、内因性オピオイド→低頻度(1〜10Hz)
- **禁忌:** ペースメーカー、心臓部、妊娠腹部、悪性腫瘍部位、感覚障害部位
- **エビデンス:** TENS・EMS共に相当量の研究あり（L1〜L2）

### 中周波治療器（Medium Frequency）
- **周波数:** 1,000〜100,000 Hz（主に2,000〜10,000 Hz）
- **主な効果:** 皮膚インピーダンス低下→深部透過、疼痛調整、筋収縮
- **特徴:** 単独では筋収縮は起こしにくい。干渉波として使用するのが一般的
- **禁忌:** 低周波と同様

### 干渉波治療器（Interferential Current: IFC）
- **原理:** 2つの中周波（例：4,000Hz と 4,100Hz）を交差させ、差分の低周波(100Hz)を深部で生成
- **主な効果:** 深部での疼痛調整・筋収縮、循環改善
- **優位性:** 皮膚抵抗を中周波で突破しつつ、深部で低周波効果を得る
- **禁忌:** 電気療法共通の禁忌

### ハイボルテージ（High Voltage Pulsed Current: HVPC）
- **特徴:** 高電圧（150〜500V）・短パルス（μs単位）・低平均電流
- **主な効果:** 浮腫吸収（マイナス極）、疼痛調整、創傷治癒促進
- **極性:** マイナス極→浮腫吸収・創傷治癒、プラス極→組織収斂
- **禁忌:** 電気療法共通の禁忌

### EMS / NMES（Electrical Muscle Stimulation）
- **目的:** 筋収縮の誘発（廃用予防・筋力増強）
- **適応:** 術後廃用、筋力低下、スポーツリハ
- **設定:** ON:OFF = 1:3〜1:5（筋の回復時間確保）
- **禁忌:** 痙縮筋への過強刺激、関節不安定部位、骨折急性期

## 超音波療法

### 超音波治療器（Therapeutic Ultrasound）
- **周波数:** 1MHz（深部）または 3MHz（表層）
- **モード:** 連続波（温熱効果中心）/ パルス波（非熱的効果・組織修復）
- **主な効果:** 深部温熱（連続）、組織修復促進（パルス・LIPUS）、疼痛調整
- **1MHz vs 3MHz:** 1MHz → 3〜5cm深部到達。3MHz → 1〜2cm表層
- **強度:** 0.5〜2.0 W/cm²（用途・部位で異なる）
- **禁忌:** 眼球、生殖腺、成長骨端、悪性腫瘍部位、脊椎固定術後の金属上（連続波）、妊娠腹部

## ショックウェーブ（ESWT）

### 体外衝撃波治療器
- **種類:** 焦点式（Focused）/ 放射状（Radial）
- **主な適応:** 腱付着部症（足底腱膜炎・膝蓋腱・アキレス腱）、石灰化腱炎、筋膜炎
- **プロトコル:** 通常3〜5回セッション（週1回程度）
- **特徴:** 治療中の痛みが強め。効果は数週間後に出てくることが多い
- **禁忌:** 抗凝固剤使用中、骨端線（小児）、肺・腸管上、悪性腫瘍
- **エビデンス:** 腱障害に対してRCT・メタ解析あり（L1〜L2）。ただしプロトコルにばらつき

## 温熱療法

### ラジオ波（Radio Frequency: RF）
- **周波数:** 300kHz〜数MHz
- **原理:** 高周波電流 → 組織のイオン振動 → ジュール熱（深部温熱）
- **主な効果:** 深部温熱（3〜5cm以上）、筋弛緩、血流改善
- **特徴:** 皮下脂肪が多くても深部に届きやすい
- **禁忌:** 急性炎症、出血リスク、金属インプラント（直上）、感覚障害、ペースメーカー

### ホットパック
- **効果深度:** 表層（皮膚〜1〜2cm）
- **主な効果:** 表層温熱、リラクゼーション、前処置（後の徒手的アプローチへの準備）
- **禁忌:** 急性炎症・出血・感覚障害・循環障害

## 牽引療法

### 腰椎牽引
- **主な適応:** 椎間板ヘルニア・椎間孔狭窄による神経根症状
- **設定:** 間欠牽引 / 持続牽引。体重の25〜50%程度から開始
- **注意:** 不安定性のある腰椎・骨粗鬆症重症・急性椎間板ヘルニアには慎重

### 頚椎牽引
- **主な適応:** 頚椎症・ヘルニアによる上肢症状
- **設定:** 持続 / 間欠。3〜5kgから開始が一般的
- **禁忌:** 環軸関節不安定症（関節リウマチ等）、骨折・脱臼
