# CLINICAL_DECISION_FRAMEWORK — 物療機器 選択・設定の思考手順

**プロジェクトID:** PMOD-08  
**最終更新:** 2026-04-10  
**位置づけ:** 「どの患者に何を使うか」を根拠を持って選択するための思考フレーム

---

## 基本思想

**「機器ありき」で考えない。「患者の問題ありき」で考える。**

最初に「干渉波にしよう」と機器を決めるのではなく、
「この患者の今の状態で、何を変えたいのか」から出発し、逆算で機器・設定を選ぶ。

---

## 思考手順（7ステップ）

### Step 1: 主訴と目標の確認

> **「今日、何を改善したいか」を明確にする**

- 患者の主訴は何か
- 本日のセッションで狙う変化は何か
  - 痛みを減らす
  - 動きを改善する
  - 腫れを引かせる
  - 筋力・活動量を上げる
  - 回復を促進する

---

### Step 2: 病態の分析

> **「体内で何が起きているか」を分析する**

| 観点 | 問い |
|---|---|
| 炎症の有無 | 発赤・熱感・腫脹・疼痛の程度は？（急性炎症か？） |
| 時期 | 急性期（受傷〜48〜72時間）か亜急性期か慢性期か |
| 組織 | どの組織が問題か（筋・腱・靭帯・関節・神経・骨 等） |
| 深度 | 表在性（1〜2cm以内）か深部（3cm以上）か |
| 循環 | 浮腫・虚血・冷感はあるか |
| 神経症状 | しびれ・放散痛・感覚異常はあるか |
| 筋の状態 | 過緊張か廃用・筋力低下か |

---

### Step 3: 物療で変えられる生体反応を選ぶ

> **「変えたい反応」を1〜2つに絞る**

| やりたいこと | 対応する生体反応 |
|---|---|
| 痛みを今すぐ和らげる | 疼痛調整（ゲートコントロール） |
| 痛みの根本から緩和する | 内因性オピオイド放出・炎症抑制 |
| 深部を温めて緩める | 深部温熱 |
| 腫れを引かせる | 循環促進・浮腫吸収 |
| 筋を動かす・鍛える | 筋収縮誘発 |
| 治癒を促進する | 組織修復促進 |

---

### Step 4: 機器の候補を選ぶ

> **Step 3 で選んだ反応を起こせる機器を複数候補として挙げる**

参照: `docs/KNOWLEDGE_MAP.md` の目的別整理

| 変えたい反応 | 機器候補 |
|---|---|
| 疼痛調整（即時・ゲートコントロール） | TENS・低周波（高頻度80〜150Hz）・干渉波 |
| 疼痛調整（オピオイド系） | 低周波（低頻度2〜4Hz）・干渉波（変調周波数低め） |
| 深部温熱 | ラジオ波・超音波（1MHz・連続） |
| 表層温熱 | ホットパック・赤外線・超音波（3MHz） |
| 浮腫吸収・循環 | ハイボルテージ（マイナス極）・低周波・干渉波 |
| 筋収縮誘発 | EMS・低周波（筋収縮モード） |
| 組織修復促進 | 超音波（パルス・低強度）・ショックウェーブ |
| 腱付着部痛 | ショックウェーブ（ESWT） |

---

### Step 5: 禁忌・注意事項の確認

> **候補機器を絞り込む前に禁忌をチェック**

#### 全電気療法共通の禁忌

- ペースメーカー・植込み型除細動器（ICD）装着者
- 心臓部・頚動脈洞付近への通電
- 妊婦（腹部・腰部への通電）
- 悪性腫瘍部位（直上または近傍）
- 感覚障害のある部位（熱傷・電気熱傷リスク）
- 血栓性静脈炎の部位

#### 温熱療法の禁忌

- 急性炎症（熱感・腫脹・発赤が著明な時期）
- 出血リスクのある部位・抗凝固剤使用中
- 感覚鈍麻のある部位
- 金属インプラント直上（超音波は特に注意）

#### ショックウェーブの禁忌

- 血液凝固障害・抗凝固剤使用中
- 肺組織・腸管上の使用
- 小児骨端線付近
- 悪性腫瘍

#### 超音波の禁忌

- 眼球・生殖腺・成長骨端
- 脊椎固定術後（金属上の連続波）
- 人工関節直上（熱産生の注意）

---

### Step 6: 設定の決定

> **機器が決まったら、モード・強度・時間・頻度を決める**

#### 設定決定のチェックリスト

| 設定項目 | 考え方 |
|---|---|
| 周波数 | 疼痛調整（高頻度）か筋収縮（中〜低頻度）かで変わる |
| パルス幅 | 感覚刺激（<100μs）か筋収縮誘発（>200μs）か |
| 強度（出力） | 「感じる程度」か「収縮を起こす程度」かで使い分け。急性期は低め |
| Duty Cycle | 筋収縮誘発は ON:OFF=1:3〜1:5。疼痛調整は連続でも可 |
| 通電時間 | 局所 10〜20分 / 広域 15〜30分 / 急性期は短め（5〜10分） |
| 治療頻度 | 急性期：毎日〜週3 / 慢性期：週2〜3 / 維持期：週1〜 |
| 治療期間 | 急性：1〜3週 / 亜急性：2〜6週 / 慢性：1〜3ヶ月（効果確認しながら） |

---

### Step 7: 反応の確認と再評価

> **実施後に変化を確認し、プロトコルに反映する**

| 確認項目 | 良好なサイン | 要調整なサイン |
|---|---|---|
| 疼痛 | VAS/NRS が下がる・軽くなった感覚 | 増悪・灼熱感・しびれの増強 |
| 動き | 可動域が広がる・動かしやすい | 変化なし・悪化 |
| 筋 | リラックスできる・収縮が引き出せる | 過剰収縮・疲労・不快感 |
| 皮膚 | 変化なし | 発赤・水疱・かゆみ |

**「効かない」と判断する前に確認すること:**
- 電極位置は適切か
- 設定（周波数・パルス幅・強度）は目的に合っているか
- 通電時間は十分か
- 対象組織の深さに届いているか

## 患者説明の基本文例

### 電気療法（TENS・低周波）

> 「電気の刺激を使って、痛みを和らげる治療です。
> 皮膚の感覚神経を刺激することで、脳への痛みの伝わりを抑える効果があります。
> ビリビリとした感覚がありますが、刺激の強さは調整できます。
> 治療中に不快感が強い場合はすぐに教えてください。」

### EMS（筋電気刺激）

> 「電気で筋肉を動かすリハビリです。
> 動かしにくくなっている筋肉を電気の力で収縮させることで、
> 筋力の回復を助けます。ピクピクと筋肉が動く感覚があります。
> 最初は弱い刺激から始めて、徐々に強さを調整します。」

### 超音波療法

> 「超音波の振動を使って深部の組織を温め、血流を改善する治療です。
> 音波なので聞こえることはありませんが、プローブを当てた部分が少しずつ温かく感じてきます。
> 骨や腱に直接働きかけて修復を助ける効果もあります。
> 熱さや痛みを感じたらすぐに教えてください。」

### ショックウェーブ

> 「衝撃波を使って、慢性的な腱の炎症や石灰化を改善する治療です。
> 機械的な刺激で組織の修復を促し、慢性化した痛みに効果が期待できます。
> 治療中は響くような圧力を感じます。痛みの強さに応じて調整します。
> 3〜5回のコースで効果が出てくることが多いです。」

### ラジオ波（深部温熱）

> 「高周波の電流を使って、深部の組織を内側から温める治療です。
> 表面のホットパックと違い、筋肉の奥まで温まります。
> 温かさを感じますが、熱すぎる場合はすぐに教えてください。
> 硬くなった筋肉や関節の柔軟性を改善するのに役立てます。」

## 臨床判断の早見表

### 急性外傷（受傷後48〜72時間）

| 目的 | 推奨 | 注意 |
|---|---|---|
| 疼痛調整 | TENS・低周波（弱め） | 温熱禁忌・強い筋収縮禁忌 |
| 浮腫軽減 | ハイボルテージ（マイナス極） | 炎症悪化に注意 |
| 温熱 | **禁忌** | 炎症が悪化する |

### 慢性腰痛

| 目的 | 推奨 | 設定の方針 |
|---|---|---|
| 深部温熱 | ラジオ波・超音波（1MHz・連続） | 中等度の温感が続く程度 |
| 疼痛調整 | 干渉波・低周波 | 高頻度で感覚刺激 |
| 筋弛緩促進 | 温熱 + 干渉波の組み合わせ | 温めてから電気刺激 |

### 腱障害（慢性）

| 目的 | 推奨 | 設定の方針 |
|---|---|---|
| 組織修復 | ショックウェーブ・超音波（パルス） | ESWT: 3〜5回コース |
| 疼痛調整 | TENS・低周波 | 補助的使用 |
