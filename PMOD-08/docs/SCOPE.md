## このプロジェクトのスコープ

単なる「物療機器の比較」ではなく、以下すべてを統合した学習・実践体系を作る。

| 領域 | スコープ |
|---|---|
| 物療機器 | 整骨院で使う主要な物療機器の原理・設定・選択 |
| 解剖学 | 物療のターゲット組織（皮膚〜骨まで）|
| 生理学 | 電気・温熱・超音波等の生体への作用機序 |
| 病理学 | 組織修復の3フェーズと病態別の考え方 |
| 疼痛科学 | 末梢性・中枢性感作、痛みの現代的理解 |
| 症状・疾患 | 整骨院でよくみる主要な症状・疾患の病態と物療の目的 |
| 臨床判断 | 患者の状態から物療を選択・設定する思考手順 |
| 安全管理 | レッドフラッグと医療連携基準 |
| note・有料教材 | 上記の知識を公開・商品化する戦略 |

## 対象とする疾患・症状（スコープ内）

整骨院でよく見る以下の症状・疾患を体系化する。

| カテゴリ | 疾患・症状 |
|---|---|
| 頚部・肩 | 頚部痛・肩こり・五十肩・腱板障害 |
| 背部・腰部 | 背部痛・腰痛・坐骨神経痛様症状 |
| 下肢 | 膝痛・変形性膝関節症・股関節周囲痛・足関節捻挫 |
| 上肢 | 肘痛・手関節痛・しびれ |
| 軟部組織 | 肉離れ・打撲・腱障害 |
| 全身・慢性 | 慢性疼痛・スポーツ障害・高齢者の機能低下 |
| 急性外傷 | 捻挫・打撲・骨折後（物療適応範囲内）|

## スコープ外（このプロジェクトでは扱わない）

| スコープ外 | 理由 |
|---|---|
| 鍼・灸 | 別の専門性 |
| 手技療法全般 | 別の学習体系が必要 |
| 運動療法の詳細プログラム | 本プロジェクトは「接続」まで扱い、詳細は別案件 |
| 薬物療法 | 医師の領域 |
| 外科的処置 | 医師の領域 |
| 診断行為 | 柔整師の業務範囲外 |

**ただし** 運動療法・生活指導との「接続」は各疾患ドキュメントに含める。  
物療単独で完結させるのではなく、「物療の後に何をするか」まで視野に入れる。

## 学習の優先順序

Phase 1 では以下の順で学ぶことを推奨する。

1. 人体知識の基盤（`docs/HUMAN_BIOLOGY_MAP.md`）
2. 組織の特性（`docs/ANATOMY_AND_TARGET_TISSUES.md`）
3. 物療の生理作用（`docs/PHYSIOLOGY_FOR_MODALITIES.md`）
4. 病態と治癒フェーズ（`docs/PATHOLOGY_AND_HEALING.md`）
5. 疼痛科学（`docs/PAIN_SCIENCE_BASICS.md`）
6. 機器の分類と設定（`docs/KNOWLEDGE_MAP.md` / `docs/DEVICE_CLASSIFICATION.md`）
7. 臨床判断の実践（`docs/CLINICAL_DECISION_FRAMEWORK.md`）
8. 疾患別の適用（`docs/COMMON_CONDITIONS_MAP.md` / `docs/DISEASE_AND_MODALITY_MATRIX.md`）
