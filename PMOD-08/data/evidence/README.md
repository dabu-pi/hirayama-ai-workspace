# data/evidence — エビデンス・文献記録

**最終更新:** 2026-04-10

---

## このフォルダの役割

物療機器・治療プロトコルに関する文献・エビデンスを記録するデータベース。

各ドキュメントの記述根拠をここに蓄積し、AIレビュー時の照合・更新に活用する。

---

## ファイル命名規則

`evidence_[テーマ].md`

例:
- `evidence_tens_pain.md` — TENSの疼痛への効果
- `evidence_eswt_tendinopathy.md` — ESWTの腱障害への効果
- `evidence_ultrasound_tissue_repair.md` — 超音波の組織修復効果

---

## 記録フォーマット

各文献は `evidence_table_template.md` のフォーマットで記録する。
