# data/cases — 症例データベース

**最終更新:** 2026-04-10

---

## このフォルダの役割

自院での治療症例を匿名化して蓄積するデータベース。

実際の症例から「どの機器をどう使ったら、どうなったか」を記録し、
再現性のあるプロトコルを導き出すための一次データ。

---

## フォルダ構成

```
cases/
├── README.md              # このファイル
├── case_template.md       # 症例記録テンプレート
└── anonymized/            # 匿名化済み症例ファイル
    └── case_YYYYMM_NNN.md
```

---

## 使い方

1. `case_template.md` をコピーして `anonymized/case_YYYYMM_NNN.md` として保存
2. `docs/DATA_POLICY.md` の匿名化手順に従って個人情報を除去してから保存
3. ファイル名の NNN は001から連番で付ける（例: case_202601_001.md）

---

## 重要: データ管理ルール

- **このフォルダに個人情報は絶対に含めない**
- push前に必ず個人情報の有無を確認する
- 詳細な院内記録はこのリポジトリ外で別途管理する
- 詳細は `docs/DATA_POLICY.md` を参照
