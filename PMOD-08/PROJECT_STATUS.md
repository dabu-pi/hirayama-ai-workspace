## 今回決めた主要方針

| 方針 | 内容 |
|---|---|
| 正本はローカル | Git管理のこのディレクトリが正本。Markdownに残す |
| 「機器名」ではなく「生体反応」で学ぶ | カタログ比較でなく、体内で何が起きるかで理解する |
| 評価→選択→設定→反応→再評価の一連の流れ | 単なる知識整理でなく臨床の流れで学ぶ |
| データは必ず匿名化 | 症例データの個人情報は絶対に含めない |
| 言い過ぎないエビデンス運用 | 研究上の評価と臨床的実用性を分けて記録する |
| AI検証を継続的に回す | 記述した知識をAIで定期チェックし鮮度を保つ |
