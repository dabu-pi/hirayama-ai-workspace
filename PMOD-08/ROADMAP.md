- [ ] 物療機器の大分類一覧（電気療法・超音波・温熱・機械的刺激 等）の整理

- [ ] `docs/KNOWLEDGE_MAP.md` の初版作成 → 更新
- [ ] `docs/DEVICE_CLASSIFICATION.md` の作成
- [ ] 各機器について「体内で何が起きるか」の整理
- [ ] 目的別マトリクスの作成（疼痛調整・筋収縮・温熱・循環・創傷修復）
- [ ] 周波数と生体反応の関係の整理
- [ ] パルス幅・duty cycle・電流強度の意味と効果の整理
- [ ] 電極配置と通電経路の基礎

### 成果物

- `docs/KNOWLEDGE_MAP.md`（更新）

## Phase 3: 症状別・病態別マニュアル

**目標:** 主要疾患・症状ごとに「何をどう使うか」が即座に引ける実務マニュアル

### 対象症状（優先順）

1. 慢性腰痛
2. 頚部痛・肩こり
3. 膝関節痛（変形性・筋力低下）
4. 腱障害（腱炎・腱付着部症）
5. 急性外傷（捻挫・打撲）
6. 神経症状（しびれ・放散痛）
7. 慢性疼痛（広汎性・中枢性）

### 各症状ドキュメントに含めること

- 病態の概要（物療と関係する観点で）
- 使える機器の候補
- 狙う生体反応
- 設定の方針（モード・出力・時間・頻度）
- 禁忌・注意事項
- 患者説明の例文
- エビデンスレベルの記載

### 成果物

- `data/protocol_database/` 内に症状別プロトコルファイル
- `note/drafts/` に各症状の記事草稿
