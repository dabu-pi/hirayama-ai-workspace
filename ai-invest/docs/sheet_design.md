# 取引記録スプレッドシート設計メモ

最終更新: 2026-03-10
ステータス: 設計中

---

## 目的

Google Sheetsで取引記録・損益管理・パフォーマンス分析を一元管理する。

---

## シート構成案

| シート名 | 内容 |
|---|---|
| 取引記録 | エントリー・エグジットの全取引履歴 |
| 保有一覧 | 現在の保有銘柄・評価損益リアルタイム表示 |
| 損益サマリー | 月次・累計の損益集計 |
| ユニバース | UNIVERSE.md の銘柄リストをシートで管理 |
| スクリーニング | スクリーニング結果の記録 |

---

## 今後の実装方針

- まずスプレッドシートを手動で作成・運用
- 操作に慣れたら GAS で自動化（損益計算・アラートなど）
- Claude API で振り返り補助を自動化するのは Phase3 以降
