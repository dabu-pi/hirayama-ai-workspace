# AI投資プロジェクト

平山克司 AI投資管理プロジェクト。
条件ベースの規律ある運用をAIで補助し、感覚売買を排除する。

---

## プロジェクト目的

- **元手:** 100万円
- **目標:** 半年〜1年で +30%（目標額 130万円）
- **運用方針:**
  - 損失制御を最優先。利益追求より資本保全を先に設計する
  - 感覚売買禁止。すべてのエントリー・エグジットは事前に定義した条件に従う
  - AIは銘柄選定・評価・監視・振り返りの補助に使い、最終判断は人間が行う
- **対象市場:** 日本株（優先）、暗号資産（サブ）
- **主拠点:** ジムPC（`C:\hirayama-ai-workspace\workspace\ai-invest`）

---

## 運用方針サマリー

| 項目 | 方針 |
|---|---|
| 損失上限（1回） | -5%（損切りライン） |
| 損失上限（月次） | -10%（強制休止ライン） |
| 集中度 | 1銘柄に資産の30%以上を入れない |
| エントリー条件 | SCREENING_RULES.md に定義した条件を満たすこと |
| エグジット条件 | 損切り / 利確ラインを事前設定し変えない |
| AIの役割 | 候補選定・条件チェック・ニュース要約・振り返り補助 |
| 最終判断 | 必ず人間が行う |
