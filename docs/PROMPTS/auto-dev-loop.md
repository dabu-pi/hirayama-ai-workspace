# auto-dev-loop.md — 自動開発モード ループ継続プロンプト

2サイクル目以降に貼り付ける短いプロンプトです。
前のサイクルの `## NEXT` セクションの内容を `[NEXT]` に貼り付けてください。

---

```
前のサイクルの NEXT を受けて、次のサイクルを開始してください。

## 引き継ぎ内容（前サイクルの NEXT）

[前サイクルの ## NEXT セクションをここに貼る]

---

## 確認事項

- ディレクトリ: `workspace/`（本番）または `claude-sandbox/`（実験）
- 実行: `rwl` 経由必須
- メモ: `note` コマンドで保存（1サイクル最低1件）
- git: 作業ブランチ上で作業、完了後 master へマージ

## 出力フォーマット（毎回）

```
## PLAN
## CHANGES
## COMMANDS
## NOTES
## NEXT
```

STOP 条件に該当したら即座に中断して報告してください。
（認証情報 / 本番 API POST / 仕様不明 / 同じエラー3回 / スコープ拡大 / テスト失敗）

次のサイクルを開始してください。
```
