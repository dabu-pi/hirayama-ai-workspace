# AUTO_LOOP.md — AI Development Automatic Loop

`scripts/auto-loop.ps1` の設計・使用ガイド

作成: 2026-03-06

---

## 概要

`auto-loop.ps1` は `auto-dev.ps1` の薄いラッパーであり、
AI 開発環境の「最終仕上げ」として以下を付加する。

| 機能 | 内容 |
|---|---|
| STOP チェック | 実行前に危険なコマンド・認証情報・破壊操作をブロック |
| 結果分類 | exit code + ログから `SUCCESS / WARNING / FAILURE` を判定 |
| サイクルサマリー | `logs/notes/cycle_TIMESTAMP.md` に全結果を保存 |
| 構造化出力 | `RESULT / LOG PATHS / ERROR SUMMARY / FIX NEXT / GIT NEXT` |

---

## 設計原則

1. **薄いラッパー** — `auto-dev.ps1` の投資を完全に活かす。ロジック重複なし。
2. **exit 99 = STOP** — コマンド失敗 (exit 1) と明確に区別する。
3. **英語のみのソース** — PS 5.1 エンコード安全のため、日本語・絵文字をソースに埋め込まない。
4. **常に workspace モード** — `-Mode sandbox` を明示した場合のみ破壊操作チェックを緩和。
5. **自動送信・自動削除は実装しない** — `auto-dev.ps1` の設計原則を引き継ぐ。

---

## サイクルフロー

```
auto-loop.ps1
│
├─ [1] STOP check
│       コマンド文字列 / コミット対象ファイル / ワークスペースパターンを検査
│       → 検知: exit 99 (STOP), メッセージ出力
│       → 通過: 次へ
│
├─ [2] ログスナップショット
│       実行前の logs/run/*.log と artifacts/*.txt を記録（新旧判別に使用）
│
├─ [3] auto-dev.ps1 呼び出し
│       ハッシュテーブルスプラッティングで引数を引き渡し
│       → rwl.ps1 → note.ps1 → (任意) git-safe-commit.ps1
│
├─ [4] 新規ログ検出
│       実行後のファイルをスナップショットと比較し、今回生成分だけを取得
│
├─ [5] 結果分類
│       exit code == 0 かつ WARNING パターンなし → SUCCESS
│       exit code == 0 かつ WARNING パターンあり → WARNING
│       exit code != 0                           → FAILURE
│
├─ [6] コミットハッシュ取得（-Commit 時）
│
├─ [7] サイクルサマリー保存
│       logs/notes/cycle_YYYYMMDD_HHmmss.md
│
└─ [8] 構造化出力
        RESULT / LOG PATHS / ERROR SUMMARY（FAILURE時） / FIX NEXT / GIT NEXT
```

## STOP 条件

### コマンドパターン（`$STOP_CMD_PATTERNS`）

| パターン | 意味 |
|---|---|
| `api[_-]?key\s*=` | APIキーのインライン指定 |
| `(client\|app)[_-]?secret\s*=` | クライアントシークレット |
| `password\s*=` | パスワードのインライン指定 |
| `\.env\b` | .env ファイル参照 |
| `service_account` | サービスアカウントJSON |
| `credentials\.json` | OAuth認証情報 |
| `DELETE\s+FROM` | SQL DELETE |
| `DROP\s+(TABLE\|DATABASE\|SCHEMA)` | SQL DROP |
| `rm\s+-rf\s+[/\\]` | ルートからの再帰削除 |
| `Format-Volume` | Windowsディスクフォーマット |
| `Clear-Disk` | Windowsディスクワイプ |

### コミットファイルパターン（`$STOP_COMMIT_PATTERNS`）

`.env`, `service_account.json`, `credentials.json`, `token.json`,
`.pem`, `.key`, `id_rsa`, `id_ed25519`, `.pfx`, `.p12`

### workspaceモード追加パターン（`$STOP_WORKSPACE_PATTERNS`）

`Remove-Item -Recurse -Force`, `rmdir /s`, `del /s`

## 結果分類

| 分類 | 条件 | exit code |
|---|---|---|
| `SUCCESS` | exit 0 + WARNINGパターンなし | 0 |
| `WARNING` | exit 0 + WARNINGパターンあり | 0 |
| `FAILURE` | exit != 0 | 1 |
| `STOP` | 事前チェックで検知 | 99 |

WARNINGパターン: `DeprecationWarning`, `UserWarning`, `FutureWarning`,
`PendingDeprecationWarning`, `RuntimeWarning`, `[STDERR].*[Ww]arning`

## パラメータ一覧

| パラメータ | 型 | 必須 | デフォルト | 説明 |
|---|---|---|---|---|
| `-Cmd` | string | ✅ | — | 実行するコマンド |
| `-Project` | string | | `""` | プロジェクト名（サマリーに記録） |
| `-Note` | string | | `""` | 作業メモ（notesログに保存） |
| `-Plan` | string | | `""` | 今サイクルの目的（サマリーに記録） |
| `-Mode` | string | | `workspace` | `workspace` or `sandbox` |
| `-Tag` | string | | `done` | notesログのタグ |
| `-Commit` | switch | | false | 成功時にgit-safe-commit.ps1を実行 |
| `-CommitMsg` | string | | `""` | コミットメッセージ |
| `-CommitFiles` | string[] | | `@()` | ステージ対象ファイル |
| `-Push` | switch | | false | コミット後にプッシュ |
| `-LogDir` | string | | `logs` | ログルートディレクトリ |
| `-SkipStopCheck` | switch | | false | STOPチェックをスキップ（安全確認済みの場合のみ） |

## 使用例

```powershell
# ── 基本 ──────────────────────────────────────────────────────────────
# smoke test
.\scripts\auto-loop.ps1 -Project "smoke" -Cmd "echo hello" -Note "T-1"

# pytest
.\scripts\auto-loop.ps1 -Project "freee-automation" `
    -Cmd "python -m pytest tests/ -v" `
    -Note "TC-01 pass" `
    -Plan "Phase2 partner resolution test"

# ── コミット付き ───────────────────────────────────────────────────────
.\scripts\auto-loop.ps1 -Project "freee-automation" `
    -Cmd "python -m pytest tests/ -v" `
    -Note "all pass" -Tag test `
    -Commit -CommitMsg "test: all pass" `
    -CommitFiles @("src/freee.js") -Push

# ── sandbox モード ─────────────────────────────────────────────────────
.\scripts\auto-loop.ps1 -Project "experiment" -Cmd "python run.py" -Mode sandbox

# ── STOP スキップ（確認済みの場合のみ） ───────────────────────────────
.\scripts\auto-loop.ps1 -Cmd "safe-cmd" -SkipStopCheck
```
