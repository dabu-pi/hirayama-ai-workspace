**対象:** 頚肩こり_初期評価モジュール（Phase 2）
**最終更新:** 2026-03-26
**ステータス:** 設計フェーズ（実装前）

> 腰痛版コメントは `COMMENT_DESIGN.md` を参照。本ファイルは頚肩こり専用。
> 共通コメント（緊急赤旗）は COMMENT_DESIGN.md と同キー体系を踏まえつつ、
> 頚肩こり固有の文言に差し替える。

---

## コメント構造の方針

- コメントは腰痛版と共通化しない（シートを別にし、文言も別に持つ）
- キー命名規則は `NS_` プレフィックスで腰痛版フラグと区別する
- カテゴリは腰痛版と同じ8カテゴリ構造で統一する（拡張時のパターンを揃えるため）
- コメントはすべて「施術者が読むもの」「患者が読むもの」を明確に分ける

```
頚部特有の危険所見が含まれているため、施術開始前に医療連携の必要性を判断してください。
```

```

神経症状は {神経症状レベル}。ROM は {ROM制限型}。
本日の施術は保留または最小限とし、整形外科受診を優先してください。
```

### KEY: NS_RADICULOPATHY_SUMMARY
```

ROM は {ROM制限型}。日常生活負荷は {生活負荷フラグ}。
発症から {発症期間} 経過。施術は神経根刺激回避を前提に進めてください。
```

### KEY: NS_CHRONIC_LIFE_SUMMARY
```
評価まとめ: NRS {NRS} 点。発症から {発症期間} 経過の慢性期です。
生活負荷が高い状態（{生活負荷概要}）が症状の持続要因になっている可能性があります。
施術と生活習慣改善を並行して進める方針を検討してください。
```

### KEY: NS_CHRONIC_SUMMARY
```
評価まとめ: NRS {NRS} 点。発症から {発症期間} 経過の慢性期です。
神経症状なし。ROM は {ROM制限型}。
セルフケア習慣化と姿勢再教育を軸に進めると改善しやすい状態です。
```

### KEY: NS_LIFESTYLE_SUMMARY
```
評価まとめ: NRS {NRS} 点。症状の主な持続要因として生活負荷が高い状態です（{生活負荷概要}）。
ROM は {ROM制限型}。神経症状なし。
施術と並行して、負荷軽減指導・姿勢再教育を早期に入れることを推奨します。
```

### KEY: NS_STANDARD_SUMMARY
```
評価まとめ: NRS {NRS} 点。神経症状なし。ROM は {ROM制限型}。
発症から {発症期間} 経過。生活負荷は {生活負荷フラグ}。
施術中心で進めながら、セルフケアを段階的に導入してください。
```

## カテゴリ 2: 注意すべき所見（施術者向け）

### KEY: NS_REDFLAG_CAUTION
```
頚部特有の危険所見（{陽性項目}）が報告されています。
施術前に受診・精査を検討してください。施術による症状悪化のリスクを説明した上で、患者の同意を確認してから進めてください。
```

### KEY: NS_MYELOPATHY_CAUTION
```
頚髄症が疑われます（手の巧緻性低下 / 両側放散）。
頚部への強い手技・牽引・急激な可動域操作は禁忌に準じます。
整形外科（頸椎専門）への紹介を強く推奨します。
```

### KEY: NS_RADICULOPATHY_CAUTION
```
神経根障害の可能性があります（{放散方向} / しびれ分布: {しびれ分布}）。
施術後に上肢症状が増悪する場合は施術を中止し再評価してください。
3〜4回で改善傾向がない場合は医療連携を検討します。
```

### KEY: NS_LIFESTYLE_CAUTION
```
生活負荷が高い状態（PC {PC時間} / スマホ {スマホ時間} / 運転 {運転時間}）が症状の持続要因として疑われます。
施術だけでは根本改善しにくい状態です。負荷軽減の指導を早期に組み込んでください。
```

### KEY: NS_STANDARD_CAUTION
```
現時点で緊急性の高い所見はありません。
症状の悪化・新たな神経症状（しびれ・筋力低下）が出現した場合は即座に再評価してください。
```

## カテゴリ 3: 初回説明の方向性（施術者→患者）

### KEY: NS_MYELOPATHY_EXPLAIN
```
説明の方向性: まず神経系に関わる可能性があることを分かりやすく伝え、精査の必要性を説明してください。「今すぐ危険な状態ではないかもしれないが、確認が必要」というトーンで伝えると患者が受け入れやすいです。
```

### KEY: NS_RADICULOPATHY_EXPLAIN
```
説明の方向性: 「首の神経が少し刺激を受けている可能性がある」と伝え、避けるべき動き・姿勢を具体的に説明してください。「安静にし過ぎず、神経の刺激になる動きだけ避ける」という伝え方が有効です。
```

### KEY: NS_CHRONIC_EXPLAIN
```
説明の方向性: 「痛みがある程度続いているのは、身体の使い方と生活習慣が関係していることが多い」と伝えてください。「施術で症状を緩和しながら、再発しにくい首・肩の使い方を一緒に覚えていく」という方針を共有すると納得されやすいです。
```

### KEY: NS_LIFESTYLE_EXPLAIN
```
説明の方向性: 「今の仕事・生活のパターンが首肩への負担を高めている可能性がある」ことを具体的に伝えてください。責める表現ではなく「少し工夫するだけで変わりやすい部分がある」という前向きなトーンで話すと受け入れやすいです。
```

### KEY: NS_STANDARD_EXPLAIN
```
説明の方向性: 「なぜ首肩が凝るのか」の原因を分かりやすく伝え、「動かし方・姿勢の見直しで改善できる部分が大きい」ことを伝えてください。施術と自分でできることの両輪で進める方針を共有してください。
```

## カテゴリ 4: 施術の優先順位（施術者向け）

### KEY: NS_MYELOPATHY_PRIORITY
```
施術の優先順位: 施術保留または最小限（頚部への強い刺激・牽引・操作を避ける）。
安全確認と医療連携が最優先。
```

### KEY: NS_RADICULOPATHY_PRIORITY
```
施術の優先順位: 神経根への刺激を避けた筋緊張緩和 → 症状緩和後に可動域改善。
伸展や同側への側屈・回旋は慎重に。牽引は症状に応じて判断。
```

### KEY: NS_CHRONIC_LIFE_PRIORITY
```
施術の優先順位: 筋緊張緩和 + 生活指導（負荷軽減） → 姿勢再教育 → セルフケア定着。
施術だけで終わらせず、1回あたり5分程度の指導時間を設けることを推奨。
```

### KEY: NS_STANDARD_PRIORITY
```
施術の優先順位: 主訴部位（{主症状}）の緊張緩和 → 可動域改善 → 姿勢・動作の再教育。
症状が強ければ緩和優先。落ち着いたらセルフケア導入に移行。
```

## カテゴリ 5: セルフケア・運動療法の方向性（施術者→患者）

### KEY: NS_MYELOPATHY_SELFCARE
```
セルフケア: 施術者の指示に従うこと。自己判断での首のストレッチ・体操は控えてください。精査結果が出てから判断します。
```

### KEY: NS_RADICULOPATHY_SELFCARE
```
セルフケア: 首に負担のかかる姿勢（うつぶせ・長時間の下向き）を避けてください。
チンタック（あご引き運動）は、症状が増悪しなければ1日5〜10回から試せます。
痛みが増す場合はすぐ中止し、次回来院時に報告してください。
```

### KEY: NS_CHRONIC_SELFCARE
```
セルフケア: チンタック（あご引き）+ 肩甲骨リトラクション（肩甲骨を引く動作）を1日2〜3回、各10回を目安に行ってください。
PC・スマホ時の姿勢を少しずつ改善していくことが長期的な改善につながります。
「完璧にやろう」とせず、まず1種目から習慣化することを優先してください。
```

### KEY: NS_LIFESTYLE_SELFCARE
```
セルフケア: まず1時間に1回、30秒でいいので首・肩を動かす習慣をつけてください（立つ・伸ばす・あご引きなど）。
スマホ・PC の持ち方・高さを少し調整するだけでも首への負担が変わります。
運動療法は体調を見ながら段階的に追加していきます。
```

### KEY: NS_STANDARD_SELFCARE
```
セルフケア: チンタック（あご引き）を1日3〜5回、各10回から始めてください。
気になるときに「首を軽く動かす」習慣をつけることが大切です。
長時間同じ姿勢を避けることも合わせて意識してください。
```

## カテゴリ 6: 再評価時に見るべきポイント（施術者向け）

### KEY: NS_MYELOPATHY_REASSESS
```
再評価: 手の巧緻性・握力・上肢放散の変化を確認。
精査の結果と紹介先からの情報を確認の上、施術継続可否を判断してください。
```

### KEY: NS_RADICULOPATHY_REASSESS
```
再評価: 上肢放散・しびれの範囲・強さの変化を確認。
NRS の変化 + 施術後の神経症状増悪有無を確認してください。
3〜4回で改善傾向がなければ医療連携を検討。
```

### KEY: NS_CHRONIC_REASSESS
```
再評価: NRS の変化 + ROM の変化 + セルフケア実行状況を確認。
「生活負荷の改善が進んでいるか」「症状の時間的パターンに変化があるか」を毎回確認してください。
```

### KEY: NS_STANDARD_REASSESS
```
再評価: NRS の変化 + 主訴部位の緊張・ROM・日常動作への影響を確認。
新たな神経症状の出現がないか毎回確認してください。
```

## カテゴリ 7: 患者さんへの説明文（要約・患者向け）

### KEY: NS_MYELOPATHY_PATIENT
```
今日の評価で、首の神経に少し注意が必要な所見がありました。詳しく調べてもらう必要があるため、専門の病院への受診をお勧めします。
```

### KEY: NS_RADICULOPATHY_PATIENT
```
首の神経が少し刺激を受けている可能性があります。腕のしびれや痛みが強くなる動きは避けながら、少しずつ改善を目指していきましょう。
```

### KEY: NS_CHRONIC_PATIENT
```
首・肩の症状が続いているのは、身体の使い方や日常の習慣が関係していることが多いです。施術でつらさを和らげながら、少しずつ再発しにくい使い方を身につけていきましょう。
```

### KEY: NS_LIFESTYLE_PATIENT
```
お仕事やスマホの使い方が首・肩への負担を高めている可能性があります。ちょっとした姿勢の工夫と簡単な体操で、症状が落ち着きやすくなります。一緒に取り組んでいきましょう。
```

### KEY: NS_STANDARD_PATIENT
```
首・肩の症状は、姿勢や筋肉の疲れが積み重なって起きていることが多いです。施術で緊張をほぐしながら、日常生活の中でできるセルフケアも取り入れていきましょう。
```

## カテゴリ 8: 医療連携を考えるべき条件（施術者向け）

### KEY: NS_MYELOPATHY_REFERRAL
```
【医療連携: 強く推奨】
頚髄症の可能性があります（手の巧緻性低下 / 両側上肢放散）。
整形外科（頸椎専門・MRI施設）への紹介を今すぐ検討してください。
施術は精査結果が判明するまで最小限にとどめてください。
```

### KEY: NS_REDFLAG_REFERRAL
```
【医療連携: 要検討】
危険所見（{陽性項目}）があります。
整形外科・内科への受診を勧奨し、経過を確認してから施術を継続するか判断してください。
```

### KEY: NS_RADICULOPATHY_REFERRAL
```
【医療連携: 経過観察】
神経根障害の可能性があります。
3〜4回の施術で上肢症状の改善傾向がない場合、または症状が増悪した場合は整形外科への紹介を検討してください。
```

### KEY: NS_STANDARD_REFERRAL
```
現時点で医療連携が必要な所見はありません。
以下の場合は即座に再評価・受診勧奨を検討してください:
- 上肢の力が急に落ちた / しびれが急に強くなった
- 頭痛が急に悪化した
- 飲み込みにくい・声がおかしいなどの症状が出た
```

## コメントマスタシートの構成

頚肩こり_コメントマスタ シートは以下の列構成とする。

| 列 | 内容 |
|---|---|
| A | コメントキー（例: NS_STANDARD_SUMMARY） |
| B | カテゴリ番号（1〜8） |
| C | カテゴリ名 |
| D | コメント本文 |
| E | 対象読者（施術者 / 患者） |
| F | 有効フラグ条件（メモ用） |
| G | 最終更新日 |

腰痛版の `コメントマスタ` シートとは別シートとして独立させる（行追加はしない）。
