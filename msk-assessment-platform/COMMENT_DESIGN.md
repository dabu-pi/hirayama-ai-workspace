## コメント階層構造

```
共通コメント（全症状で使える）
  ├── 赤旗アラート（CAUDA_EMERGENCY / REDFLAG_ALERT）
  ├── 医療連携を考えるべき条件（全症状共通テンプレート）
  └── PSFS変化の評価コメント

腰痛モジュール固有コメント
  ├── 評価まとめ（腰痛版）
  ├── NRS・RMDQ・STarT に基づく注意所見
  ├── 腰痛の説明方向性
  ├── 腰痛施術の優先順位
  ├── 腰痛向けセルフケア・運動療法
  ├── 腰痛再評価ポイント
  └── 腰痛患者向け説明文
```

将来モジュール追加時は「頸肩固有コメント」「膝固有コメント」を同じカテゴリ構造で追加する。

## コメント出力の8カテゴリ

評価入力が完了すると、以下の6種類のコメントが自動生成される。
各コメントはコメントマスタ（シート6）から条件キー（フラグ）で選択される。

| # | カテゴリ | 対象読者 | 役割 |
|---|---|---|---|
| 1 | 評価まとめ | 施術者 | 今回の評価結果の要点整理 |
| 2 | 注意すべき所見 | 施術者 | 見逃してはいけないリスク・サイン |
| 3 | 初回説明の方向性 | 施術者 | 患者への説明で強調すべきこと |
| 4 | 施術の優先順位 | 施術者 | 今回の施術で何を中心にすべきか |
| 5 | セルフケア・運動療法の方向性 | 施術者→患者 | 家でやること・動き方の指導内容 |
| 6 | 再評価時に見るべきポイント | 施術者 | 次回評価での確認ポイント |
| 7 | 患者さんへの説明文（要約） | 患者 | そのまま患者に読ませられる要約 |
| 8 | 医療連携を考えるべき条件 | 施術者 | 紹介・受診勧奨の判断基準 |

## コメントテンプレート一覧

### カテゴリ1: 評価まとめ

#### KEY: 標準（赤旗なし・神経症状なし）
```
評価まとめ: NRS{NRS現在}点（{強度}）、RMDQ-10 {RMDQ}点（{機能}障害）、
STarT {STarT}点（{リスク}リスク）。
発症から{期間}経過。動作パターンは{動作パターン}。
現時点での主な問題は{痛み/機能障害/慢性化リスク}です。
```

#### KEY: REDFLAG_ALERT
```
評価まとめ: 赤旗スクリーニングで注意すべき所見が{赤旗スコア}件あります。
施術を開始する前に、医療連携の必要性を確認してください。
（{赤旗陽性項目リスト}）
```

#### KEY: CAUDA_EMERGENCY
```
【緊急】馬尾症候群が疑われます。
排尿・排便障害 / 会陰部感覚異常が報告されています。
本日の施術は行わず、速やかに整形外科・脊椎外科への紹介を検討してください。
```

### カテゴリ2: 注意すべき所見

#### KEY: 標準
```
現時点で特に緊急性の高い所見はありません。
ただし、症状の悪化・新たな神経症状出現時は即座に再評価してください。
```

#### KEY: NERVE_SEVERE
```
重度の神経症状があります（{神経症状詳細}）。
下肢筋力低下・両側SLR陽性は神経障害の進行を示唆します。
施術による悪化のリスクがあるため、施術内容を慎重に選択し、
整形外科的精査（MRI等）を強く推奨します。
```

#### KEY: NERVE_MOD
```
神経根障害の可能性があります（SLR陽性: {陽性側}、デルマトーム: {部位}）。
施術後に下肢症状が増悪する場合は施術を中止し再評価してください。
3〜4回の施術で改善傾向がない場合は医療連携を検討します。
```

#### KEY: START_HIGH
```
慢性化リスクが高い状態です（STarT {STarT}点）。
「動くのが怖い」「治らないかもしれない」という思考が症状を維持している可能性があります。
身体的アプローチと同時に、患者さんの認知・行動面への介入が重要です。
```

#### KEY: FALL_HIGH
```
転倒リスクが高い状態です（{転倒リスク要因}）。
施術台の昇降・診察室内での移動に注意が必要です。
転倒予防の指導・環境整備の提案も治療計画に含めてください。
```

### カテゴリ3: 初回説明の方向性

#### KEY: 標準（急性〜亜急性・STarT低）
```
説明の方向性: 「なぜ痛いのか」の原因を分かりやすく説明し、
「回復の見通し（多くは4〜6週で改善する）」を伝えることが重要です。
過度な安静は逆効果であること、動ける範囲で動くことを推奨してください。
「痛みがあるからこそ、正しく動く練習が必要」という文脈で伝えると納得されやすいです。
```

#### KEY: START_HIGH（行動変容重視）
```
説明の方向性: 「痛みは身体のSOSであり、避けるほど悪化する」という認知の修正が最優先です。
「痛みがあっても、適切に動くことが回復を早める」ことを繰り返し説明してください。
患者さんの不安・心配事を先に聞き取り、それに答える形で説明を組み立てると有効です。
「一緒に改善を目指す」という二人三脚のスタンスを強調してください。
```

#### KEY: CHRONIC（慢性期）
```
説明の方向性: 「完全に痛みをなくすこと」を目標にするより、
「痛みをコントロールしながら日常を取り戻すこと」を目標として共有してください。
「痛みと付き合いながら、できることを増やしていく」という慢性疼痛管理の考え方を、
専門用語を使わずに伝えることが大切です。
```

#### KEY: REDFLAG_ALERT
```
説明の方向性: 今日の評価で気になる点があるため、
「専門医で精密検査を受けることをお勧めしたい」と伝えてください。
「施術を止める」のではなく「より安全に治療するために」という文脈で話すと、
患者さんが受け入れやすくなります。
```

### カテゴリ4: 施術の優先順位

#### KEY: NRS_HIGH（疼痛管理優先）
```
施術優先順位:
1. まず痛みを和らげること（疼痛管理）
2. 痛みが下がったら動作範囲を広げること（機能改善）
3. 安定したらセルフケア・運動療法へ移行
今回は疼痛緩和を最優先に。手技・物理療法で症状を落ち着かせてから機能改善へ進めてください。
```

#### KEY: NERVE_MOD〜SEVERE（神経根優先）
```
施術優先順位:
1. 神経根の除圧・刺激軽減を最優先
2. 過剰なストレッチ・強圧手技は禁忌
3. 症状の増悪がないことを毎回確認
腰部の牽引・ポジショニング・筋緊張緩和を中心に。下肢症状の変化を毎回記録してください。
```

#### KEY: START_HIGH（行動変容優先）
```
施術優先順位:
1. 患者さんとの関係構築・信頼形成を最優先
2. 評価結果の共有と目標設定（PSFS目標の合意）
3. 段階的な活動増加プログラムの設計
4. 手技は「症状を取る」より「動ける体験を作る」ことを目的に
施術そのものより「できた体験を増やす」ことがこの時期の最優先事項です。
```

#### KEY: FALL_HIGH（ADL・転倒予防優先）
```
施術優先順位:
1. 転倒リスク低減（バランス・下肢筋力強化）
2. 日常動作（移乗・立ち上がり）の改善
3. 疼痛管理
施術内容だけでなく、自宅環境の確認（手すり・段差・滑り止め）も重要です。
```

### カテゴリ5: セルフケア・運動療法の方向性

#### KEY: 標準（急性期）
```
セルフケア方向性:
- 安静より「動ける範囲での活動継続」を推奨
- 横になり続けるのは逆効果。短時間の歩行を1日数回
- ヒートパック（温罨法）: 慢性期の筋緊張緩和に有効
- アイシング: 急性炎症が強い場合のみ（15〜20分）
- 睡眠姿勢: 横向き・膝の間にクッションが腰への負担を減らす
```

#### KEY: 機能改善期（発症4〜6週以上・NRS中以下）
```
セルフケア方向性:
- キャット＆カウ（猫背・反り腰の繰り返し）: 腰椎モビライゼーション
- ドローイン: 腹横筋の活性化（痛みが出ない範囲で）
- ヒップヒンジ練習: 腰でなく股関節を使った動作パターンの再学習
- 歩行: 1日20〜30分を目標に段階的に増やす
- 座り方: 骨盤を立てて座る練習（タオルを坐骨下に置く等）
```

#### KEY: CHRONIC（慢性期・再発予防）
```
セルフケア方向性:
- 有酸素運動（ウォーキング・水中歩行）: 週3〜5回・30分目標
- コアスタビリティ練習: ドローイン → バードドッグ → ブリッジと段階的に
- ストレッチ: 股関節前面（腸腰筋）・梨状筋・ハムストリングスを重点的に
- 動作パターン改善: 前屈時は腰から曲げずに股関節から折る習慣化
- 生活習慣: 同一姿勢を20〜30分以上続けない。立ち座りの頻度を増やす
長期目標: 「再発しにくい身体」作り。痛みが出てから来院するパターンを変えていく。
```

#### KEY: START_HIGH（行動変容優先）
```
セルフケア方向性:
- 最初は「できそうな小さなこと」から始める（ハードルを極限まで下げる）
- 「やってみたら痛みが増えた→体が壊れた」ではなく「使っていない筋肉が反応した」と伝える
- セルフケア日誌（簡易版）: 何をやったか・痛みがどう変わったかを記録させる
- 達成感の積み重ねが行動変容の鍵
```

### カテゴリ6: 再評価時に見るべきポイント

#### KEY: 標準
```
再評価ポイント:
- NRS（現在・最悪・最良）の変化
- RMDQ-10の変化（-3点以上で有意な改善）
- PSFS: 指定した3活動の変化（+2点以上が目標）
- 新たな神経症状の出現がないか
- セルフケアの実施状況・阻害要因
```

#### KEY: START_HIGH
```
再評価ポイント:
- STarTスコアの変化（特にQ3「痛みのせいで動くのが怖い」「治らない」の変化）
- PSFS活動の変化（小さな達成があれば必ず言語化する）
- 「できたこと」の確認。できていないことより「できたこと」に注目
- 患者さんの不安・心配事の変化
```

#### KEY: NERVE_MOD〜SEVERE
```
再評価ポイント:
- 下肢放散痛の範囲・強度の変化
- 下肢筋力の変化（悪化していないか）
- SLRテストの変化
- 排尿・排便に問題が出ていないか（必ず毎回確認）
```

### カテゴリ7: 患者さんへの説明文（要約）

#### KEY: 標準（急性〜亜急性・STarT低）
```
患者さんへの説明文:
「今日の評価では、痛みの強さは{NRS}点で{強度}の状態です。
腰の動きは{動作評価まとめ}の制限があります。
今の痛みの原因は{原因概要}によるものと考えられます。
多くの場合、{期間見通し}で症状は改善していきます。
今すぐ安静にしすぎるより、無理のない範囲で動き続けることが回復を早めます。
次回は{再評価予定}に評価しますので、その時に変化を確認しましょう。」
```

#### KEY: CHRONIC + START_HIGH
```
患者さんへの説明文:
「腰痛が{期間}以上続いている状態で、今日の評価では慢性化リスクが高めの結果でした。
これは「治らない」という意味ではなく、「身体だけでなく、ストレスや不安が痛みに影響している」ということです。
慢性腰痛は、適切なケアと少しずつ動く練習を続けることで、確実に改善できます。
一緒に取り組んでいきましょう。今日はまず{目標活動}から始めてみましょう。」
```

### カテゴリ8: 医療連携を考えるべき条件

#### KEY: 常時表示（全評価に含める）
```
医療連携を検討すべき条件:
【即日紹介】
□ 排尿・排便障害 または 会陰部感覚異常（馬尾症候群疑い）
□ 進行性の下肢筋力低下

【早期紹介（1週間以内）】
□ 安静時痛・夜間痛 + 体重減少（悪性腫瘍疑い）
□ 発熱 + 背部痛（感染性脊椎炎疑い）
□ 外傷後の強い腰痛（骨折疑い）

【経過観察後の紹介（4〜6週で改善なし）】
□ 神経根症状が継続・増悪
□ RMDQ・NRSが改善しない
□ STarTスコアが高値のまま改善しない
```
