## 共通ロジック（全症状で使う）

以下は腰痛以外のモジュールでも同様に使われる共通判定ロジック。

### 共通: 赤旗チェック（最優先・全症状共通）

すべての症状評価で最初に実行される共通判定。

```
FLAG_CAUDA（馬尾）, FLAG_REDFLAG（赤旗1件以上）
→ 詳細は腰痛モジュール内に実装（他モジュールでも同フラグ名を使う）
```

### 共通: NRS強度判定

```
NRS 0〜3 = 軽度 / 4〜6 = 中等度 / 7〜10 = 重度
（腰痛・頸肩・膝すべてで同一）
```

### 共通: 転倒リスク判定

移乗動作評価から算出。高齢者評価モジュールでも流用。

### 共通: PSFS変化判定

```
PSFS +2以上 = 有意な改善（MCID達成）
（全症状共通のアウトカム指標）
```

## 判定3層構造の概念

```
Layer 1: ルールベース判定（GAS自動）  ← Phase 1〜2 で実装
Layer 2: 施術者判定（人間入力）        ← Phase 1 から運用
Layer 3: AI判定（Claude API）         ← Phase 5 で実装
         ↓
最終コメント出力
```

初版では Layer 1 + Layer 2 で完全に稼働できるよう設計する。

## ルールベース判定フロー（Layer 1）

### Step 1: 赤旗チェック（最優先）

```
if (赤旗22行「排尿・排便障害」== "あり" || 赤旗23行「会陰部感覚異常」== "あり") {
    判定 = "【緊急】馬尾症候群疑い — 即日医療機関紹介を検討"
    アラート背景 = 赤
    以降の評価は参考値として記録するが、施術は原則中止
}
else if (赤旗スコア >= 1) {
    判定 = "赤旗あり — 医療連携を検討してから施術方針を決定"
    アラート背景 = オレンジ
}
else {
    赤旗クリア
}
```

### Step 2: 神経症状レベル判定

```
if (SLRテスト == "両側陽性" || 下肢筋力低下 == "あり") {
    神経レベル = "重度"
    → 神経所見を最優先。医療連携の検討を促す
}
else if (SLRテスト == "陽性（右）" || "陽性（左）") {
    神経レベル = "中等度"
    → 神経根障害の可能性。施術方針に反映
}
else if (下肢放散痛 == "片側" || 両側) {
    神経レベル = "軽度"
    → 神経刺激症状あり。悪化監視が必要
}
else {
    神経レベル = "なし"
}
```

### Step 3: 慢性化リスク判定（STarT）

```
if (STarTスコア >= 6) {
    リスク = "高"
    → 心理社会的因子への介入必須
    → 説明・行動変容・患者教育を施術と同等に重視
}
else if (STarTスコア >= 4) {
    リスク = "中"
    → 心理社会的因子に注意。セルフエフィカシー向上を意識した説明を
}
else {
    リスク = "低"
    → 身体的アプローチ中心で対応可
}
```

### Step 4: 痛み強度判定（NRS）

```
if (NRS現在 >= 7) { 強度 = "高強度" }
else if (NRS現在 >= 4) { 強度 = "中等度" }
else { 強度 = "軽度" }
```

### Step 5: 機能障害レベル判定（RMDQ-10）

```
if (RMDQ >= 8) { 機能 = "重度障害" }
else if (RMDQ >= 4) { 機能 = "中等度障害" }
else { 機能 = "軽度障害" }
```

### Step 6: 動作障害パターン判定

```
if (前屈=著明制限 && 後屈=著明制限) { 動作パターン = "全方向制限型" }
else if (前屈=著明制限 || 中等度制限) { 動作パターン = "屈曲優位制限型" }
else if (後屈=著明制限 || 中等度制限) { 動作パターン = "伸展優位制限型" }
else { 動作パターン = "軽度制限型" }
```

### Step 7: 転倒リスク判定（移乗動作）

```
if (立ち上がり != "自立" || 歩行 == "中等度/著明障害") {
    転倒リスク = "高"
    → 転倒予防・ADL視点を方針に追加
}
else if (段差昇降 == "困難/不可" || 床からの立ち上がり == "困難/不可") {
    転倒リスク = "中"
}
else {
    転倒リスク = "低"
}
```

## 総合判定マトリクス（治療優先方針）

| 条件組合せ | 自動推奨方針 |
|---|---|
| 赤旗 or 馬尾 | **医療連携優先** |
| 神経レベル=重度 | **神経所見優先（医療連携検討）** |
| 神経レベル=中等度 | **神経根対応優先** |
| STarT高 + NRS中〜高 | **行動変容・説明優先** |
| STarT高 + NRS低〜中 | **行動変容・段階的機能改善** |
| 機能重度障害 + 転倒リスク高 | **ADL・転倒予防優先** |
| NRS高 + 機能中等度以上 | **疼痛管理優先 → 機能改善** |
| 発症2週未満 + NRS中 | **急性期管理（安静・保護）** |
| 発症6週以上 + STarT低 | **機能改善・運動療法開始** |
| 発症3か月以上 + STarT低〜中 | **機能改善+セルフケア習慣化** |
| 発症3か月以上 + STarT高 | **生活指導+行動変容+段階的運動** |

## フラグシステム

各ステップで以下のフラグを立て、コメント生成の条件キーとして使用する。

| フラグ名 | 条件 | コメントマスタ参照キー |
|---|---|---|
| FLAG_CAUDA | 馬尾症候群疑い | CAUDA_EMERGENCY |
| FLAG_REDFLAG | 赤旗1以上 | REDFLAG_ALERT |
| FLAG_NERVE_SEVERE | 神経レベル=重度 | NERVE_SEVERE |
| FLAG_NERVE_MOD | 神経レベル=中等度 | NERVE_MOD |
| FLAG_START_HIGH | STarT>=6 | START_HIGH |
| FLAG_START_MID | STarT>=4 | START_MID |
| FLAG_NRS_HIGH | NRS>=7 | NRS_HIGH |
| FLAG_RMDQ_SEVERE | RMDQ>=8 | RMDQ_SEVERE |
| FLAG_MOTION_ALL | 全方向制限型 | MOTION_ALL |
| FLAG_FALL_HIGH | 転倒リスク=高 | FALL_HIGH |
| FLAG_CHRONIC | 発症3か月以上 | CHRONIC |
| FLAG_RECURRENT | 既往=複数回以上 | RECURRENT |

## 判定優先順位（競合時）

複数フラグが同時に立つ場合、以下の優先順位でコメントを選択する。

```
1. FLAG_CAUDA           （最優先・緊急）
2. FLAG_REDFLAG         （医療連携必須）
3. FLAG_NERVE_SEVERE    （神経所見優先）
4. FLAG_NERVE_MOD       （神経所見注意）
5. FLAG_START_HIGH      （心理社会的介入優先）
6. FLAG_FALL_HIGH       （転倒予防優先）
7. FLAG_NRS_HIGH        （疼痛管理優先）
8. FLAG_RMDQ_SEVERE     （機能障害対応優先）
9. FLAG_MOTION_ALL      （動作改善優先）
10. FLAG_CHRONIC        （慢性期管理）
11. その他              （標準方針）
```

## 再評価時の変化判定

| 評価 | 前回比較 | 判定 |
|---|---|---|
| NRS | -2以上 | 有意な改善 |
| NRS | +2以上 | 悪化傾向（方針見直し） |
| RMDQ | -3以上 | 有意な改善 |
| PSFS | +2以上 | 有意な改善（MCID達成） |
| STarT | 低下 | 慢性化リスク低減 |
| STarT | 上昇 | 心理社会的因子悪化（注意） |

## Phase 5 向け AI判定連携設計

Layer 3（Claude API）に渡すデータ構造案：

```json
{
  "evaluation_id": "E0001",
  "patient_age": 52,
  "sex": "女性",
  "onset_duration": "3か月以上",
  "nrs": { "current": 6, "worst": 8, "best": 3 },
  "rmdq_score": 6,
  "start_score": 5,
  "start_risk": "中リスク",
  "psfs_avg": 4.5,
  "red_flags": 0,
  "nerve_level": "なし",
  "motion_pattern": "屈曲優位制限型",
  "fall_risk": "低",
  "flags": ["FLAG_START_MID", "FLAG_NRS_HIGH", "FLAG_CHRONIC"],
  "rule_based_judgment": "行動変容・疼痛管理優先",
  "clinician_judgment": "機能改善優先",
  "free_text": "デスクワーク8時間。寝返りで目が覚める。"
}
```

このJSONを `AI判定用出力` シートに書き込み、GASからClaude APIを呼び出す。
