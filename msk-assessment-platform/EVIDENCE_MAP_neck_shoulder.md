# EVIDENCE_MAP_neck_shoulder.md

**対象:** 頚肩こり_初期評価モジュール（JASSESS-01 Phase 2）
**最終更新:** 2026-03-26

> 本ファイルは、頚肩こり評価モジュールの各評価ブロックに対する**根拠IDと参照情報**を管理する。
> シート本体に URL を直書きせず、根拠ID（EV-NS-XXX）で参照する方式をとる。
> 各評価項目の根拠IDは `modules/neck-shoulder/README.md` を参照。

---

## 根拠ID の命名規則

```
EV-NS-XXX
  EV  = Evidence
  NS  = Neck-Shoulder（頚肩こりモジュール固有）
  XXX = 3桁通番
```

共通基盤（赤旗・NRS・PSFS）に関する根拠は、将来 `EVIDENCE_MAP_common.md` で管理予定。

## 根拠マップ一覧

### EV-NS-001
| 項目 | 内容 |
|---|---|
| **根拠ID** | EV-NS-001 |
| **タイトル** | 非特異的頚部痛の総論 |
| **用途** | 主訴・経過の分類、総合方針分岐、患者説明の根拠 |
| **要点** | 頚部痛の約70〜80%は非特異的（構造的病変が特定されない）。急性は予後良好が多いが、慢性化すると心理社会的因子の影響が増す。大半は自然経過で改善するが、再発率は高い。 |
| **source_title** | Côté P, et al. "The burden and determinants of neck pain in the general population." *Spine* 2008 |
| **URL / DOI** | https://doi.org/10.1097/BRS.0b013e3181643f2a |
| **note** | GBD 2019 でも頚部痛は筋骨格系の主要な障害負荷として記録されている。臨床方針の基本背景として参照。 |

### EV-NS-002
| 項目 | 内容 |
|---|---|
| **根拠ID** | EV-NS-002 |
| **タイトル** | 頚部痛の分類枠組み（機械的 / 神経根性 / WAD / 非特異的） |
| **用途** | セクション NS-B（神経症状スクリーニング）、ROM評価、方針分岐の根拠 |

| **source_title** | Bogduk N, McGuirk B. *Medical Management of Acute and Chronic Low Back Pain.* Elsevier, 2002（頚部痛版の分類枠組みとして引用） |
| **URL / DOI** | — |
| **note** | IASP / IFOMPT の分類も参照。ROM と神経症状スクリーニングの組み合わせで分類を支援する。 |

### EV-NS-004
| 項目 | 内容 |
|---|---|
| **根拠ID** | EV-NS-004 |
| **タイトル** | 頚部痛における姿勢評価の位置づけ |
| **用途** | セクション NS-E（姿勢観察）の根拠と臨床的意義 |
| **要点** | 前傾頭位（Forward Head Posture）は頚部筋への負荷増加と関連するが、姿勢と痛みの直接的な因果関係は限定的という研究もある。姿勢評価は「観察」として行い、病態説明より「どう動いているか・どう負荷がかかっているか」の確認に用いるのが適切。 |
| **source_title** | Fernández-de-las-Peñas C, et al. "The immediate effect of ischemic compression technique and transverse friction massage on tenderness of active and latent myofascial trigger points." *Journal of Bodywork and Movement Therapies*, 2006 |
| **URL / DOI** | — |
| **note** | 姿勢評価単独での診断的価値は高くないが、運動療法導入（チンタック・肩甲骨リトラクション）の観察根拠として有用。「姿勢が悪いから痛い」という過剰な説明には注意。 |

### EV-NS-005
| 項目 | 内容 |
|---|---|
| **根拠ID** | EV-NS-005 |
| **タイトル** | 頚部痛・肩こりに対する運動療法のエビデンス |
| **用途** | 総合方針分岐（運動療法主軸 / 早期接続）の根拠、患者説明 |
| **要点** | 慢性頚部痛に対する運動療法は、短期的な疼痛軽減と機能改善に有効。特に頚部安定化訓練（チンタックを含む深部屈筋強化）と肩甲帯の運動を組み合わせたものが比較的エビデンスが蓄積している。マニュアルセラピーとの併用効果も示されている。 |
| **source_title** | Kay TM, et al. "Exercises for mechanical neck disorders." *Cochrane Database of Systematic Reviews*, 2012 |
| **URL / DOI** | https://doi.org/10.1002/14651858.CD004250.pub4 |
| **note** | 短期のエビデンスは比較的確立。長期的な再発予防については継続的な自主トレの重要性が強調されている。種目選定の参考として使う。 |

### EV-NS-006
| 項目 | 内容 |
|---|---|
| **根拠ID** | EV-NS-006 |
| **タイトル** | 職場・生活背景が頚部痛に与える影響（職業的要因） |
| **用途** | セクション NS-D（日常生活負荷）の根拠、生活指導の根拠 |
| **要点** | 長時間のPC作業・同一姿勢・精神的ストレス・睡眠障害は頚部痛の持続・再発と関連している。スマートフォンの長時間使用（テキストネック）は頚椎への負荷増大と関連。これらの修正が介入の一環として重要。 |
| **source_title** | Côté P, et al. "The burden and determinants of neck pain in workers." *Spine* 2008 |
| **URL / DOI** | https://doi.org/10.1097/BRS.0b013e3181644ee2 |
| **note** | 日常生活負荷の評価は、運動療法だけでなく「負荷を減らす生活指導」の根拠として使う。特にPC・スマホ長時間使用者への指導で参照する。 |

### EV-NS-007
| 項目 | 内容 |
|---|---|
| **根拠ID** | EV-NS-007 |
| **タイトル** | Neck Disability Index（NDI）— 機能障害評価 |
| **用途** | 将来的な機能障害評価ツール導入時の根拠（現時点は未実装） |
| **要点** | NDI は頚部痛による機能障害を10項目で評価する自記式質問票。0〜100点。MCD（最小重要変化量）は約7点。腰痛モジュールの RMDQ に相当する頚部版として広く使われている。 |
| **source_title** | Vernon H, Mior S. "The Neck Disability Index: a study of reliability and validity." *Journal of Manipulative and Physiological Therapeutics*, 1991 |
| **URL / DOI** | https://doi.org/10.1016/S0161-4754(05)80076-4 |
| **note** | 今回の軽量版には含めないが、慢性化ケースや運動療法導入後の再評価ツールとして今後追加を検討する。 |

### EV-NS-008
| 項目 | 内容 |
|---|---|
| **根拠ID** | EV-NS-008 |
| **タイトル** | STarT の頚部痛への適用可能性 |
| **用途** | 慢性化リスク評価の根拠（頚部版の適用方針を記録） |
| **要点** | STarT Back Screening Tool は腰痛向けに開発されたが、心理社会的リスク因子の概念は頚部痛にも共通する部分がある。頚部痛特化版（STarT Neck）は研究段階であり、現時点では汎用版の概念参照にとどめ、固有スコアとしては使わない。 |
| **source_title** | Hill JC, et al. "Comparison of stratified primary care management for low back pain with current best practice (STarTBack): a randomised controlled trial." *Lancet*, 2011 |
| **URL / DOI** | https://doi.org/10.1016/S0140-6736(11)60937-9 |
| **note** | 今回の頚肩こりモジュールでは、STarT スコアを直接使用せず、「慢性化リスク / 心理社会的背景の確認」として類似の観点を含む。将来、頚部版ツールが実用化されれば差し替える。 |
