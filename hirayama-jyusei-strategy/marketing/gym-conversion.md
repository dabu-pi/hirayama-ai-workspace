# ジム転換戦略

## 目的

施術患者をジム月会員へ転換し、定期収益を確立する。

## 転換ステップ

| ステップ | アクション |
|---|---|
| 1. 運動療法導入 | 施術後に「再発予防には運動が必要」と説明 |
| 2. 体験誘導 | 無料or低価格のジム体験を案内 |
| 3. 入会提案 | 体験後に月会員プランを提案 |
| 4. 継続管理 | 月1回の状態確認・施術継続案内 |

## トークポイント

- 「施術で痛みは取れましたが、筋力がないとまた痛くなります」
- 「運動療法を続ける場所として当院のジムが使えます」
- 「月〇〇円で通い放題なので、継続しやすいですよ」
