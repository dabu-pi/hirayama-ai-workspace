# 実務連携メモ（IMPLEMENTATION_LINK）

> **目的:** 【毎日記録】来店管理施術録ver3.1 と JBIZ-04 の役割分担を固定し、
> 新しい日次入力ブックを増やさずに月次KPIへつなぐ構成を整理する。

## 背景

平山接骨院は現在、`gas-projects/jyu-gas-ver3.1/` で保険請求管理を運用している。  
戦略目標を計測するには、患者ごとの **慢性患者化 → 手技自費 → 再来 → ジム体験** の通過状況を記録する必要がある。

このファイルでは、「どちらを正本にするか」「毎日記録ブックへ何を最小追加するか」「JBIZ-04へ何を月次で渡すか」を整理する。

## 推奨構成

### 1. 役割分担

| 対象 | 正本 | 主な役割 |
|---|---|---|
| 【毎日記録】来店管理施術録ver3.1 | 日次入力の正本 | 来院入力 / case管理 / 来院ヘッダ / 施術明細 / 窓口負担額 / 保険請求額 / 自費来院の実績記録 |
| JBIZ-04 | 月次KPI・逆算の正本 | 月次KPI集計 / KPI逆算 / 価格仮説 / 経営判断 / 月次レビュー |

### 2. 基本ルール

- 新しい別ブックは作らない
- JBIZ-04 側に日次入力シートは新設しない
- 毎日記録ブックを現場入力の正本にする
- JBIZ-04 は月次で集約された数値だけを持つ
- 保険患者だけでなく、`保険のみ / 保険+自費 / 自費のみ` を同じ来院管理の流れで扱える形へ寄せる
- `保険新規 / 自費直新規` の分離管理は次段階で必ず実施し、今回は入れ物だけ準備する

## 追加方法の判断

### 結論

**原則は列追加案を推奨する。**

### 理由

- 既存の来院入力 → 来院ヘッダ → 施術明細 の流れを崩さずに済む
- `保険のみ / 保険+自費 / 自費のみ` を1来院単位で扱いやすい
- 月次集計時に `会計区分` と `自費売上額` を見るだけで必要集計に近づく
- 現場オペレーションが「どのブックに入れるか」で迷わない

### 補助ページ1枚を許容する条件

- 自費メニュー区分の候補一覧が増えて入力UIが見づらい
- 将来の `保険新規 / 自費直新規` の分離運用で説明欄や補助集計が必要

この場合でも、**同じ毎日記録ブック内に補助ページ1枚まで** に留める。  
新しい別ブックは作らない。

### 定義確定メモ

- `手技自費患者数` は **月内ユニーク患者数** として扱う
- `手技回数` は **総来院件数** として扱う
- `自費平均単価` は **自費売上 ÷ 手技回数** とし、患者数では割らない
- `慢性候補人数` と `手技自費患者数` は件数ではなく人数、`手技回数` と `次回予約あり件数` は来院件数ベースで扱う

### KPI整合メモ

- `月次総保険売上` / `月次自費売上` / `自費平均単価` は、既存の `KPI目標` 実績セルにそのまま接続でき、既存KPIと矛盾しない
- `慢性候補人数` は `保険・来院前提` の率見直しと `KPI逆算` の必要慢性候補人数確認に使う補助値で、現行は `KPI目標` の正式入力項目ではない
- `手技自費患者数` は月次比較用の補助値としては整合するが、`strategy/kpi.md` の「初回件数 / 再来件数」管理とは粒度が違うため、**厳密な初回移行率KPIとは混在させない**
- `次回予約あり件数` は再来の先行指標としては有効だが、現行JBIZ-04ではまだ正式KPIセル未設定のため、月次レビュー用メモとして扱う

## 月末運用手順（最小構成）

1. 毎日記録ver3.1 の当月分を見て、`月次総保険売上 / 月次自費売上 / 慢性候補人数 / 手技自費患者数 / 手技回数 / 次回予約あり件数` を集計する
2. 集計結果から `自費平均単価 = 月次自費売上 ÷ 手技回数` を算出する
3. JBIZ-04 の `KPI目標` に `D5=月次総保険売上`、`D6=月次自費売上`、`D11=自費平均単価` を入力する
4. `保険・来院前提` の `B10` を見直す必要があるかを、`慢性候補人数` の月次実績で確認する。`数値前提` と `価格設定` は、前提変更がある月だけ更新する
5. 入力後は `KPI目標` の `月間売上 / 自費売上比率 / 保険平均単価 / 自費平均単価` を確認し、補助値として `慢性候補人数 / 手技自費患者数 / 手技回数 / 次回予約あり件数` を月次レビューに使う

## 追加すべき管理項目の考え方

### 1. 保険来院

| 記録項目 | 内容 | 型 |
|---|---|---|
| 初回保険来院日 | 患者との接触開始日 | 日付 |
| 累計来院回数（保険） | 手技提案タイミング判断に使う | 数値 |
| 主訴 | 慢性腰痛・首肩こり等 | テキスト |
| 慢性化フラグ | 初診時に慢性対象かどうか | 真偽値 |

**KPI計測への使い方:**
- 慢性化フラグ ON の人数 = 手技自費移行候補数
- 月次でカウントし、移行率の分母として使う

---

### 2. 手技自費提案

| 記録項目 | 内容 | 型 |
|---|---|---|
| 手技提案日 | 初回の手技メニュー案内日 | 日付 |
| 提案したメニュー | 手技50分 / 運動療法 / ジム体験など | テキスト |
| 提案時の反応 | 興味あり / 保留 / 断り | 選択肢 |

**KPI計測への使い方:**
- 手技提案患者数 / 慢性化患者数 = 慢性患者移行率の前段指標
- 提案後の反応分布を見て、トークスクリプト改善に使う

---

### 3. 手技自費開始

| 記録項目 | 内容 | 型 |
|---|---|---|
| 手技自費開始日 | 実際に手技自費を受けた初回日 | 日付 |
| 成約メニュー | 何を受けたか | テキスト |
| 成約金額 | 実際の売上額 | 数値 |

**KPI計測への使い方:**
- 手技自費初回件数 = 月次の初回件数
- 成約金額の月次合計 = 自費収益KPI（目標: +20万円/月）

---

### 4. 手技自費再来

| 記録項目 | 内容 | 型 |
|---|---|---|
| 再来日 | 2回目以降の手技自費来院日 | 日付 |
| 再来回数 | 手技自費として何回目か | 数値 |

**KPI計測への使い方:**
- 再来患者数 / 初回手技自費患者数 = 手技自費再来率
- 月次の手技自費総回数を集計できる

---

### 5. 運動療法・セルフケア

| 記録項目 | 内容 | 型 |
|---|---|---|
| セルフケア案内日 | 自宅ケアを案内した日 | 日付 |
| 運動療法開始日 | 初回の運動療法セッション日 | 日付 |
| セッション形式 | パーソナル / グループ / セルフ指導 | 選択肢 |

**KPI計測への使い方:**
- 手技自費患者のうち、どれだけ継続支援へ進んだかを確認する
- ジム体験案内の前段階として実績を追う

---

### 6. ジム体験・入会

| 記録項目 | 内容 | 型 |
|---|---|---|
| ジム体験案内日 | ジム利用を案内した日 | 日付 |
| 体験実施日 | 実際に体験した日 | 日付 |
| 体験後の反応 | 入会希望 / 検討中 / 不要 | 選択肢 |
| ジム入会日 | 実際に会員契約した日 | 日付 |
| 入会プラン | スタンダード / プレミアム | 選択肢 |
| 月額費用 | 実際の会員費 | 数値 |
| 退会日 | 退会した場合に記録 | 日付（空白可） |

**KPI計測への使い方:**
- ジム体験案内数 / 手技自費患者数 = ジム体験誘導率
- 入会数 / 体験数 = ジム入会率
- 在籍会員数 × 月額費用 = ジム月次定期収益

## KPIダッシュボードとの接続イメージ

```text

    ↓

ジム体験誘導率
    ↓（入会数 ÷ 体験数）
ジム入会率
```
