## このファイルの使い方

- **1件実施するごとに**: 下の「記録テンプレート（1件分）」をコピーして記入し、このファイルの末尾に追加する
- **3件たまったら**: `strategy/gym-trial-summary.md` のまとめ表を埋めて採用判定を行う
- **採用確定後**: `strategy/gym-lead-design.md` の採用状態を「採用確定」に更新する

> 目的は「記録すること」より「見返して判断できること」。簡潔でよい。

---

## 記録テンプレート（1件分）

> コピー用。実施したら「試行 #N」として末尾に追加する。

```
### 試行 #N（YYYY-MM-DD）

**基本情報**

| 項目 | 記入 |
|---|---|
| 実施日 | |
| 対象者 | 例: 患者A（50代 男性）|
| 年代 / 性別 | |
| 症状区分 | 慢性腰痛 / 首肩こり |
| 主訴 | 例: 長年の腰痛・前屈時の痛み |
| 実施場面 | 施術後当日 / 別日予約 |
| 実施パターン | 20分版 / 30分版 |
| 担当 | 施術者単独 / ジムスタッフ補助あり |

**種目別評価**

| 種目名 | 合格可否 | 代償動作 | 痛み増悪 | 改善反応 |
|---|---|---|---|---|
| | 合格 / 条件付き / 保留 | | なし / あり（部位: ）| あり / なし |
| | 合格 / 条件付き / 保留 | | なし / あり（部位: ）| あり / なし |

※ 合格可否の定義:
- 合格   = 合格基準をすべて満たした
- 条件付き = 一部満たしたが代償動作が残る / 声かけで改善できた
- 保留   = 合格基準を満たせなかった（代替を検討）

**セッションまとめ**

| 項目 | 記入 |
|---|---|
| 中止基準 該当 | なし / あり（内容: ） |
| 患者の反応 | 良好 / 普通 / 難しい（理由: ） |
| 改善反応の詳細 | |
| 次回修正点 | |
| 種目の続行 / 変更 | 続行 / 代替へ変更（→種目名: ）/ 強度調整 |
| ジム誘導の反応 | 興味あり / なし / 次回聞く |
| 総合所見 | |
| 暫定判定 | 良好 / 継続観察 / 要修正 |
```

**種目別評価**

| 種目名 | 合格可否 | 代償動作 | 痛み増悪 | 改善反応 |
|---|---|---|---|---|
| ヒップヒンジ（自重） | 合格 | なし | なし | あり |

**セッションまとめ**

| 項目 | 記入 |
|---|---|
| 中止基準 該当 | なし |
| 患者の反応 | 難しい |
| 改善反応の詳細 | （未記入） |
| 次回修正点 | 代償動作を少なくする |
| 種目の続行 / 変更 | 続行 |
| ジム誘導の反応 | 興味あり |
| 総合所見 | デッドバグで痛み増悪が確認された。次試行でデッドバグの実施方法を修正（動作範囲を狭める・腰部安定重視）するか、代替（ドローイン）への切替を検討する。ヒップヒンジは合格・改善反応あり。ジム誘導への関心は良好。 |
| 暫定判定 | 継続観察 |
