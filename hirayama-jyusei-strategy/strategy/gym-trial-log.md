## このファイルの使い方

- **1件実施するごとに**: 下の「記録テンプレート（1件分）」をコピーして記入し、このファイルの末尾に追加する
- **3件たまったら**: `strategy/gym-trial-summary.md` のまとめ表を埋めて採用判定を行う
- **採用確定後**: `strategy/gym-lead-design.md` の採用状態を「採用確定」に更新する

> 目的は「記録すること」より「見返して判断できること」。簡潔でよい。

```

| 担当 | 施術者単独 / ジムスタッフ補助あり |

**種目別評価**

| 種目名 | 合格可否 | 代償動作 | 痛み増悪 | 改善反応 |
|---|---|---|---|---|
| | 合格 / 条件付き / 保留 | | なし / あり（部位: ）| あり / なし |
| | 合格 / 条件付き / 保留 | | なし / あり（部位: ）| あり / なし |

※ 合格可否の定義:
- 合格   = 合格基準をすべて満たした
- 条件付き = 一部満たしたが代償動作が残る / 声かけで改善できた
- 保留   = 合格基準を満たせなかった（代替を検討）

**セッションまとめ**

| 項目 | 記入 |
|---|---|
| 中止基準 該当 | なし / あり（内容: ） |
| 患者の反応 | 良好 / 普通 / 難しい（理由: ） |
| 改善反応の詳細 | |
| 次回修正点 | |
| 種目の続行 / 変更 | 続行 / 代替へ変更（→種目名: ）/ 強度調整 |
| ジム誘導の反応 | 興味あり / なし / 次回聞く |
| 総合所見 | |
| 暫定判定 | 良好 / 継続観察 / 要修正 |
```
