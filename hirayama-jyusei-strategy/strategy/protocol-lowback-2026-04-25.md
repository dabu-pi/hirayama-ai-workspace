# 慢性腰痛 症状別プロトコル

案件: JBIZ-04（慢性疼痛強化プロジェクト2026）
作成: 2026-04-25
価格: すべて税別
位置づけ: 初回標準施術 → 継続標準施術 → メンテナンス施術 → ジム月会員 への導線設計

---

## 1. 対象患者像

### 適応（このプロトコルの対象）

- 慢性的に腰が重い・だるい
- 立ち上がり時に腰が痛い
- 長時間座ると腰が痛くなる
- 朝起きた時に腰が固い（起床後30分以内に動きやすくなる）
- 反る・曲げる・捻る動作に不安がある
- 運動不足・体幹筋力低下・股関節可動性低下が疑われる
- 3か月以上続く腰痛（反復性含む）

### 除外・医療機関受診を優先する所見

以下がある場合はこのプロトコルを適用せず、受診を勧める。

- 下肢のしびれ・感覚異常・筋力低下（神経症状の疑い）
- 安静時痛・夜間痛（炎症・腫瘍等の疑い）
- 発熱を伴う腰痛（感染の疑い）
- 外傷後（骨折等の可能性）
- 排尿・排便障害（馬尾症候群の疑い）
- 近いうちに受診するよう強く勧めた方がいいと施術者が判断した場合

> **原則:** 疑わしい所見がある場合は「念のため専門医に診てもらってください」と伝え、
> 施術の可否は患者と相談の上で決める。医学的診断はしない。

## 2. 初回標準施術プロトコル

**メニュー:** SELFPAY_INITIAL_FULL / 5,000円税別 / 約48分

### 物療構成と役割

| 物療 | 単価 | 時間 | 腰痛での役割 |
|---|---:|---:|---|
| 初診料（問診・評価）| 1,500円 | 約15分 | 症状把握・動作確認・施術計画の説明 |
| 干渉波 | 1,000円 | 約10分 | 深部の血流改善・筋緊張緩和・慢性痛の閾値引き上げ |
| マイクロカレント | 1,000円 | 約10分 | 細胞レベルの修復促進・疼痛抑制 |
| ハイボルテージ | 500円 | 約5分 | 急性的な痛み部位への集中的な鎮痛・筋スパズム緩和 |
| 超音波 | 500円 | 約5分 | 深部組織（筋・腱付着部）への温熱・組織修復促進 |
| 手技 | 500円 | 約3分 | 硬結部位への直接アプローチ・可動域確認 |

### 初回問診チェック項目

| カテゴリ | 確認項目 |
|---|---|
| 発症 | いつから / きっかけはあるか / 以前にも同様の症状があったか |
| 症状 | 痛みの部位 / 片側・両側 / 深さ（表面か奥か）|
| 増悪 | 反る・曲げる・捻る / 長時間座位 / 立位 / 歩行 |
| 緩解 | 動いたら楽になるか / 横になると楽か |
| 除外確認 | しびれ / 夜間痛・安静時痛 / 発熱 / 外傷歴 |
| 生活 | 仕事（デスクワーク・立ち仕事）/ 運動習慣 / 体重 |
| 目標 | どうなりたいか（痛みをなくしたい / 仕事に支障なく動きたい 等）|

### 痛みスコア記録（VAS）

施術前後にVAS（0〜10）で記録する。

- 施術前: ___
- 施術直後: ___

### 物療使い分けの考え方（腰痛）

```
急性・局所的な強い痛み     → ハイボルテージを先に使う
深部の筋緊張・重だるさ    → 干渉波 + マイクロカレント
組織修復・腱付着部         → 超音波を患部に当てる
手技                        → 触診で硬結を確認してから補助的に使う
```

### 初回後の説明トーク（院長・スタッフ用）

```
「今日確認したところ、腰の深い筋肉が緊張していて、
 骨盤まわりの動きも少し硬くなっています。

 まず物療で筋肉を柔らかくして、
 手技で固まった部分をほぐすのが最初のステップです。

 今日が1回目なので、明日・明後日あたりに
 少し反応が出る場合がありますが、
 それは治療の経過として普通のことです。

 効果を確かめながら進めたいので、
 次は○日後くらいに来ていただけると理想的です。
 いかがでしょうか？」
```

### 継続標準施術への案内文

```
「症状が落ち着いてきたら、次のステップとして
 少し短い時間で同じ物療を続けることができます。

 継続施術は33分ほどで、3,500円でお受けできます。
 最初は週に2〜3回を目安に、
 症状が安定してきたら頻度を下げていけます。

 まずは3〜4回ほど通っていただいて、
 変化を一緒に確認させてください。」
```

## 3. 継続標準施術プロトコル

**メニュー:** SELFPAY_CONTINUE20 / 3,500円税別 / 約33分

### 物療構成（腰痛向け）

| 物療 | 腰痛での使い方 |
|---|---|
| 干渉波（10分）| 深部筋（多裂筋・腸腰筋周辺）への継続的な緊張緩和 |
| マイクロカレント（10分）| 痛みが残る部位への修復継続・神経系の感作抑制 |
| ハイボルテージ（5分）| 前回より痛みが残っている部位に集中使用 |
| 超音波（5分）| 腰仙部・梨状筋付近など深部の硬結がある場合に使用 |
| 手技（3分）| 施術前の触診で硬結確認→緊張が強い部位を補助 |

### 2回目以降の確認項目

| タイミング | 確認内容 |
|---|---|
| 来院時 | 前回との比較（良くなった点・変わらない点・気になった点）|
| VAS | 施術前・施術後を毎回記録 |
| 動作確認 | 前屈・後屈・側屈・回旋の可動域変化 |
| 日常生活 | 座位・立位・睡眠の変化 |

### 変化を見る目安回数

| 段階 | 目安 | 判断 |
|---|---|---|
| 1〜3回 | 初期変化確認 | VAS が下がる / 動作が楽になる / 翌日の痛みが和らぐ |
| 4〜6回 | 安定化確認 | 痛みが出にくくなる / 動作の不安が減る |
| 7回以降 | 維持フェーズ検討 | メンテナンス施術への移行を提案 |

> **3回で変化がない場合:** 施術内容・頻度・物療の組み合わせを見直す。
> 6回で改善傾向がない場合は医療機関への受診を検討する。

### メンテナンス施術へ移行する条件

以下が揃ったら案内する。

- VAS が施術前で 3 以下で安定している
- 日常生活で腰痛が気にならない日が増えている
- 動作時の不安が減っている

### ジム月会員への案内条件

- 継続施術を3〜4回以上受けて改善傾向がある
- 「再発しないようにしたい」という言葉が出た
- 運動に興味がある・体力が落ちていると感じている

## 4. メンテナンス施術プロトコル

**メニュー:** SELFPAY_MAINT15 / 2,500円税別 / 約23分

### 物療構成（腰痛向け）

| 物療 | 腰痛での使い方 |
|---|---|
| 干渉波（10分）| 維持目的。週1〜2回の物療で再緊張を予防 |
| マイクロカレント（10分）| 低強度での継続刺激。普段使わない筋を目覚めさせる |
| 手技（3分）| 触診で硬結が出ていないか確認。あれば重点的に |

### 使い方の目安

- 頻度: 月1〜2回が標準
- ジム利用者: 施術とジムを組み合わせて再発を予防する
- 状態が良ければ: 月1回でも維持できる場合がある

### 患者への案内文

```
「だいぶ楽になってきましたね。
 このまま終わりにしてもいいのですが、
 慢性的な腰痛は一度楽になっても
 生活習慣や姿勢が変わらないと戻りやすいです。

 月に1〜2回、短い施術で今の状態をキープするのが
 メンテナンスの役割です。

 ジムと組み合わせていただくと、
 自分で体を動かして予防することもできます。
 25分ほどで2,500円なので、
 医療費よりずっとコスパが良い方法だと思います。」
```

## 5. ジム誘導プロトコル

**メニュー:** GYM_MONTHLY_REF / ジム月会員 / 7,480円（男性区分代表値）

### 案内するタイミング

| タイミング | 状況 |
|---|---|
| 継続施術3〜4回目 | 「再発しないようにしたい」という言葉が出た |
| メンテナンス移行時 | 「ずっと通い続けるのが大変」という話が出た |
| 症状が落ち着いた時 | 「もっと体を動かしたい」「筋肉をつけたい」という話が出た |

### 案内する患者像

- 痛みが改善してきている
- 「再発予防」に興味がある
- 「体力・筋力が落ちている」という自覚がある
- 仕事でデスクワーク・長時間座位が多い

### 案内しない方がいい場合

- 現在も強い痛みがある（まず施術で安定させる）
- ジム自体に興味がない（押しつけにならないよう注意）
- 高齢で運動に不安がある（別途相談する）

### 腰痛再発予防としての説明トーク

```
「腰痛が繰り返される一番の原因は、
 体幹の筋力と股関節の柔軟性が落ちることです。

 施術で痛みをとることはできますが、
 根本的に繰り返さないためには
 自分で体を動かせる状態にしていくことが大切です。

 隣にジムがあるので、
 最初は週1〜2回でもいいので
 股関節まわりと体幹のトレーニングをやってもらえると、
 腰痛が戻りにくい体になっていきます。

 まずは無料体験から試してみませんか？」
```

### ジムで最初にやるべき運動カテゴリ

| カテゴリ | 目的 | 代表種目 |
|---|---|---|
| 股関節可動性 | 腰への代償動作を減らす | ヒップヒンジ・股関節外旋ストレッチ |
| 体幹安定 | 腰椎を守る内圧コントロール | ドローイン・デッドバグ低負荷版 |
| ヒップヒンジ | 腰を使わず股関節で動ける | ルーマニアンデッドリフト・バックエクステンション |
| スクワット基礎 | 立ち上がり動作の改善 | ゴブレットスクワット・椅子スクワット |
| 軽い有酸素 | 血流改善・体重管理 | ウォーキングマシン・クロストレーナー |

> **初回体験での案内:** 6種目の「再発予防トレーニング体験」（採用済み種目）から始める。
> 詳細: `strategy/gym-exercise-guide.md`

## 7. KPI設計

### 追うべきKPI（慢性腰痛プロトコル）

| KPI | 目標値（初期）| 計測方法 |
|---|---|---|
| 慢性腰痛初回件数 | 10件/月 | 来院ヘッダ: 初回+慢性腰痛 |
| 継続施術移行率 | 70%以上 | 初回後に継続した件数 ÷ 初回件数 |
| 3回継続率 | 50%以上 | 3回以上来院した件数 ÷ 初回件数 |
| メンテナンス移行数 | 3件/月以上 | 継続→メンテに移行した件数 |
| ジム月会員移行数 | 1名/月以上 | 腰痛患者からジム入会した件数 |
| 初回からジム案内までの日数 | 30日以内 | 初回来院日〜ジム案内日の差 |
| 初回VAS → 3回後VAS 変化 | ▲2点以上 | 施術ノートより |

> KPI目標値は初期運用の仮値。実績2〜3ヶ月後に見直す。

## 8. 患者説明トーク集

### 物療を複数使う理由の説明

```
「腰の痛みには、筋肉の緊張・血流の悪さ・
 神経の過敏・組織の炎症など
 複数の要素が絡んでいることが多いです。

 電気治療は種類によって効き方が違うので、
 深部の緊張をほぐすもの・修復を促すもの・
 痛みを直接抑えるものを組み合わせて使っています。

 それぞれ短時間ですが、組み合わせることで
 効果が出やすくなります。」
```

### 痛みが戻る理由の説明

```
「施術で痛みは取れますが、
 姿勢・筋力・動き方が変わらないと
 同じ負担が腰にかかり続けるので
 痛みは戻りやすいです。

 施術で症状を改善しながら、
 生活の中での負担を少しずつ減らしていくのが
 長く良い状態をキープする方法です。」
```

### 継続施術への案内（2回目来院時）

```
「前回と比べていかがでしたか？
 少し楽になっているようなら、
 この方向で続けていきましょう。

 最初は週2〜3回を目安に来ていただくと、
 症状が安定するのが早いです。

 今のペースで通うのが難しい場合は、
 週1回でもやるより全然違います。」
```

### メンテナンス施術への案内

```
「だいぶ安定してきましたね。
 このペースで来なくても
 日常生活に支障がなければ、
 月に1〜2回の短いメンテナンスに切り替えませんか？

 23分・2,500円で今の状態をキープできます。
 体が固まってから来るより、
 定期的に少しほぐす方が長持ちします。」
```

### ジムへの案内（初回案内）

```
「再発予防には、
 自分で体を動かせる状態をつくることが大切です。

 隣のジムで、腰に負担をかけずに
 股関節まわりと体幹を鍛えるメニューができます。

 まずは無料体験があるので、
 気軽に試してみてください。
 
 施術と組み合わせると効果が出やすいですよ。」
```

## 注意事項

- このプロトコルは院内運用ガイドであり、医学的診断の代替ではない
- 患者の状態変化に応じて柔軟に対応すること
- 痛みが増悪する・新たな症状が出た場合は医療機関受診を勧める
- 数回の施術で改善がない場合は施術内容・頻度を見直す
