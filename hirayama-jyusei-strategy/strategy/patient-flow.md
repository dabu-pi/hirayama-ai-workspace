# 患者フロー

## 導線設計

```
新患来院（保険・急性期）
    ↓
慢性化 or 再発リスク説明
    ↓
自費施術プラン提案
    ↓
運動療法導入
    ↓
ジム入会
    ↓
月会員（継続）
```

## ターゲット症状

- 慢性腰痛
- 肩こり
- 首痛
- 膝痛

## 転換ポイント

| ステップ | トリガー |
|---|---|
| 保険→自費 | 慢性化の説明・根本改善の訴求 |
| 自費→運動療法 | 「再発させない」運動指導の提案 |
| 自費プログラム→ジム体験（主導線） | 自費プログラム成約後、「再発予防トレーニング体験」として自然に案内（プログラムの流れの一部として提示） |
| 改善途中→ジム体験（補助導線） | 症状が落ち着き始めた頃に補助的に案内。主導線に乗らなかった患者への入口として活用 |
| ジム体験→ジム入会 | 体験後に「続けられそうなら月会員がある」と案内。強制しない |

> 詳細な導線方針・体験メニュー叩き台は `strategy/gym-lead-design.md` を参照。
