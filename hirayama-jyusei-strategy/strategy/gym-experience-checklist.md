# 再発予防トレーニング体験 — 現場確認チェックリスト

最終更新: 2026-03-21
案件: JBIZ-04（慢性疼痛強化プロジェクト2026）
参照元: `strategy/gym-lead-design.md`

---

## 使い方

このチェックリストを持って現場で確認し、種目・担当・パターンを最終確定する。
確認した項目に ✅ を入れ、OK / NG の結果を「確認結果」欄に記入する。

---

## A. 基本設備・動線の確認

### A-1. ケーブルマシン

| 項目 | 内容 |
|---|---|
| **確認内容** | ケーブルマシンは体験で使用できるか。空き状況・使用優先度はどうか |
| **確認理由** | 首肩こり向け「肩甲骨リトラクション」のケーブル版に使用。使えない場合は壁押し版に変更 |
| **OK → 採用** | 肩甲骨リトラクション（ケーブル版）を採用。負荷調整がしやすく体験の質が上がる |
| **NG → 代替** | 肩甲骨リトラクション（壁押し版）に変更。器具不要で完結する |

- [ ] 確認済み　確認結果: OK / NG　備考: _______________

### A-2. フォームローラー

|---|---|
| **確認内容** | フォームローラーは何本あるか。保管場所・取り出しやすさはどうか |
| **確認理由** | 首肩こり向け「胸椎モビリティ」のフォームローラー版に使用。ない場合はキャットカウ（自重）に変更 |
| **OK → 採用** | 胸椎モビリティ（フォームローラー版）を採用。胸椎の伸展をより引き出しやすい |
| **NG → 代替** | キャットカウ変法（自重・マット上）に変更。フォームローラー不要で完結する |

### A-3. マット

| 項目 | 内容 |
|---|---|
| **確認内容** | マットは何枚あるか。設置場所・取り出し・片付けの手間はどうか |

### A-4. 施術者単独で20分版を回せるか

| 項目 | 内容 |
|---|---|
| **確認内容** | 施術者1人で導入→運動体験（2種目）→まとめの20分を回せるか。後続の施術予約との兼ね合いはどうか |
| **確認理由** | 初期運用は施術者メインを想定。単独で回せない場合はジムスタッフとの連携フローを確定する必要がある |
| **OK → 採用** | 施術者単独で20分版を実施。ジムスタッフは必要時のみサポート |
| **NG → 代替** | ジムスタッフへの引き継ぎフローを設計する。引き継ぎに必要な情報（症状・注意点）の伝達方法を決める |

- [ ] 確認済み　確認結果: OK / NG　備考: _______________

### A-5. ジムスタッフの常駐状況

| 項目 | 内容 |
|---|---|
| **確認内容** | ジムスタッフは常駐しているか。呼び出し対応になるか。対応できる時間帯はいつか |
| **確認理由** | 機器の安全管理・補助はジムスタッフに依頼する想定。不在時の安全管理フローを決める必要がある |
| **OK（常駐）→ 採用** | ケーブルマシン使用時の補助・安全確認をジムスタッフに依頼できる |
| **NG（常駐なし）→ 代替** | ケーブルマシン・バンドなど器具の単独使用リスクを評価し、自重種目に絞って実施する |

- [ ] 確認済み　確認結果: 常駐 / 呼び出し / 不在時間帯あり　備考: _______________

### A-6. 施術スペース → 体験スペースへの動線

| 項目 | 内容 |
|---|---|
| **確認内容** | 施術後に患者をジムスペースへ案内するまでの動線・距離・所要時間はどうか |
| **確認理由** | 「施術後そのまま体験」を想定している。動線が複雑または時間がかかる場合は体験のハードルが上がる |
| **OK → 採用** | 施術後すぐに体験スペースへ案内する流れで設計 |
| **NG（動線が遠い / 複雑）→ 代替** | 体験は別日予約制に変更する。当日案内の代わりに「次回体験しましょう」と提案する流れにする |

- [ ] 確認済み　確認結果: OK / NG　動線メモ: _______________

---

### A-7. 患者導線上で無理なく案内できるか

| 項目 | 内容 |
|---|---|
| **確認内容** | 施術終了→体験案内のタイミングで、患者が時間的・心理的に受け入れやすいか |
| **確認理由** | 「売り込みではなく治療の次段階」として提案するためには、患者が急いでいない・疲弊していないタイミングが重要 |
| **OK → 採用** | 施術後に「今日少し体験してみますか？」と自然に案内できる |
| **NG（施術後が多忙 or 疲弊 / 時間が取れない）→ 代替** | 施術中または終了時に「次回体験しませんか」と別日予約で案内する流れにする |

- [ ] 確認済み　確認結果: OK / NG　備考: _______________

---

### A-8. 体験中の安全確認で詰まりやすい点

| 項目 | 内容 |
|---|---|
| **確認内容** | 体験中に安全確認や中断が必要になりやすい場面はどこか（患者の体調急変・器具トラブル等） |
| **確認理由** | 初期運用で施術者単独の場合、緊急対応できる体制を事前に決めておく必要がある |

| **NG（緊急対応できる体制がない）→ 代替** | ジムスタッフが不在の時間帯は体験を実施しない。常駐時間帯に体験を限定する |

## B. 慢性腰痛向け 種目別確認

### B-1. デッドバグ

| 項目 | 内容 |
|---|---|
| **前提条件** | マットが使用できること |
| **確認内容** | マットで床に仰向けになれる場所があるか |
| **OK → 採用** | デッドバグをそのまま採用（20分版・30分版ともに）|
| **NG → 代替種目** | ドローイン（立位・椅子座位）に変更。体幹安定の目的は共通で維持できる |

- [ ] 確認済み　確認結果: 採用 / 代替へ　代替種目（NG時）: ドローイン（立位 or 座位）

---

### B-2. ヒップヒンジ

| 項目 | 内容 |
|---|---|
| **前提条件** | 自重版は器具不要。ダンベル版は軽重量ダンベルの使用可否を確認 |
| **確認内容** | ダンベルを使用するか（使用する場合は保管場所・重量ラインナップを確認） |
| **OK（ダンベルあり）→ 採用** | ヒップヒンジ（ダンベル版）で採用。負荷感覚を体験させやすい |
| **NG（ダンベルなし or 管理煩雑）→ 対応** | ヒップヒンジ（自重版）で採用。十分に目的を達成できる |

- [ ] 確認済み　確認結果: ダンベル版 / 自重版　ダンベル最軽量: ___kg

---

### B-3. ブリッジ（股関節エクステンション）

| 項目 | 内容 |
|---|---|
| **前提条件** | マットが使用できること。バンド版はバンドの使用可否を確認 |
| **確認内容** | マットで床に仰向けになれるか。バンドがあるか（必須ではない） |
| **OK → 採用** | ブリッジ（自重）を採用。バンドがあればバンド版も選択肢に追加 |
| **NG（マットなし）→ 代替種目** | ヒップリフト（椅子 or ベンチ使用）に変更。臀筋活性化の目的は共通で維持できる |

- [ ] 確認済み　確認結果: 採用 / 代替へ　バンドあり: あり / なし
