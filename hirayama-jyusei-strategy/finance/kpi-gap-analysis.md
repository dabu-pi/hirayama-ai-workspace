## 差分を次元別に分解する

差分 **▲112,000円** をどの軸で埋めるかを整理する。

### A. 件数だけで埋める場合

| | 計算式 | 結果 |
|---|---|---|
| 必要件数 | 200,000 ÷ (4回 × 5,500) | **9.1件/月** |
| 現状比 | 9.1 ÷ 4 | **2.3倍** |
| 1件追加の効果 | 4回 × 5,500 | +22,000円/月 |
| 差分を件数で埋めるには | 112,000 ÷ 22,000 | **+5.1件 → 計9〜10件** |

→ 月9件以上の自費成約が必要。保険実人数50人の慢性候補20人から9件獲得は慢性移行率45%相当（実現可能だが現状より高い）。

### B. 回数だけで埋める場合

| | 計算式 | 結果 |
|---|---|---|
| 必要回数 | 200,000 ÷ (4件 × 5,500) | **9.1回/人/月** |
| 現状比 | 9.1 ÷ 4 | **2.3倍** |

→ 1人あたり9回以上は月4件コースを2.3周することになり、**現実的ではない**。回数単独では解決不可。

### C. 単価だけで埋める場合

| | 計算式 | 結果 |
|---|---|---|
| 必要単価 | 200,000 ÷ (4件 × 4回) | **12,500円/回** |
| 現状比 | 12,500 ÷ 5,500 | **2.3倍** |

→ **現実的ではない**。単価単独では解決不可。

### D. ジム転換率で補完する場合

> ジム月会費（asago7区分）平均 ≒ 7,480円/月（KPI逆算 C9 参照）

| ジム入会人数 | ジム月収 | 自費手技で必要な額 |
|---|---|---|
| 5人 | 37,400円 | 200,000 − 37,400 = **162,600円**（手技3件×10回または6件×5回で可能）|
| 10人 | 74,800円 | 200,000 − 74,800 = **125,200円**（手技5〜6件×4回で可能）|
| 15人 | 112,200円 | 200,000 − 112,200 = **87,800円**（手技4件×4回でほぼ達成）|

→ 現状4件×4回 = 88,000円 + **ジム15人入会 = 112,200円** で合計 **200,200円**（目標達成）。

## 損益構造の課題

> **月 +20万円 自費だけでは黒字化しない**（重要事項）

| 収益 | 金額 |
|---|---|
| 保険売上（仮）| 150,000円 |
| 自費売上（目標）| 200,000円 |
| **合計** | **350,000円** |
| **固定費（現金）**| **466,000円** |
| **現金収支** | **▲116,000円** |
| 固定費（全体）| 566,000円 |
| 経営収支 | ▲216,000円 |

→ **月 +20万円 は「黒字化目標」ではなく「自費収益基準」**。黒字化には保険売上の増加または固定費の見直し（特に機器リース150,000円）も必要。

**黒字化に必要な自費売上（現金ベース）:** 466,000 − 150,000 = **316,000円/月**

## 「月自費+20万円目標」と「黒字化目標」の違い

> 2つの目標は**別物**。混同すると判断基準がずれる。

| | 月自費 +20万円目標 | 現金黒字化目標 |
|---|---|---|
| **定義** | 自費売上を月200,000円以上にする | 月次現金収支をプラスにする |
| **基準** | 自費売上 ≧ 200,000円 | 総収益 ≧ 固定費（現金）466,000円 |
| **現在の達成条件** | 自費単独で達成可能（B案: 6件×4回+ジム10人）| 保険150,000円 + 自費316,000円以上 |
| **現状の位置** | 88,000円（44%）— 未達 | 88,000 + 150,000 = 238,000円（51%）— 大幅赤字 |
| **役割** | フェーズ1の成果指標（まず達成すべき目標）| フェーズ2の経営指標（事業継続の基準）|
| **目標達成の意味** | 「自費収益の仕組みが機能し始めた」確認 | 「事業として利益が出ている」確認 |

### 数字で見る段階目標

```
現状（実績ベース）
  総収益: 150,000（保険）+ 88,000（自費）= 238,000円
  固定費: 466,000円（現金）
  収支:   ▲228,000円

↓ フェーズ1: 月自費 +20万円 達成（B案）
  総収益: 150,000（保険）+ 206,800（自費+ジム）= 356,800円
  固定費: 466,000円（現金）
  収支:   ▲109,200円　← まだ赤字だが半減

↓ フェーズ2: 現金黒字化（自費316,000円以上）
  総収益: 150,000（保険）+ 316,000（自費）= 466,000円
  固定費: 466,000円（現金）
  収支:   ±0円　← 損益分岐点

↓ フェーズ3: 経営黒字化（自費416,000円以上）
  総収益: 150,000（保険）+ 416,000（自費）= 566,000円
  固定費: 566,000円（経営・スタッフ含む）
  収支:   ±0円
```

**暫定結論:**
- 月自費 +20万円 = **フェーズ1の達成確認基準**（仕組みが動いているかの確認）
- 現金黒字化 = **自費316,000円/月以上**（保険150,000円前提）
- 経営黒字化 = **自費416,000円/月以上**（スタッフ人件費100,000円含む）
