# メニューマスター再編方針

最終更新: 2026-03-21（症状別初回評価 叩き台確定）
案件: JBIZ-04（慢性疼痛強化プロジェクト2026）

---

## なぜ再編が必要か

現行のメニューマスター（`pricing.md` / `treatment-menu.md`）は「単品の商品一覧」として設計されており、患者がどの順番でどのメニューに進むかという**導線の流れ**が表現されていない。

現状の課題は価格水準よりも**患者導線設計**にある。具体的には以下の問題がある。

| 課題 | 内容 |
|---|---|
| 保険→自費の転換根拠が薄い | 慢性化の説明トリガーが設計に反映されていない |
| 症状別の自費入口がない | 「腰痛だから運動療法へ」という入口が存在しない |
| 運動療法→ジムの橋渡しが弱い | PT・4回コースがジム誘導と切り離されている |
| ジム月会員が1行しかない | 実際は利用コースで分かれる想定だが整理されていない |
| 「卒業設計」の概念が欠けている | 「治療して終わり」になりやすく再発→再来院の繰り返しになる |

---

## 新しい6ブロック構成

メニューマスターを「単品リスト」から「**保険→保険外→運動療法→ジムの流れ**」で再編する。

```
【ブロック1】保険入口
    ↓ 慢性化・再発リスクの説明
【ブロック2】保険単価アップ（オプション）
    ↓ 根本改善の訴求
【ブロック3】保険外入口（症状別）
    ↓ 「痛みを取るだけでなく再発させない」提案
【ブロック4】慢性主力施術
    ↓ 「身体の使い方を変える」段階へ
【ブロック5】運動再教育・卒業設計
    ↓ 「自分で管理できる身体」→継続の場へ
【ブロック6】ジム導線・月額コース
```

### ブロック1 — 保険入口

| メニュー | 内容 | 料金 |

| 保険施術 | 検査＋電気治療（約15分）＋手技（約5分） | 保険適用 |

- 急性期・新患の最初の接点
- ここで慢性化リスクを丁寧に説明し、ブロック2〜3への導線を作る

### ブロック2 — 保険単価アップ（オプション）

|---|---|---|---|

| 温熱追加 | 5分追加 | 550円 | 467円 |

- 保険患者の単価アップを図る希望制オプション
- 保険施術と同日提供・追加負担は小さく設計

> **シート（価格設定）への反映は未実施。** 運用ルール確定・人員体制確認後に追加する。

|---|---|---|---|

#### 各メニューの詳細

**① 腰痛改善 運動療法 初回評価**

- **内容:** 腰痛の原因になりやすい姿勢・股関節・体幹の使い方を評価し、その場で改善運動を体験してもらう初回メニュー
- **対象:** 慢性腰痛、繰り返す腰痛、保険適用外、保険終了後も不安が残る人
- **次提案先:** 慢性ケア手技50分 / 4回集中コース / ジム体験

**② 首肩こり改善 運動療法 初回評価**

- **内容:** 首肩こりの原因になりやすい姿勢・胸椎・肩甲骨の動きを評価し、その場で改善運動を体験してもらう初回メニュー
- **対象:** 慢性的な首こり肩こり、デスクワーク負担、保険適用外、保険終了後も再発不安がある人
- **次提案先:** 慢性ケア手技50分 / 4回集中コース / ジム体験

**③ 膝改善 運動療法 初回評価**

- **内容:** 膝の負担に関わる立ち方・股関節・足部の使い方を評価し、その場で改善運動を体験してもらう初回メニュー
- **対象:** 階段や立ち上がりで膝が気になる人、保険適用外、保険終了後も不安がある人
- **次提案先:** 慢性ケア手技50分 / 4回集中コース / ジム体験

#### 運用ルール

- この3メニューは中身の型は共通。見せ方（評価部位・説明内容）だけ症状別にする
- 初回評価の最後に必ず次の提案先を1つ示す

**次提案の基準:**

| 状態 | 提案先 |
|---|---|
| 痛みや筋緊張が強い | 慢性ケア手技50分 |
| 動きの癖が強く短期改善したい | 4回集中コース |
| 自分で続けられそう | ジム体験 |
| 既にジム会員でフォーム再教育が必要 | パーソナルトレーニング |

**対象外（必要時は医療受診を優先）:**

- 強い炎症・夜間痛・しびれ増悪・荷重困難などの症状がある場合は本メニューの対象外とし、医療受診を優先する

## ジム月額コース分岐の必要性

現行の「月会員 1行・月額7,480円」は実態と乖離する可能性がある。

| 想定利用者 | 現行の問題 | 必要な対応 |
|---|---|---|
| 軽度利用者（週1回程度） | 7,480円は割高感がある | ライトコース設定 |
| 標準利用者（週2〜3回） | 7,480円が妥当な水準 | スタンダードとして整理 |
| 治療継続者（PT込みを望む） | 別途8,800円が発生し不透明 | プレミアム（PT込み）設定 |
| ジム初体験者 | 入会前の体験機会がない | 体験メニュー設定 |

> **方針:** 将来的に4プランへ分ける前提で設計するが、**現時点での月額・内容はすべて未確定**。
> 月会員移行数が一定になってから価格・コース設計を確定する。

## 重要な考え方（導線設計の軸）

1. **保険患者の単価アップ** — ブロック2オプションで保険施術1回あたりの売上を底上げする
2. **保険適用外患者の自費導線** — 症状別の初回評価（ブロック3）を入口にする
3. **腰痛・膝などを入口にした運動療法導線** — 症状名でファーストステップを明確にする
4. **運動療法からジムへの橋渡し** — 4回集中コース完了後に自然にジム誘導できる流れを作る
5. **保険売上単価は当面仮置き** — 実績が出るまで現想定値（1,200円）で運用する

## 次にシートへ反映する際の方針

### 反映してよいもの（今すぐシート更新可）

- `価格設定` シートへのブロック見出し追加（既存行の上下に見出し行として追記する方式）
- メニュー名称の統一（例:「月会員」→「月会員 スタンダード（仮）」として暫定表記）

> **注意:** 既存行の並べ替えは行わない。KPI逆算の主力手技フラグ参照が行番号に依存しているため、行番号を変えると参照がずれる可能性がある。

### 反映前に確定が必要なもの

| 項目 | 確定前にやること |
|---|---|
| 症状別 運動療法 初回評価 | 内容・時間・料金を設計する |
| ジム体験メニュー | 体験の提供方法・料金を決める |
| ジム月会員 ライト / プレミアム | 月額・利用条件を決める |

### シート反映時の注意

- **既存行の行番号は変えない**（並べ替えは行わない）
- ブロック整理は、見出し行の追加・新設メニュー行の末尾追加で行う
- `KPI逆算` の `主力手技フラグ` 参照を壊さないことを最優先にする
