## 色分けルール（全シート共通）

| 色 | 意味 | 操作 |
|---|---|---|
| 🟡 黄色（#FFF2CC） | ユーザー入力セル | 自由に入力してよい |
| 🔵 水色（#CFE2F3） | 自動計算セル | 編集禁止（保護） |
| ⬜ グレー（#EFEFEF） | 見出し・ラベル | 編集禁止（保護） |
| 🔴 赤文字 | 未確定・要注意の値 | 数値が空 or 仮値のまま |
| 🟢 緑文字 | 確定済みの値 | 入力完了 |

---

## 編集保護の考え方

- **保護対象:** 数式セル（水色）・見出し行（グレー）
- **保護方法:** 「警告を表示する」モード（編集は可能だが確認ダイアログが出る）
- **入力可能セル:** 黄色セルのみ。列・行は固定し、追加削除はしない
- **注意:** 行を挿入・削除すると数式参照がズレるため、行の増減は禁止

## シート間の参照関係

```
Numeric_Assumptions ──→ Dashboard（固定費合計・患者数）
Pricing             ──→ Dashboard（未確定数・月次試算額）
                    ──→ KPI_Targets（プログラム単価）
Roadmap_Status      ──→ Dashboard（戦略完成度％）
Open_Items          ──→ Dashboard（未確定件数・次やること）
KPI_Targets         ──→ Dashboard（必要成約件数・達成見込み）
```

## 1. Dashboard

### 目的
プロジェクト全体の現在地を1枚で把握する。毎週ここを開けば状況が分かる状態にする。

### 主要項目

| 項目 | 内容 | セル種別 |
|---|---|---|
| プロジェクト名 | 平山接骨院 慢性疼痛強化プロジェクト | ラベル |
| 最終更新日 | 手入力 | 🟡 入力 |
| 現在フェーズ | Roadmap_Status から自動取得（「進行中」のフェーズ名） | 🔵 計算 |
| 戦略完成度 | Roadmap_Status の完了フェーズ数 ÷ 6 × 100% | 🔵 計算 |
| 数値確定度 | Open_Items の確定済み件数 ÷ 全件数 × 100% | 🔵 計算 |
| 未確定項目数 | Open_Items で「未確定」の件数（COUNTIF） | 🔵 計算 |
| 月自費目標 | KPI_Targets から参照 | 🔵 計算 |
| 必要成約件数/月 | KPI_Targets から参照 | 🔵 計算 |
| 次にやることTop3 | Open_Items の優先度「高」上位3行を参照 | 🔵 計算 |
| 一言メモ | 今週の状況など自由記入 | 🟡 入力 |

### このシートで何が分かるか
- 「今どのフェーズか」「戦略と数値のどちらが遅れているか」が数秒で分かる
- 次にやることが3行で表示されるため、毎日確認する習慣が作りやすい

## 5. KPI_Targets

### 目的
転換率の目標を入力し、「月+20万円」達成に必要な件数を逆算する。

### 主要項目

#### ブロックA: 目標値入力

| 項目名 | セル種別 | 初期値 |
|---|---|---|
| 月次来院患者数（目標） | 🟡 入力（Numeric_Assumptionsから手動転記も可） | 空白 |
| 慢性疼痛患者の割合（%） | 🟡 入力 | 空白 |
| 自費提案率（%） | 🟡 入力 | 空白 |
| 自費成約率（%） | 🟡 入力 | 空白 |
| ジム体験案内率（%） | 🟡 入力 | 空白 |
| ジム体験→入会率（%） | 🟡 入力 | 空白 |
| 月次退会率（%） | 🟡 入力 | 空白 |

#### ブロックB: 自動逆算

| 項目名 | 計算式 | セル種別 |
|---|---|---|
| 慢性疼痛候補数/月 | 来院数 × 慢性割合 | 🔵 計算 |
| 自費提案件数/月 | 候補数 × 提案率 | 🔵 計算 |
| 自費成約件数/月 | 提案件数 × 成約率 | 🔵 計算 |
| 見込み自費収益/月 | 成約件数 × 主力商品価格（Pricingから） | 🔵 計算 |
| 目標達成ギャップ | 200,000 − 見込み収益 | 🔵 計算 |
| ジム新規入会数/月 | 成約件数 × 体験案内率 × 入会率 | 🔵 計算 |

| ジム月次収益（6ヶ月後） | 会員数 × スタンダード月額（Pricingから） | 🔵 計算 |

### このシートで何が分かるか
- 「今の来院数で提案・成約をどの割合に設定すれば+20万円を達成できるか」が分かる
- ジム会員が何人になれば月次定期収益がいくらになるかの見通しが持てる

## 実装時の注意事項

1. **IFERROR 必須:** 入力欄が空のときに計算エラーが出ないよう、全計算式に IFERROR を付ける
2. **行の増減禁止:** 保護とともに「行の挿入・削除はしない」旨を Dashboard に注意書きとして表示する
3. **シート名は日本語不可:** Apps Script ではシート名にスペースや特殊文字を避ける（英字シート名を使用し、A1セルにタイトルを日本語表示）
4. **初回入力ガイド:** Dashboard の最下部に「最初にやること」として入力手順を3ステップで表示する
