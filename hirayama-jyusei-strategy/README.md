## このプロジェクトの一言要約

慢性疼痛に特化した整骨院 × ジムの複合モデルを作り、
「保険施術 → 自費施術 → 運動療法 → ジム継続」の導線で患者のLTVを最大化する。
初期主軸は **慢性腰痛**。月次自費収益 **+20万円** が最初のKPI目標。

---
