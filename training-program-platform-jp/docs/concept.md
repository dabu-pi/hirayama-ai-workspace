# concept

## 企画の本質

この企画の本質は、**日本語のプログラム配布型トレーニングアプリ**を作ることにある。
単なる筋トレ記録アプリではなく、「どのプログラムを選ぶか」「今日何をやるか」「どう積み上げたか」を一連で扱えることが重要。

## 価値の中心

初期価値の中心は、次の 3 点にある。

- 有名既存プログラムを日本語で導入しやすくする
- 当日の実行内容を迷わず表示する
- 記録を継続しやすくし、前回比較で成長を感じやすくする

## 構造の考え方

このアプリは、次の 3 層構造で考える。

1. ライブラリ
2. 実行 UI
3. 記録管理

### 1. ライブラリ

- プログラムを探す入口
- 対象者、頻度、目的、期間などで比較できる場
- 初心者にも「何を選べばいいか」が分かることを重視する

### 2. 実行 UI

- 今日やる内容を明確に見せる場
- セット、回数、重量、RPE、メモをその場で入力できることが重要
- スマホで片手でも扱いやすいことを優先する

### 3. 記録管理

- 前回何をやったか、どこまで進んだかを振り返る場
- 継続率を上げるため、入力と参照の負担を下げる
- 初期は個人利用中心でも、将来のユーザー別履歴保持に備える

## 優先するユーザー体験

初心者から中級者が、迷わず使えることを最優先にする。

優先順位:

- 難しい理論説明より、使い始めやすさ
- 高機能さより、当日の迷いにくさ
- 細かいカスタム性より、継続しやすさ

## 初期フェーズの価値

初期は、有名既存プログラムを導入しやすくすることに強い価値がある。
そのため、初期収録ではゼロから独自プログラムを大量に作るよりも、既に認知のあるプログラムを整理し、日本語で使いやすく再構成する方向を優先する。

ただし、名称・説明・構成の扱いは権利や表現方針の確認が必要であり、公開前に整理する。
