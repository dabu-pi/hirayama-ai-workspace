# glossary

## 用語方針

- UI では日本語を主、必要に応じて英語を補助表記する
- 種目名は日本語だけで分かりにくい場合、英語併記を許容する
- 強度やトレーニング用語は、意味が変わりやすいものほど用語集で固定する

## 用語辞書たたき台

| 英語 | 日本語 | 推奨表記 | メモ |
|---|---|---|---|
| Exercise | 種目 | 種目 | 画面上の一般表現は「種目」を優先 |
| Exercise Name | 種目名 | 種目名 | 必要に応じて英語名を補助表示 |
| Intensity | 強度 | 強度 | 重量だけでなく主観強度も含む文脈に注意 |
| RPE | 自覚的運動強度 | RPE | UI では `RPE` を優先し、説明文で補足する |
| AMRAP | できる限り反復 | AMRAP | 初見ユーザー向けに意味説明を併記する |
| Deload | 負荷調整週 / 軽減週 | デロード | 日本語だけだと伝わりにくいためカタカナ併記を推奨 |
| Volume | 総負荷量 / ボリューム | ボリューム | 文脈により「総セット量」「総負荷量」の補足が必要 |
| Progression | 漸進 / 進行設計 | プログレッション | 仕組み説明では日本語補足を付ける |

## 表記ゆれ整理

### 種目名

- 原則:
  日本語名を主表示、必要に応じて英語名を補助表示
- 例:
  `ベンチプレス (Bench Press)`
  `スクワット (Squat)`
  `デッドリフト (Deadlift)`

### 強度表現

- `重量`
- `回数`
- `セット`
- `RPE`

「強度」を重量だけの意味で使わないように注意する。

### RPE

- 推奨表記:
  `RPE`
- 補助表現:
  `主観的きつさ`

### AMRAP

- 推奨表記:
  `AMRAP`
- 補助表現:
  `できる限り反復`

### Deload

- 推奨表記:
  `デロード`
- 補助表現:
  `負荷調整週`

### Volume

- 推奨表記:
  `ボリューム`
- 文脈に応じて:
  `総負荷量`
  `総セット量`

### Progression

- 推奨表記:
  `プログレッション`
- 補助表現:
  `段階的な負荷進行`
