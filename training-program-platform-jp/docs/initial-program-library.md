# initial-program-library

## 方針

初期収録は、有名既存プログラムを中心に候補整理する。
ただし、名称・構成・説明文の扱いは公開前に権利や表現方針の確認が必要なものがあるため、必要に応じて `要確認` とする。

## カテゴリ別候補

### 筋力向上系

| 候補 | 収録優先度 | 想定対象 | 備考 |
|---|---|---|---|
| Starting Strength | 高 | 初心者 | 知名度が高い。日本語説明方針は要整理 |
| StrongLifts 5x5 | 高 | 初心者 | 導入しやすい。名称や説明の扱いは要確認 |
| Wendler 5/3/1 | 高 | 中級者 | 派生が多いため収録範囲整理が必要 |
| Texas Method | 中 | 中級者 | 負荷が高め。対象者の明確化が必要 |
| Madcow 5x5 | 中 | 初心者〜中級者 | 文脈整理が必要。要確認 |

### 筋肥大系

| 候補 | 収録優先度 | 想定対象 | 備考 |
|---|---|---|---|
| Push Pull Legs 系 | 高 | 初心者〜中級者 | 一般化された構成として扱いやすい |
| Upper Lower Split 系 | 高 | 初心者〜中級者 | 汎用的。独自再構成しやすい |
| PHUL | 中 | 中級者 | 構成整理が必要。要確認 |
| PHAT | 低〜中 | 中級者 | 初期 MVP ではやや複雑 |

### 初心者向け系

| 候補 | 収録優先度 | 想定対象 | 備考 |

| フルボディ 3日法 | 高 | 初心者 | 独自設計しやすい基礎カテゴリ |
| 週 2 回ベーシック法 | 高 | 初心者 | 継続しやすさ重視で有用 |

## 収録メモ

- 初期は「有名名称そのものの再現」より、「導入しやすい日本語体験」を優先する
- 既存名称をそのまま前面に出すか、参考元として扱うかは別途方針決定が必要
- プログラム構成、説明文、出典表記は公開前に確認する

## 権利・確認メモ

- `Starting Strength`: 要確認
- `StrongLifts 5x5`: 要確認
- `Wendler 5/3/1`: 要確認
- `Texas Method`: 要確認
- `Madcow 5x5`: 要確認
- `PHUL`: 要確認
- `PHAT`: 要確認

## 初期実装に向く考え方

- 最初は複雑すぎる派生プログラムを増やしすぎない
- 初心者が選びやすいベーシック構成を厚めにする
- 収録候補と公開候補を分けて管理できるようにする
